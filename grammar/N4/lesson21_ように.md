## 1. 〜ようにする（努力做到……/注意做……）||〜ようにする (Make an Effort To... / Be Careful To...)

### 接续||Conjugation

:::zh
- 动词辞书形 + ようにする
- 动词ない形 + ようにする
:::

:::en
- Verb dictionary form + ようにする
- Verb ない-form + ようにする
:::

### 含义||Meaning

:::zh
"努力做到……"、"注意/留心做……"，表示有意识地养成某种习惯或改变行为。
:::

:::en
"Make an effort to...," "be careful to...," "be sure to..." — expresses consciously trying to form a habit or change one's behavior.
:::

### 例句||Example Sentences

:::zh
1. 毎日野菜を食べる**ようにしている**。（注意每天吃蔬菜。→ 正在养成的习惯）
2. 遅刻しない**ようにしてください**。（请注意不要迟到。）
3. なるべく歩く**ようにしている**。（尽量走路。）
4. 早く寝る**ようにします**。（我会注意早点睡的。）
5. 甘いものを食べない**ようにしている**。（注意不吃甜食。）
6. 体に悪いものは飲まない**ようにしよう**。（注意不喝对身体不好的东西吧。）
:::

:::en
1. 毎日野菜を食べる**ようにしている**。(I make a point of eating vegetables every day. → an ongoing habit being formed)
2. 遅刻しない**ようにしてください**。(Please be careful not to be late.)
3. なるべく歩く**ようにしている**。(I try to walk as much as possible.)
4. 早く寝る**ようにします**。(I'll make sure to go to bed early.)
5. 甘いものを食べない**ようにしている**。(I'm being careful not to eat sweets.)
6. 体に悪いものは飲まない**ようにしよう**。(Let's try not to drink things that are bad for the body.)
:::

### ようにしている vs ようにする||ようにしている vs ようにする

:::zh
- ようにする = 从现在开始要注意（决意）
- ようにしている = 一直在注意（持续中的习惯）
:::

:::en
- ようにする = I will start being careful from now on (resolution)
- ようにしている = I have been making a point of it (ongoing habit)
:::

## 2. 〜ようになる（变得……/开始能……）||〜ようになる (Come To... / Have Reached the Point Where...)

### 接续||Conjugation

:::zh
- 动词辞书形 + ようになる
- 动词ない形 + ようになる（变得不再……）
- 可能形 + ようになる（变得能够……）
:::

:::en
- Verb dictionary form + ようになる
- Verb ない-form + ようになる (come to no longer...)
- Potential form + ようになる (come to be able to...)
:::

### 含义||Meaning

:::zh
表示**状态的自然变化**——"渐渐变得能/会做某事"或"变得不再做某事"。
:::

:::en
Expresses a **natural change of state** — "gradually come to be able to do something" or "come to no longer do something."
:::

### 例句||Example Sentences

:::zh
1. 日本語が話せる**ようになった**。（变得会说日语了。→ 能力的获得）
2. 子供が一人で着替えられる**ようになった**。（孩子变得能自己换衣服了。）
3. 漢字が読める**ようになりたい**。（想变得能读汉字。）
4. 最近、よく運動する**ようになった**。（最近开始经常运动了。）
5. お酒を飲まない**ようになった**。（变得不再喝酒了。）
6. 前は嫌いだったけど、納豆が食べられる**ようになった**。（以前讨厌，但现在变得能吃纳豆了。）
:::

:::en
1. 日本語が話せる**ようになった**。(I've come to be able to speak Japanese. → acquiring an ability)
2. 子供が一人で着替えられる**ようになった**。(The child has come to be able to change clothes on their own.)
3. 漢字が読める**ようになりたい**。(I want to reach the point where I can read kanji.)
4. 最近、よく運動する**ようになった**。(I've come to exercise frequently lately.)
5. お酒を飲まない**ようになった**。(I've come to not drink alcohol anymore.)
6. 前は嫌いだったけど、納豆が食べられる**ようになった**。(I used to dislike it, but I've come to be able to eat natto.)
:::

### ようにする vs ようになる||ようにする vs ようになる

:::zh
| | ようにする | ようになる |
|---|---|---|
| 性质 | **有意识的努力** | **自然的变化** |
| 主体 | 人的意志 | 状态的转变 |
| 例 | 毎日走るようにしている（我注意每天跑步）| 毎日走るようになった（变得每天跑了） |
| 例 | 野菜を食べるようにする（注意吃蔬菜）| 野菜が食べられるようになった（变得能吃蔬菜了） |
:::

:::en
| | ようにする | ようになる |
|---|---|---|
| Nature | **Conscious effort** | **Natural change** |
| Subject | Human will/intention | Shift in state |
| Ex. | 毎日走るようにしている (I make a point of running every day) | 毎日走るようになった (I've come to run every day) |
| Ex. | 野菜を食べるようにする (I'll be careful to eat vegetables) | 野菜が食べられるようになった (I've come to be able to eat vegetables) |
:::

## 3. 〜ように（目的 — "为了……"）||〜ように (Purpose — "So That..." / "In Order To...")

### 接续||Conjugation

:::zh
- 动词辞书形 + ように
- 动词ない形 + ように
- 可能形 + ように
:::

:::en
- Verb dictionary form + ように
- Verb ない-form + ように
- Potential form + ように
:::

### 含义||Meaning

:::zh
"为了能够……"，表示目的。和「ために」不同的是，ように用于**非意志性动词**（可能形、わかる、見える等）。
:::

:::en
"So that...," "in order to be able to..." — expresses purpose. Unlike 「ために」, ように is used with **non-volitional verbs** (potential forms, わかる, 見える, etc.).
:::

### 例句||Example Sentences

:::zh
1. 聞こえる**ように**、大きい声で話してください。（为了能听见，请大声说。）
2. 忘れない**ように**、メモした。（为了不忘，做了笔记。）
3. 日本語が上手になる**ように**、毎日練習している。（为了日语变好，每天练习。）
4. 遅刻しない**ように**、早く起きた。（为了不迟到，早起了。）
5. 試験に合格できる**ように**、頑張ります。（为了能通过考试，我会努力。）
:::

:::en
1. 聞こえる**ように**、大きい声で話してください。(Please speak loudly so that [I/everyone] can hear.)
2. 忘れない**ように**、メモした。(I took a note so I wouldn't forget.)
3. 日本語が上手になる**ように**、毎日練習している。(I practice every day so that my Japanese will improve.)
4. 遅刻しない**ように**、早く起きた。(I woke up early so as not to be late.)
5. 試験に合格できる**ように**、頑張ります。(I'll do my best so that I can pass the exam.)
:::

### ように vs ために（目的表达对比）||ように vs ために (Purpose Expression Comparison)

:::zh
| | ために | ように |
|---|---|---|
| 接续 | 意志动词辞书形 + ために | 非意志动词/可能形/ない形 + ように |
| 含义 | 为了做…… | 为了能够……/为了不…… |
| 主语 | 前后主语一致 | 前后主语可以不同 |

- 日本語を勉強する**ために**、日本に来た。（为了学日语来了日本。→ 意志动词）
- 日本語が話せる**ように**、毎日練習している。（为了能说日语，每天练习。→ 可能形）
- 子供が安全に遊べる**ように**、公園を作った。（为了孩子能安全玩耍，建了公园。→ 前后主语不同）
:::

:::en
| | ために | ように |
|---|---|---|
| Conjugation | Volitional verb (dict. form) + ために | Non-volitional verb / potential / ない-form + ように |
| Meaning | In order to do... | So that one can... / So as not to... |
| Subject | Same subject before and after | Subjects can differ |

- 日本語を勉強する**ために**、日本に来た。(I came to Japan in order to study Japanese. → volitional verb)
- 日本語が話せる**ように**、毎日練習している。(I practice every day so that I can speak Japanese. → potential form)
- 子供が安全に遊べる**ように**、公園を作った。(They built a park so that children could play safely. → different subjects)
:::

### 简单判断法||Simple Rule of Thumb

:::zh
> 前半句能用意志控制 → ために
> 前半句不能直接用意志控制（变化/能力/否定）→ ように
:::

:::en
> If the first clause can be directly controlled by will → ために
> If the first clause cannot be directly controlled by will (a change / ability / negation) → ように
:::

## 4. 〜ように言う / 〜ように頼む（转述请求/命令）||〜ように言う / 〜ように頼む (Relaying Requests / Commands)

### 含义||Meaning

:::zh
"叫某人做……"、"拜托某人做……"，用于转述请求或命令。
:::

:::en
"Tell someone to...," "ask someone to..." — used to relay requests or commands in indirect speech.
:::

### 例句||Example Sentences

:::zh
1. 先生は学生に遅刻しない**ように言った**。（老师对学生说不要迟到。）
2. 医者に酒をやめる**ように言われた**。（被医生说要戒酒。）
3. 母に早く帰る**ように頼まれた**。（被妈妈拜托早点回去。）
4. 静かにする**ように言ってください**。（请告诉他安静。）
:::

:::en
1. 先生は学生に遅刻しない**ように言った**。(The teacher told the students not to be late.)
2. 医者に酒をやめる**ように言われた**。(I was told by the doctor to quit drinking.)
3. 母に早く帰る**ように頼まれた**。(My mom asked me to come home early.)
4. 静かにする**ように言ってください**。(Please tell them to be quiet.)
:::

## 5. 〜ようにする/なる 相関表達汇总||Summary of ように Expressions

:::zh
| 表达 | 含义 | 例句 |
|------|------|------|
| 〜ようにする | 努力做到 | 早く寝るようにする |
| 〜ようになる | 自然变得 | 日本語が話せるようになった |
| 〜ように（目的） | 为了能够 | 忘れないように、メモした |
| 〜ように言う | 叫某人做 | 静かにするように言った |
| 〜ような＋名词 | 像……一样的 | 夢のような話（像梦一样的话） |
| 〜ように＋动词 | 像……一样地 | 鳥のように飛びたい（想像鸟一样飞） |
:::

:::en
| Expression | Meaning | Example |
|------------|---------|---------|
| 〜ようにする | Make an effort to | 早く寝るようにする (I'll try to go to bed early) |
| 〜ようになる | Come to naturally | 日本語が話せるようになった (I came to be able to speak Japanese) |
| 〜ように (purpose) | So that / in order to | 忘れないように、メモした (I took notes so I wouldn't forget) |
| 〜ように言う | Tell someone to | 静かにするように言った (I told them to be quiet) |
| 〜ような + noun | Like..., similar to... | 夢のような話 (a story like a dream) |
| 〜ように + verb | In the manner of... | 鳥のように飛びたい (I want to fly like a bird) |
:::

### A. 选择ようにする或ようになる||A. Choose ようにする or ようになる

:::zh
1. 毎日運動（　　）います。（一直注意每天运动）
2. 赤ちゃんが歩ける（　　）。（婴儿变得会走了）
3. 甘いものを食べない（　　）ください。（请注意不要吃甜食）
4. 前は読めなかった漢字が、読める（　　）。（变得能读了）
:::

:::en
1. 毎日運動（　　）います。(I have been making a point of exercising every day)
2. 赤ちゃんが歩ける（　　）。(The baby has come to be able to walk)
3. 甘いものを食べない（　　）ください。(Please be careful not to eat sweets)
4. 前は読めなかった漢字が、読める（　　）。(Kanji I couldn't read before, I've come to be able to read)
:::

<details>
<summary>答案</summary>

1. ようにして（有意识的努力 → ようにする）
2. ようになった（自然变化 → ようになる）
3. ようにして（请注意做到 → ようにする）
4. ようになった（能力获得 → ようになる）

</details>

### B. ために vs ように||B. ために vs ように

1. 健康の（　　）、毎日走っている。
2. 日本語が上手になる（　　）、毎日勉強している。
3. 留学する（　　）、お金を貯めている。
4. 忘れない（　　）、ノートに書いた。

<details>
<summary>答案 / Answers</summary>

:::zh
1. **ために**（健康是名词+のために）
2. **ように**（上手になる是非意志性变化）
3. **ために**（留学する是意志动词）
4. **ように**（忘れない是否定形）
:::

:::en
1. **ために** (健康 is a noun + のために)
2. **ように** (上手になる is a non-volitional change)
3. **ために** (留学する is a volitional verb)
4. **ように** (忘れない is a negative form)
:::

</details>

### C. 翻译||C. Translation

:::zh
1. 为了不迟到，早起了。
2. 孩子变得会自己穿衣服了。
3. 我注意每天喝水。（正在持续的习惯）
4. 老师叫学生不要在走廊跑。
5. 为了能看日语电影，每天学日语。
6. 以前不能吃的东西，现在变得能吃了。
:::
:::en
1. I got up early so as not to be late.
2. The child has come to be able to dress themselves.
3. I make a point of drinking water every day. (ongoing habit)
4. The teacher told the students not to run in the hallway.
5. I study Japanese every day so that I can watch Japanese movies.
6. Things I couldn't eat before, I can eat now.
:::

<details>
<summary>答案 / Answers</summary>

1. 遅刻しないように、早く起きた。
2. 子供が一人で服を着られるようになった。
3. 毎日水を飲むようにしている。
4. 先生は学生に廊下を走らないように言った。
5. 日本語の映画が見られるように、毎日日本語を勉強している。
6. 前は食べられなかったものが、食べられるようになった。

</details>

### D. 改错||D. Error Correction

:::zh
1. 日本語を話すために、毎日練習している。（想表达"为了能说日语"）
2. 子供が一人で食べられるようにした。（想表达"孩子自然变得能吃了"）
3. 先生に静かにするように言ってあげた。（想表达"被老师说要安静"）
4. 忘れないために、メモした。
:::

<details>
<summary>答案 / Answers</summary>

:::zh
1. △ 没有语法错误，但「话せる」是可能形 → ✅ 日本語が**話せる**ように、毎日練習している。（ように更准确表达"为了能够"）
2. ❌ ようにした = 有意识地做到 → ✅ ようになった（自然变化用なる不用する）
3. ❌ 言ってあげた = 我帮老师说 → ✅ 先生に静かにするように**言われた**（被老师说→受身形）
4. △ 不算错，但忘れない是非意志性 → ✅ 忘れない**ように**、メモした。（ように更标准）
:::

</details>
