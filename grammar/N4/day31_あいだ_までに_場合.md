## 1. 〜あいだ / 〜あいだに（……期间 / ……期间内）||〜あいだ / 〜あいだに (During~ / While~)

### 接续||Conjugation

:::zh
- 動詞ている形 + あいだ（に）
- 動詞辞書形 + あいだ（に）
- い形容詞い + あいだ（に）
- な形容詞な + あいだ（に）
- 名詞の + あいだ（に）
:::

:::en
- Verb ている-form + あいだ(に)
- Verb dictionary form + あいだ(に)
- い-adjective + あいだ(に)
- な-adjective + な + あいだ(に)
- Noun + の + あいだ(に)
:::

### 含义||Meaning

:::zh
- **あいだ**：表示整个持续期间，前后两个动作同时进行。"在……的整段时间里"
- **あいだに**：表示在某段时间内的某一时点，发生了另一件事。"在……期间的某个时候"
:::

:::en
- **あいだ**: The entire duration — both actions continue simultaneously. "Throughout the whole time that..."
- **あいだに**: At some point within a period, another event occurs. "At some point while..."
:::

### 例句||Example Sentences

:::zh
1. 赤ちゃんが寝ている**あいだ**、私は本を読んでいた。（婴儿睡觉的整段时间里，我在看书。→ 同时持续）
2. 赤ちゃんが寝ている**あいだに**、買い物に行った。（趁婴儿睡觉的时候，去买了东西。→ 期间内完成）
3. 留守の**あいだに**、泥棒が入った。（不在家的时候，小偷进来了。→ 某一时点发生）
4. 夏休みの**あいだ**、毎日プールに行った。（暑假的整段时间里，每天去游泳池。→ 整个期间）
5. 電車に乗っている**あいだ**、音楽を聴いていた。（坐电车的整段时间里，在听音乐。→ 同时持续）
6. 若い**あいだに**、たくさん旅行したい。（趁年轻的时候，想多旅行。→ 期间内做某事）
7. 日本にいる**あいだに**、富士山に登りたい。（在日本的期间内，想爬富士山。→ 期间内完成）
:::

:::en
1. 赤ちゃんが寝ている**あいだ**、私は本を読んでいた。(While the baby was sleeping, I was reading a book. → both continuous)
2. 赤ちゃんが寝ている**あいだに**、買い物に行った。(While the baby was sleeping, I went shopping. → completed during)
3. 留守の**あいだに**、泥棒が入った。(While I was away, a thief broke in. → event at a point)
4. 夏休みの**あいだ**、毎日プールに行った。(Throughout summer vacation, I went to the pool every day. → entire period)
5. 電車に乗っている**あいだ**、音楽を聴いていた。(The whole time I was on the train, I was listening to music. → both continuous)
6. 若い**あいだに**、たくさん旅行したい。(While I'm still young, I want to travel a lot. → within the period)
7. 日本にいる**あいだに**、富士山に登りたい。(While I'm in Japan, I want to climb Mt. Fuji. → within the period)
:::

### あいだ vs あいだに||あいだ vs あいだに

:::zh
| | あいだ | あいだに |
|---|---|---|
| 焦点 | **整段时间** | **时间段中的某一点** |
| 后句 | 持续性动作（〜ていた） | 瞬间/完成性动作 |
| 例 | 待っているあいだ、本を読んでいた | 待っているあいだに、電話が来た |
| 翻译 | 整段时间里一直…… | 在那段时间里（某一刻）…… |
:::

:::en
| | あいだ | あいだに |
|---|---|---|
| Focus | **Entire duration** | **A point within the duration** |
| Following clause | Continuous action (~ていた) | Momentary / completed action |
| Ex. | 待っているあいだ、本を読んでいた (While waiting, I was reading) | 待っているあいだに、電話が来た (While waiting, a call came) |
| Translation | Throughout the whole time... | At some point during that time... |
:::

### あいだに vs うちに||あいだに vs うちに

:::zh
- **あいだに**：单纯表示"在……期间内"
- **うちに**：含有"趁还来得及/趁这个状态还在"的急迫感

| | あいだに | うちに |
|---|---|---|
| 语感 | 中性，客观地描述期间 | 含有"赶紧趁现在"的紧迫感 |
| 例 | 休みのあいだに旅行した（假期期间旅行了） | 若いうちに旅行したい（趁年轻赶紧旅行） |
| 例 | 母が買い物のあいだに掃除した（妈妈买东西期间打扫了） | 暗くならないうちに帰ろう（趁天还没黑快回去） |
:::

:::en
| | あいだに | うちに |
|---|---|---|
| Nuance | Neutral, objectively describes the period | Carries urgency: "before it's too late" |
| Ex. | 休みのあいだに旅行した (I traveled during the break) | 若いうちに旅行したい (I want to travel while I'm still young) |
| Ex. | 母が買い物のあいだに掃除した (I cleaned while mom was shopping) | 暗くならないうちに帰ろう (Let's go home before it gets dark) |
:::

## 2. 〜までに（在……之前 / 截止……）||〜までに (By~ / Before the Deadline)

### 接续||Conjugation

:::zh
- 時間名詞 + までに
- 動詞辞書形 + までに
:::

:::en
- Time noun + までに
- Verb dictionary form + までに
:::

### 含义||Meaning

:::zh
表示**截止期限**——"在……之前（必须完成某事）"。强调的是最后期限这个时间点。
:::

:::en
Expresses a **deadline** — "by (a certain time), something must be done." The focus is on the deadline as a point in time.
:::

### 例句||Example Sentences

:::zh
1. 5時**までに**レポートを出してください。（请在5点之前交报告。→ 截止5点）
2. 来週の月曜日**までに**届けます。（在下周一之前送达。）
3. 授業が始まる**までに**、教室に入ってください。（在上课之前，请进教室。）
4. 締め切り**までに**間に合わなかった。（没赶上截止日期。）
5. 旅行に行く**までに**、パスポートを取らなければならない。（在去旅行之前，必须拿到护照。）
:::

:::en
1. 5時**までに**レポートを出してください。(Please submit the report by 5 o'clock. → deadline is 5)
2. 来週の月曜日**までに**届けます。(I will deliver it by next Monday.)
3. 授業が始まる**までに**、教室に入ってください。(Please enter the classroom before class starts.)
4. 締め切り**までに**間に合わなかった。(I didn't make it by the deadline.)
5. 旅行に行く**までに**、パスポートを取らなければならない。(I have to get a passport before going on the trip.)
:::

### まで vs までに||まで vs までに

:::zh
| | まで | までに |
|---|---|---|
| 含义 | "一直到……"（持续） | "在……之前"（截止） |
| 后句 | 持续性动作 | 完成性动作 |
| 例 | 5時**まで**働く（一直工作到5点） | 5時**までに**終わらせる（在5点之前完成） |
| 例 | 雨がやむ**まで**待つ（一直等到雨停） | 雨がやむ**までに**帰りたい（想在雨停之前回去） |
| 图解 | ──────→ 5時 | 　　　↑完成　5時 |
:::

:::en
| | まで | までに |
|---|---|---|
| Meaning | "Until..." (continuous) | "By..." (deadline) |
| Following clause | Ongoing action | Completed action |
| Ex. | 5時**まで**働く (I work until 5) | 5時**までに**終わらせる (I finish it by 5) |
| Ex. | 雨がやむ**まで**待つ (I wait until the rain stops) | 雨がやむ**までに**帰りたい (I want to go home before the rain stops) |
| Diagram | ──────→ 5:00 | 　　↑done　5:00 |
:::

## 3. 〜おきに（每隔……）||〜おきに (Every~ / At Intervals Of~)

### 接续||Conjugation

:::zh
- 数量词 + おきに
:::

:::en
- Counter/quantity word + おきに
:::

### 含义||Meaning

:::zh
"每隔……"，表示固定间隔反复发生。
:::

:::en
"Every..." or "at intervals of..." — expresses something recurring at fixed intervals.
:::

### 例句||Example Sentences

:::zh
1. この薬は4時間**おきに**飲んでください。（请每隔4小时吃一次这个药。）
2. 2メートル**おきに**木を植えた。（每隔2米种了一棵树。）
3. 30分**おきに**休憩する。（每隔30分钟休息一次。）
4. オリンピックは4年**おきに**行われる。（奥运会每隔4年举行一次。）
5. 1ページ**おきに**写真がある。（每隔一页有一张照片。）
:::

:::en
1. この薬は4時間**おきに**飲んでください。(Please take this medicine every 4 hours.)
2. 2メートル**おきに**木を植えた。(Trees were planted every 2 meters.)
3. 30分**おきに**休憩する。(I take a break every 30 minutes.)
4. オリンピックは4年**おきに**行われる。(The Olympics are held every 4 years.)
5. 1ページ**おきに**写真がある。(There is a photo every other page.)
:::

### おきに vs ごとに||おきに vs ごとに

:::zh
| | おきに | ごとに |
|---|---|---|
| 核心 | 强调"间隔" | 强调"每一个" |
| 数量用法 | 3日おきに = 隔3天（第4天做） | 3日ごとに = 每3天（第3天做） |
| 注意 | 数字大时两者差异明显 | 日常会话中常可互换 |
| 非数量用法 | ❌ 不可 | ✅ 国ごとに文化が違う（每个国家文化不同） |

> 小贴士：「1つおきに」=「隔一个」，所以「1日おきに」= 隔一天 = 每两天一次。
:::

:::en
| | おきに | ごとに |
|---|---|---|
| Core idea | Emphasizes "interval/gap" | Emphasizes "each one" |
| With numbers | 3日おきに = every 4th day (3-day gap) | 3日ごとに = every 3 days |
| Note | Difference is clear with larger numbers | Often interchangeable in casual use |
| Non-number use | ❌ Not possible | ✅ 国ごとに文化が違う (Culture differs from country to country) |

> Tip: 「1つおきに」 = "every other one," so 「1日おきに」 = every other day = once every 2 days.
:::

## 4. 〜場合は（……的情况下）||〜場合は (In the Case Of~ / If~)

### 接续||Conjugation

:::zh
- 動詞普通形 + 場合は
- い形容詞い + 場合は
- な形容詞な + 場合は
- 名詞の + 場合は
:::

:::en
- Verb plain form + 場合は
- い-adjective + 場合は
- な-adjective + な + 場合は
- Noun + の + 場合は
:::

### 含义||Meaning

:::zh
"在……的情况下"、"如果……的话"。比「たら」更正式，常用于说明书、规则、指南等书面场景。
:::

:::en
"In the case of...," "if..." — more formal than 「たら」. Commonly used in manuals, rules, guidelines, and written instructions.
:::

### 例句||Example Sentences

:::zh
1. 地震の**場合は**、すぐに机の下に入ってください。（地震的情况下，请立即躲到桌子下面。）
2. 熱がある**場合は**、学校を休んでください。（发烧的情况下，请不要来上学。）
3. パスワードを忘れた**場合は**、こちらをクリックしてください。（忘记密码的情况下，请点击这里。）
4. 雨の**場合は**、試合は中止です。（下雨的情况下，比赛取消。）
5. 分からない**場合は**、先生に聞いてください。（不明白的情况下，请问老师。）
:::

:::en
1. 地震の**場合は**、すぐに机の下に入ってください。(In case of an earthquake, please get under a desk immediately.)
2. 熱がある**場合は**、学校を休んでください。(If you have a fever, please stay home from school.)
3. パスワードを忘れた**場合は**、こちらをクリックしてください。(If you forgot your password, please click here.)
4. 雨の**場合は**、試合は中止です。(In case of rain, the match will be canceled.)
5. 分からない**場合は**、先生に聞いてください。(If you don't understand, please ask the teacher.)
:::

### 場合は vs たら vs と||場合は vs たら vs と

:::zh
| | 場合は | たら | と |
|---|---|---|---|
| 语感 | 正式、书面 | 口语、日常 | 客观规律/必然 |
| 场景 | 说明书、规则 | 日常对话 | 自然现象、机器操作 |
| 例 | 故障の場合は連絡してください | 壊れたら連絡してね | ボタンを押すと、ドアが開く |
:::

:::en
| | 場合は | たら | と |
|---|---|---|---|
| Tone | Formal, written | Casual, everyday | Objective/inevitable |
| Scene | Manuals, rules | Daily conversation | Natural phenomena, mechanical operations |
| Ex. | 故障の場合は連絡してください (In case of malfunction, contact us) | 壊れたら連絡してね (If it breaks, let me know) | ボタンを押すと、ドアが開く (When you press the button, the door opens) |
:::

## 5. 〜たらどう（ですか）（……怎么样？）||〜たらどう(ですか) (How About Doing~?)

### 接续||Conjugation

:::zh
- 動詞た形 + らどう（ですか）
:::

:::en
- Verb た-form + らどう(ですか)
:::

### 含义||Meaning

:::zh
"做……怎么样？"，用于向对方提建议。语气随场合不同，可以是轻松的建议，也可以带有催促/不耐烦的语气。
:::

:::en
"How about doing...?" — used to make a suggestion to someone. Depending on tone and context, it can be a light suggestion or carry a pushy/impatient nuance.
:::

### 例句||Example Sentences

:::zh
1. 疲れたなら、少し休んだ**らどうですか**。（累了的话，休息一下怎么样？→ 温和建议）
2. 医者に行っ**たらどう**？（去看看医生怎么样？→ 朋友间建议）
3. 一度、先生に相談し**たらどうですか**。（去和老师商量一次怎么样？）
4. そんなに嫌なら、やめ**たらどう**？（那么讨厌的话，不做了怎么样？→ 带有催促语气）
5. 日本語の映画を見**たらどうですか**。（看看日语电影怎么样？）
:::

:::en
1. 疲れたなら、少し休んだ**らどうですか**。(If you're tired, how about resting a bit? → gentle suggestion)
2. 医者に行っ**たらどう**？(How about going to the doctor? → casual, between friends)
3. 一度、先生に相談し**たらどうですか**。(How about consulting your teacher?)
4. そんなに嫌なら、やめ**たらどう**？(If you hate it that much, why don't you just quit? → pushy tone)
5. 日本語の映画を見**たらどうですか**。(How about watching a Japanese movie?)
:::

### たらどう vs たほうがいい vs ませんか||たらどう vs たほうがいい vs ませんか

:::zh
| | たらどう | たほうがいい | ませんか |
|---|---|---|---|
| 语气 | 轻建议～催促 | 忠告、推荐 | 礼貌邀请 |
| 方向 | 让对方自己判断 | 说话人认为这样比较好 | 一起做的邀请 |
| 例 | 休んだらどう？ | 休んだほうがいいよ | 休みませんか |
| 翻译 | 休息一下怎么样？ | 还是休息比较好哦 | 要不要休息？ |
:::

:::en
| | たらどう | たほうがいい | ませんか |
|---|---|---|---|
| Tone | Light suggestion ~ pushy | Advice, recommendation | Polite invitation |
| Direction | Let the listener decide | Speaker thinks it's better | Invitation to do together |
| Ex. | 休んだらどう？ | 休んだほうがいいよ | 休みませんか |
| Translation | How about resting? | You should rest | Would you like to rest? |
:::

## 6. 〜てよかった / 〜なくてよかった（做了……真好 / 没做……真好）||〜てよかった / 〜なくてよかった (Glad I Did / Glad I Didn't)

### 接续||Conjugation

:::zh
- 動詞て形 + よかった
- 動詞なくて + よかった
:::

:::en
- Verb て-form + よかった
- Verb なくて + よかった
:::

### 含义||Meaning

:::zh
表示对过去做过（或没做过）的事感到庆幸、欣慰。"做了……真是太好了" / "没做……真是太好了"。
:::

:::en
Expresses relief or satisfaction about a past action taken (or not taken). "I'm glad I did..." / "I'm glad I didn't..."
:::

### 例句||Example Sentences

:::zh
1. 傘を持っ**てきてよかった**。（带了伞来真好。→ 因为后来下雨了）
2. 日本語を勉強し**てよかった**。（学了日语真好。→ 感到有意义）
3. あの映画を見**てよかった**。（看了那部电影真好。）
4. 早く起き**てよかった**。（早起了真好。→ 赶上了电车等）
5. あの時、諦め**なくてよかった**。（那时候没有放弃真好。）
6. 変なものを食べ**なくてよかった**。（没吃奇怪的东西真好。→ 别人吃了肚子疼）
7. 嘘をつか**なくてよかった**。（没说谎真好。）
:::

:::en
1. 傘を持っ**てきてよかった**。(I'm glad I brought an umbrella. → because it rained later)
2. 日本語を勉強し**てよかった**。(I'm glad I studied Japanese. → feels worthwhile)
3. あの映画を見**てよかった**。(I'm glad I watched that movie.)
4. 早く起き**てよかった**。(I'm glad I woke up early. → caught the train, etc.)
5. あの時、諦め**なくてよかった**。(I'm glad I didn't give up back then.)
6. 変なものを食べ**なくてよかった**。(I'm glad I didn't eat that weird thing. → others got a stomachache)
7. 嘘をつか**なくてよかった**。(I'm glad I didn't lie.)
:::

### 〜ばよかった（要是做了就好了……/早该……）||〜ばよかった (I Should Have… / I Wish I Had…)

:::zh
接续：動詞ば形 + よかった

表示对**过去没做的事**感到后悔。"要是当时做了就好了"。

1. もっと早く起き**ればよかった**。（要是早点起来就好了。）
2. あの時、彼女に告白す**ればよかった**。（那时候要是向她告白就好了。）
3. 大学で日本語を勉強す**ればよかった**。（大学时要是学了日语就好了。）
4. 傘を持ってく**ればよかった**。（要是带伞就好了。）

否定形：〜なければよかった = "要是没做就好了"（后悔做了）

5. あんなに食べ**なければよかった**。（要是没吃那么多就好了。）
6. あの映画を見**なければよかった**。（要是没看那部电影就好了。）

> **てよかった vs ばよかった**：
> - てよかった → 做了，感到**庆幸** ✓
> - ばよかった → 没做，感到**后悔** ✗
:::

:::en
Conjugation: Verb ば-form + よかった

Expresses regret about something you **didn't do** in the past. "I wish I had…" / "I should have…"

1. もっと早く起き**ればよかった**。(I should have woken up earlier.)
2. あの時、彼女に告白す**ればよかった**。(I wish I had confessed to her at that time.)
3. 大学で日本語を勉強す**ればよかった**。(I wish I had studied Japanese in university.)
4. 傘を持ってく**ればよかった**。(I should have brought an umbrella.)

Negative: 〜なければよかった = "I wish I hadn't done…" (regret doing it)

5. あんなに食べ**なければよかった**。(I wish I hadn't eaten so much.)
6. あの映画を見**なければよかった**。(I wish I hadn't watched that movie.)

> **てよかった vs ばよかった:**
> - てよかった → Did it, feel **glad** ✓
> - ばよかった → Didn't do it, feel **regret** ✗
:::

## 7. 〜てすみません / 〜なくてすみません（做了……对不起 / 没做……对不起）||〜てすみません / 〜なくてすみません (Sorry for Doing / Sorry for Not Doing)

### 接续||Conjugation

:::zh
- 動詞て形 + すみません
- 動詞なくて + すみません
:::

:::en
- Verb て-form + すみません
- Verb なくて + すみません
:::

### 含义||Meaning

:::zh
用て形表示道歉的原因。"因为做了……，对不起" / "因为没做……，对不起"。
:::

:::en
Uses the て-form to express the reason for an apology. "I'm sorry for doing..." / "I'm sorry for not doing..."
:::

### 例句||Example Sentences

:::zh
1. 遅れ**てすみません**。（迟到了对不起。）
2. うるさくし**てすみません**。（太吵了对不起。）
3. 連絡しなく**てすみません**。（没有联系你，对不起。）
4. 約束を守れなく**てすみません**。（没能遵守约定，对不起。）
5. 昨日行かなく**てすみませんでした**。（昨天没去，对不起。→ 过去式更礼貌）
6. お待たせし**てすみません**。（让您久等了，对不起。）
:::

:::en
1. 遅れ**てすみません**。(I'm sorry for being late.)
2. うるさくし**てすみません**。(I'm sorry for being noisy.)
3. 連絡しなく**てすみません**。(I'm sorry for not contacting you.)
4. 約束を守れなく**てすみません**。(I'm sorry for not being able to keep the promise.)
5. 昨日行かなく**てすみませんでした**。(I'm sorry for not going yesterday. → past tense is more polite)
6. お待たせし**てすみません**。(I'm sorry for keeping you waiting.)
:::

### てすみません vs てごめん vs て申し訳ない||てすみません vs てごめん vs て申し訳ない

:::zh
| | てすみません | てごめん（なさい） | て申し訳ありません |
|---|---|---|---|
| 丁寧度 | 普通の丁寧 | カジュアル | 非常に丁寧 |
| 场景 | 日常、对不太熟的人 | 朋友、家人 | 工作、对上级/客户 |
| 例 | 遅れてすみません | 遅れてごめん | 遅れて申し訳ありません |
| 翻译 | 迟到了不好意思 | 迟到了抱歉啦 | 迟到了非常抱歉 |
:::

:::en
| | てすみません | てごめん(なさい) | て申し訳ありません |
|---|---|---|---|
| Politeness | Standard polite | Casual | Very formal/polite |
| Scene | Daily life, acquaintances | Friends, family | Work, superiors, clients |
| Ex. | 遅れてすみません | 遅れてごめん | 遅れて申し訳ありません |
| Translation | Sorry I'm late | Sorry I'm late (casual) | I deeply apologize for being late |
:::

## 今日练习||Today's Practice

### A. あいだ vs あいだに||A. あいだ vs あいだに

:::zh
空所に「あいだ」または「あいだに」を入れてください。

1. 母が料理をしている（　　）、私は掃除をしていた。
2. 母が料理をしている（　　）、宅配便が来た。
3. 留守の（　　）、猫が花瓶を割った。
4. 冬休みの（　　）、毎日スキーをした。
5. 日本にいる（　　）、日本語が上手になった。
:::

:::en
Fill in the blank with あいだ or あいだに.

1. 母が料理をしている（　　）、私は掃除をしていた。(While mom was cooking, I was cleaning.)
2. 母が料理をしている（　　）、宅配便が来た。(While mom was cooking, a delivery came.)
3. 留守の（　　）、猫が花瓶を割った。(While I was out, the cat broke the vase.)
4. 冬休みの（　　）、毎日スキーをした。(During winter break, I skied every day.)
5. 日本にいる（　　）、日本語が上手になった。(While in Japan, my Japanese improved.)
:::

<details>
<summary>答案 / Answers</summary>

:::zh
1. **あいだ** — 掃除していた＝持続動作 → 全期間を通じて
2. **あいだに** — 宅配便が来た＝一回限りの出来事 → ある時点
3. **あいだに** — 花瓶を割った＝一回の出来事
4. **あいだ** — 毎日スキーをした＝期間全体にわたる行為
5. **あいだに** — 上手になった＝期間内に起きた変化
:::

</details>

### B. まで vs までに||B. まで vs までに

:::zh
空所に「まで」または「までに」を入れてください。

1. 3時（　　）ここで待っていてください。
2. 3時（　　）ここに来てください。
3. 来月（　　）にこの本を返してください。
4. 雨がやむ（　　）、図書館にいよう。
:::

:::en
Fill in with まで or までに.

1. 3時（　　）ここで待っていてください。(Please wait here until 3.)
2. 3時（　　）ここに来てください。(Please come here by 3.)
3. 来月（　　）にこの本を返してください。(Please return this book by next month.)
4. 雨がやむ（　　）、図書館にいよう。(Let's stay in the library until the rain stops.)
:::

<details>
<summary>答案 / Answers</summary>

:::zh
1. **まで** — 「待つ」は持続動作 → 3時まで待ち続ける
2. **までに** — 「来る」は完了動作 → 3時が締め切り
3. **までに** — 「返す」は完了動作 → 来月が締め切り
4. **まで** — 「いる」は持続 → 雨がやむ時点まで
:::

</details>

### E. 改错||E. Error Correction

:::zh
間違いを直してください。

1. 母が寝ているあいだに、私はテレビを見ていた。（想表达"妈妈睡觉的整段时间里我在看电视"）
2. 5時までにずっと勉強していた。（想表达"一直学到5点"）
3. 分からないの場合は、聞いてください。（接续有误）
4. 薬を飲んでよかった。明日までに届けます。（两句都有问题吗？）
:::

<details>
<summary>答案 / Answers</summary>

:::zh
1. ❌ あいだに → ✅ **あいだ**（テレビを見ていた＝持続動作、全期間を通じて → あいだ）
2. ❌ までに → ✅ **まで**（ずっと＝持続 → 「5時まで勉強していた」）
3. ❌ 分からないの場合は → ✅ **分からない場合は**（動詞普通形は直接＋場合、「の」は不要）
4. 「薬を飲んでよかった」は正しい（薬を飲んで良かった＝飲んで安心）。「明日までに届けます」も正しい（締め切りの用法）。→ 両方とも正しい。
:::

</details>
