# N4 文法チェックリスト

> JLPT N4 核心语法约 74 条（不含 N5 已讲内容）。在 N5 基础上扩展使役、被动、授受等复杂表达。

## 条件表达（N5 で学んだ ば/たら/なら の応用）

- [ ] 〜ても — 即使~也~
- [ ] 〜場合は — 在~情况下
- [ ] 〜たらどう — 做~怎么样？（建议）

## 受身形（被动）

- [ ] 直接受身 — 被~做（食べられる）
- [ ] 間接受身（迷惑の受身）— 被~（不情愿/受害）
- [ ] 持ち主の受身 — 所有者被动（足を踏まれた）
- [ ] 〜によって — 被~（正式，施动者标志）

## 使役形

- [ ] 使役形変形 — させる/させる（三类动词规则）
- [ ] 強制の使役 — 强制做（〜を〜させる）
- [ ] 許可の使役 — 允许做（〜に〜させる）
- [ ] 使役受身形 — 被迫做（〜させられる）

## 授受表现

- [ ] あげる — 给（说话人→他人 / 他人→他人）
- [ ] もらう — 得到（说话人←他人）
- [ ] くれる — 给我（他人→说话人/内群）
- [ ] 〜てあげる — 为~做（动作授受）
- [ ] 〜てもらう — 请~做/得到~做
- [ ] 〜てくれる — 为我做~
- [ ] 差し上げる/いただく/くださる — 敬语版授受

## ように 系列

- [ ] 〜ようにする — 努力做到/注意
- [ ] 〜ようになる — 变得能够/开始~
- [ ] 〜ないようにする — 注意不要~
- [ ] 〜ように（目的）— 为了（不可控动词）

## ことに 系列

- [ ] 〜ことにする — 决定做~
- [ ] 〜ことにしている — 一直坚持做~（个人习惯）
- [ ] 〜ことになる — 变成了~（被决定）
- [ ] 〜ことになっている — 规定是~/预定是~

## はず 系列

- [ ] 〜はずだ — 应该是~（有根据的推测）
- [ ] 〜はずがない — 不可能~（强烈否定）
- [ ] 〜はずだったのに — 本应该~却~

## 时间相关

- [ ] 〜ばかり — 刚刚做完~
- [ ] 〜ところだ（するところ）— 正要做~
- [ ] 〜ところだ（しているところ）— 正在做~
- [ ] 〜ところだ（したところ）— 刚做完~
- [ ] 〜て以来 — 自从~以来

## 补助动词

- [ ] 〜てしまう（ちゃう/じゃう）— 做完了/遗憾
- [ ] 〜ておく（とく）— 事先准备/放置不管
- [ ] 〜てある — 已做好的状态（他动词）
- [ ] 〜ていく — 逐渐变化（离说话人方向）
- [ ] 〜てくる — 逐渐变化（向说话人方向）

## という 系列

- [ ] 〜という — 叫做~/所谓的~
- [ ] 〜ということだ（伝聞）— 据说~
- [ ] 〜ということだ（意味）— 也就是说~
- [ ] 〜というのは〜ことだ — 所谓~就是~

## わけ 系列

- [ ] 〜わけだ — 难怪/当然/也就是说
- [ ] 〜わけがない — 不可能~
- [ ] 〜わけではない — 并非~
- [ ] 〜わけにはいかない — 不能~/必须~

## もの 系列

- [ ] 〜ものだ（感嘆）— 真是~啊
- [ ] 〜ものだ（回想）— 以前常常~
- [ ] 〜ものだ（当然）— 本该~
- [ ] 〜ものだから — 因为~（带辩解语气）

## 复合动词

- [ ] 〜出す — 突然开始~（泣き出す、降り出す）
- [ ] 〜始める — 有意识开始~（勉強し始める）
- [ ] 〜続ける — 持续做~（走り続ける）
- [ ] 〜方（かた）— ~的做法/方法（食べ方、使い方）

## 建议与后悔

- [ ] 〜ばよかった — 早该~/要是~就好了（后悔）

## 敬语基础

- [ ] 〜です/〜ます — 丁寧語（礼貌体）
- [ ] お/ご〜になる — 尊敬語基本形
- [ ] お/ご〜する — 謙譲語基本形
- [ ] 〜てもらえませんか — 能否请您~（礼貌请求）

## 引用・思考

- [ ] 〜と思う — 我觉得~/我认为~
- [ ] 〜かどうか — 是否~/不知道是不是~
- [ ] 〜のように/〜のような — 像~一样
- [ ] 〜なさい — 做~！（命令，比て形强）

## 时间补充

- [ ] 〜あいだ/〜あいだに — ~期间/趁~的时候
- [ ] 〜までに — 在~之前（截止期限）
- [ ] 〜おきに — 每隔~

## 其他

- [ ] 〜やすい/〜にくい — 容易~/难以~
- [ ] 〜がる — 表现出~（第三人称感情）
- [ ] 〜てほしい — 希望（别人）做~
- [ ] 〜てよかった — 做了~真好（庆幸）
- [ ] 〜てすみません — 做了~很抱歉
- [ ] 〜必要がある — 有必要做~
- [ ] 〜がする — 有~的感觉/味道/声音
