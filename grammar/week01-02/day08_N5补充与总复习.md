## 1. 〜ことができる（能力/可能 — 正式版）||1. 〜ことができる (Ability/Possibility — Formal)

### 接続

:::zh
动词辞书形 + ことができる
:::

:::en
Verb dictionary form + ことができる
:::

### 含义

:::zh
和可能形意思相同，但更正式/书面。多用于写作、演讲、规章说明等场合。
:::

:::en
Same meaning as the potential form, but more formal and written. Commonly used in essays, speeches, rules, and official notices.
:::

### 例句

:::zh
1. 日本語を話す**ことができます**。（能说日语。）
2. ここで泳ぐ**ことができません**。（这里不能游泳。）
3. インターネットで予約する**ことができる**。（可以在网上预约。）
4. 18歳以上の方は参加する**ことができます**。（18岁以上的人可以参加。）
5. この図書館では本を借りる**ことができます**。（在这个图书馆可以借书。）
6. 外国人でもこのサービスを利用する**ことができる**。（外国人也可以使用这项服务。）
:::

:::en
1. 日本語を話す**ことができます**。(I am able to speak Japanese.)
2. ここで泳ぐ**ことができません**。(Swimming is not permitted here.)
3. インターネットで予約する**ことができる**。(Reservations can be made online.)
4. 18歳以上の方は参加する**ことができます**。(Those 18 and over may participate.)
5. この図書館では本を借りる**ことができます**。(Books may be borrowed at this library.)
6. 外国人でもこのサービスを利用する**ことができる**。(Even foreigners can use this service.)
:::

### 辨析：可能形 vs ことができる||Contrast: Potential Form vs ことができる

:::zh
- 日本語が話**せる**。（口语/日常 ✅）
- 日本語を話す**ことができる**。（书面/正式 ✅）

| | 可能形 | ことができる |
|---|---|---|
| 语感 | 口语、自然 | 书面、正式 |
| 场合 | 日常会话 | 演讲、公告、写作 |
| 长度 | 短 | 长 |
| 意思 | 相同 | 相同 |

> 考试中两者都会出现，日常会话更多用可能形。正式文章/公告更多用ことができる。
:::

:::en
- 日本語が話**せる**。(casual / everyday ✅)
- 日本語を話す**ことができる**。(written / formal ✅)

| | Potential form | ことができる |
|---|---|---|
| Nuance | Casual, natural | Written, formal |
| Setting | Daily conversation | Speeches, notices, essays |
| Length | Short | Longer |
| Meaning | Identical | Identical |

> Both appear in exams. The potential form is more common in daily conversation; ことができる is more common in formal writing and announcements.
:::

## 2. 〜ことがある / 〜ことはない||2. 〜ことがある / 〜ことはない

### 〜ことがある（有时会……）||〜ことがある (Sometimes…)

:::zh
**接续：** 动词辞书形 + ことがある
:::

:::en
**Conjugation:** Verb dictionary form + ことがある
:::

### 例句

:::zh
1. 朝ご飯を食べない**ことがある**。（有时候不吃早饭。）
2. バスが遅れる**ことがある**。（巴士有时会晚点。）
3. 日本語で夢を見る**ことがある**。（有时会做日语的梦。）
4. 疲れた日は、何もしないで寝る**ことがある**。（累的时候，有时什么都不做就睡了。）
5. 朝、目覚まし時計が聞こえない**ことがある**。（早上有时听不到闹钟。）
6. 週末は夜遅くまで起きている**ことがある**。（周末有时熬到很晚。）
:::

:::en
1. 朝ご飯を食べない**ことがある**。(Sometimes I skip breakfast.)
2. バスが遅れる**ことがある**。(The bus is sometimes late.)
3. 日本語で夢を見る**ことがある**。(Sometimes I dream in Japanese.)
4. 疲れた日は、何もしないで寝る**ことがある**。(On tired days, I sometimes just go straight to sleep without doing anything.)
5. 朝、目覚まし時計が聞こえない**ことがある**。(Sometimes I don't hear my alarm in the morning.)
6. 週末は夜遅くまで起きている**ことがある**。(On weekends I sometimes stay up very late.)
:::

### 〜ことはない（没必要/不会……）||〜ことはない (No need to… / Won't happen)

:::zh
**接续：** 动词辞书形 + ことはない
:::

:::en
**Conjugation:** Verb dictionary form + ことはない
:::

### 例句

:::zh
1. そんなに心配する**ことはない**。（没必要那么担心。）
2. 急ぐ**ことはない**。ゆっくりでいい。（不用着急，慢慢来。）
3. わざわざ行く**ことはない**。電話でいい。（不必特意去，打电话就行。）
4. 一人で全部やる**ことはない**。手伝うよ。（不用一个人全做，我来帮你。）
5. そんなに怒る**ことはない**でしょう。（没必要那么生气吧。）
:::

:::en
1. そんなに心配する**ことはない**。(There's no need to worry so much.)
2. 急ぐ**ことはない**。ゆっくりでいい。(No need to rush. Take your time.)
3. わざわざ行く**ことはない**。電話でいい。(No need to make a special trip — a phone call is fine.)
4. 一人で全部やる**ことはない**。手伝うよ。(You don't have to do it all by yourself. I'll help.)
5. そんなに怒る**ことはない**でしょう。(There's probably no need to be that angry.)
:::

### 辨析：辞書形+ことがある vs た形+ことがある||Contrast: Dictionary form + ことがある vs た-form + ことがある

:::zh
| | 辞書形+ことがある | た形+ことがある |
|---|---|---|
| 意思 | 有时会…… | 曾经……过（经验） |
| 时态 | 习惯性、反复发生 | 过去的一次性经验 |
| 例句 | 朝ご飯を食べないことがある | 日本に行ったことがある |
| 翻译 | 有时候不吃早饭 | 去过日本 |

> **常见错误：** 把「辞書形+ことがある」和「た形+ことがある」搞混。前者是"有时会"，后者是"曾经有过……的经历"。
:::

:::en
| | Dictionary form + ことがある | た-form + ことがある |
|---|---|---|
| Meaning | Sometimes… | Have done… before (experience) |
| Tense/aspect | Habitual, recurring | One-time past experience |
| Example | 朝ご飯を食べないことがある | 日本に行ったことがある |
| Translation | I sometimes don't eat breakfast | I've been to Japan |

> **Common mistake**: Confusing 「辞書形+ことがある」with 「た形+ことがある」. The former means "sometimes," the latter means "have had the experience of."
:::

## 3. 〜のだ / 〜んだ（说明/强调/求解释）||3. 〜のだ / 〜んだ (Explanation / Emphasis / Seeking Explanation)

### 接続

:::zh
- 动词/い形容词 简体 + のだ（んだ）
- な形容词 + **な** + のだ（んだ）
- 名词 + **な** + のだ（んだ）
:::

:::en
- Verb / い-adjective plain form + のだ（んだ）
- な-adjective + **な** + のだ（んだ）
- Noun + **な** + のだ（んだ）
:::

### 含义

:::zh
「のだ」（口语：んだ，敬体：んです）是日语最重要的语气表达之一。核心有三大功能：
:::

:::en
「のだ」(casual: んだ, polite: んです) is one of the most important sentence-final expressions in Japanese. It has three core functions:
:::

#### ① 解释原因/背景||① Explaining Reason/Background

:::zh
给前面的话补充说明理由或背景信息。

1. A：どうして遅れたんですか。（为什么迟到了？）
   B：電車が止まった**んです**。（因为电车停了。→ 解释原因）
2. 今日は早く帰ります。明日試験がある**んです**。（今天早点回去。明天有考试。→ 补充说明理由）
3. すみません、ちょっと体調が悪い**んです**。（不好意思，身体有点不舒服。→ 解释）
4. 今日はパーティーに行けません。用事がある**んです**。（今天不能去派对。有事情。→ 说明理由）
5. あまり食べられません。お腹がいっぱいな**んです**。（吃不了太多。肚子饱了。→ 解释状况）
:::

:::en
Provides explanatory context or reason for a statement already made.

1. A：どうして遅れたんですか。(Why were you late?)
   B：電車が止まった**んです**。(Because the train stopped. → explaining the reason)
2. 今日は早く帰ります。明日試験がある**んです**。(I'm leaving early today. I have an exam tomorrow. → adding the reason)
3. すみません、ちょっと体調が悪い**んです**。(I'm sorry, I'm not feeling well. → explaining)
4. 今日はパーティーに行けません。用事がある**んです**。(I can't go to the party today. I have something to do. → giving a reason)
5. あまり食べられません。お腹がいっぱいな**んです**。(I can't eat much. I'm full. → explaining the situation)
:::

#### ② 发现/感叹||② Discovery / Exclamation

:::zh
看到某个情况时的发现或恍然大悟。

1. あ、雨が降っている**んだ**。（啊，原来在下雨啊。→ 刚发现）
2. そうだった**んだ**。（原来是这样啊。→ 恍然大悟）
3. へえ、日本語が上手な**んだ**。（哇，日语说得很好嘛。→ 发现/感叹）
4. もう12時な**んだ**。早いなあ。（已经12点了啊。时间过得好快。→ 刚注意到）
5. あ、ここにあった**んだ**。ずっと探していた。（啊，原来在这里。一直在找。→ 发现）
:::

:::en
Expressing a realization or moment of dawning understanding.

1. あ、雨が降っている**んだ**。(Oh, it's raining! → just noticed)
2. そうだった**んだ**。(Oh, I see! That's how it was. → sudden realization)
3. へえ、日本語が上手な**んだ**。(Wow, your Japanese is really good! → discovery/exclamation)
4. もう12時な**んだ**。早いなあ。(It's already noon! Time flies. → just noticed)
5. あ、ここにあった**んだ**。ずっと探していた。(Oh, here it is! I've been looking for it. → discovery)
:::

#### ③ 询问理由/追问||③ Asking for Explanation / Follow-up Questions

:::zh
看到某个异常情况，想了解原因。

1. どうした**んですか**。（怎么了？→ 看到异常情况追问）
2. 何を食べた**んですか**。（你吃了什么？→ 想了解情况）
3. どこに行く**んですか**。（你要去哪？→ 看到对方准备出门，追问）
4. なぜ泣いている**んですか**。（为什么在哭？→ 想知道原因）
5. 何があった**んですか**。（发生了什么事？→ 追问）
:::

:::en
Noticing something unusual and wanting to understand the reason.

1. どうした**んですか**。(What's the matter? → following up on something unusual)
2. 何を食べた**んですか**。(What did you eat? → trying to understand the situation)
3. どこに行く**んですか**。(Where are you going? → seeing someone heading out, asking)
4. なぜ泣いている**んですか**。(Why are you crying? → wanting to know the reason)
5. 何があった**んですか**。(What happened? → pressing for information)
:::

### んです vs です的区别||Difference between んです and です

:::zh
| | です | んです |
|---|---|---|
| 単純陳述 | 明日は休みです。 | — |
| 解释说明 | — | 明日は休みなんです。（所以……） |
| 语气 | 平淡叙述 | 有"请理解我的情况"的语气 |
| 追问 | 何を食べましたか。（普通提问） | 何を食べたんですか。（好奇追问） |

> **使用频率极高。** 日本人说话时大量使用んです，不用的话听起来会很生硬。建议多听多模仿。
:::

:::en
| | です | んです |
|---|---|---|
| Plain statement | 明日は休みです。 | — |
| Explanation | — | 明日は休みなんです。(That's why…) |
| Tone | Flat, neutral | "Please understand my situation" |
| Question | 何を食べましたか。(ordinary question) | 何を食べたんですか。(curious follow-up) |

> **Extremely high usage frequency.** Japanese speakers use んです constantly — without it, speech can sound stiff and blunt. It's well worth absorbing through lots of listening and imitation.
:::

## 4. 〜のに（转折 — 明明……却……）||4. 〜のに (Concessive — "Even though…")

### 接続

:::zh
- 动词/い形容词 简体 + のに
- な形容词+**な** + のに
- 名词+**な** + のに
:::

:::en
- Verb / い-adjective plain form + のに
- な-adjective + **な** + のに
- Noun + **な** + のに
:::

### 含义

:::zh
"明明……却……"，带有**不满、遗憾、意外**的语气。说话人觉得现实与预期不符。
:::

:::en
"Even though…" — carries a tone of **dissatisfaction, regret, or surprise**. The speaker feels reality doesn't match expectations.
:::

### 例句

:::zh
1. 約束した**のに**、来なかった。（明明约好了，却没来。→ 不满）
2. たくさん勉強した**のに**、試験に落ちた。（明明学了很多，却没考过。→ 遗憾）
3. 高い**のに**、美味しくない。（明明很贵，却不好吃。→ 不满）
4. 日曜日な**のに**、仕事がある。（明明是周日，却要工作。→ 遗憾）
5. まだ3月な**のに**、暑い。（明明才3月，就很热了。→ 意外）
6. 薬を飲んだ**のに**、まだ頭が痛い。（明明吃了药，头还是痛。→ 不满）
7. ダイエットしている**のに**、全然痩せない。（明明在减肥，却完全没瘦。→ 遗憾）
:::

:::en
1. 約束した**のに**、来なかった。(Even though we made a promise, they didn't come. → dissatisfaction)
2. たくさん勉強した**のに**、試験に落ちた。(Even though I studied so much, I failed the exam. → regret)
3. 高い**のに**、美味しくない。(Even though it's expensive, it's not good. → dissatisfaction)
4. 日曜日な**のに**、仕事がある。(Even though it's Sunday, I have to work. → regret)
5. まだ3月な**のに**、暑い。(Even though it's only March, it's hot. → surprise)
6. 薬を飲んだ**のに**、まだ頭が痛い。(Even though I took medicine, my head still hurts. → dissatisfaction)
7. ダイエットしている**のに**、全然痩せない。(Even though I'm dieting, I'm not losing any weight. → regret)
:::

### 辨析：のに vs けど/が||Contrast: のに vs けど/が

:::zh
| | のに | けど/が |
|---|---|---|
| 语气 | 不满/遗憾/意外 | 客观对比，中性 |
| 感情色彩 | 强（觉得不应该这样） | 弱 |
| 例句 | 雨が降ったのに、出かけた | 雨が降ったけど、出かけた |
| 翻译 | 明明下了雨，还出门了（不该啊） | 虽然下了雨，但出门了（就是这样） |

> **注意：** のに句尾不能接命令、请求、建议。
> - ✅ 雨が降っている**のに**、出かけた。
> - ❌ 雨が降っている**のに**、出かけてください。
>
> 如果要在转折后接请求，用けど：雨が降っている**けど**、出かけてください。
:::

:::en
| | のに | けど/が |
|---|---|---|
| Tone | Dissatisfaction / regret / surprise | Objective contrast, neutral |
| Emotional charge | Strong (this shouldn't be the case) | Weak |
| Example | 雨が降ったのに、出かけた | 雨が降ったけど、出かけた |
| Translation | Even though it rained, they went out (shouldn't have!) | It rained, but they went out (just stating facts) |

> **Note:** のに cannot be followed by commands, requests, or suggestions at the end of the sentence.
> - ✅ 雨が降っている**のに**、出かけた。
> - ❌ 雨が降っている**のに**、出かけてください。
>
> If you need a request after a concessive clause, use けど: 雨が降っている**けど**、出かけてください。
:::

## 5. 〜ので / 〜から（原因）||5. 〜ので / 〜から (Cause and Reason)

### 〜から（主観理由）||〜から (Subjective Reason)

:::zh
**接续：** 简体/敬体 + から

1. 暑い**から**、窓を開けてください。（因为热，请开窗。）
2. お腹が空いた**から**、何か食べたい。（因为肚子饿了，想吃点什么。）
3. 明日は休みです**から**、ゆっくりできます。（因为明天休息，可以放松。）
4. もう遅い**から**、帰ろう。（已经晚了，回去吧。）
5. 好きだ**から**、毎日練習している。（因为喜欢，所以每天练习。）
:::

:::en
**Conjugation:** Plain or polite form + から

1. 暑い**から**、窓を開けてください。(It's hot, so please open the window.)
2. お腹が空いた**から**、何か食べたい。(I'm hungry, so I want to eat something.)
3. 明日は休みです**から**、ゆっくりできます。(Tomorrow is a day off, so I can relax.)
4. もう遅い**から**、帰ろう。(It's already late, let's go home.)
5. 好きだ**から**、毎日練習している。(Because I love it, I practice every day.)
:::

### 〜ので（客观理由，更礼貌）||〜ので (Objective Reason, More Polite)

:::zh
**接续：** 简体 + ので（な形/名词：+**なので**）

1. 体調が悪い**ので**、早退してもいいですか。（因为身体不舒服，可以早退吗？）
2. 電車が遅れた**ので**、遅刻しました。（因为电车延误，迟到了。）
3. 明日は休みな**ので**、出かける予定です。（因为明天休息，打算出去。）
4. 雨が降っている**ので**、タクシーで行きましょう。（因为下雨，坐出租车去吧。）
5. まだ準備ができていない**ので**、少々お待ちください。（因为还没准备好，请稍等。）
6. 初めてな**ので**、よくわかりません。（因为是第一次，不太明白。）
:::

:::en
**Conjugation:** Plain form + ので (な-adjective / noun: + **なので**)

1. 体調が悪い**ので**、早退してもいいですか。(Because I'm not feeling well, may I leave early?)
2. 電車が遅れた**ので**、遅刻しました。(Because the train was delayed, I was late.)
3. 明日は休みな**ので**、出かける予定です。(Because tomorrow is a day off, I plan to go out.)
4. 雨が降っている**ので**、タクシーで行きましょう。(Since it's raining, let's go by taxi.)
5. まだ準備ができていない**ので**、少々お待ちください。(Since we're not ready yet, please wait a moment.)
6. 初めてな**ので**、よくわかりません。(Since it's my first time, I don't quite understand.)
:::

### から vs ので 对照表||Comparison: から vs ので

:::zh
| | から | ので |
|---|---|---|
| 语气 | 主观、直接 | 客观、委婉 |
| 礼貌度 | 普通 | 更礼貌 |
| 后接请求 | ✅可以 | ✅更适合 |
| 适用场合 | 日常/朋友 | 正式/对上级 |
| 接续 | 简体/敬体均可 | 一般接简体 |

> 向别人请求/道歉时，用ので更得体：
> - ✅ すみません、電車が遅れた**ので**…
> - △ すみません、電車が遅れた**から**…（稍显生硬）
>
> 强调个人意志/感情时，から更自然：
> - ✅ 絶対に行きたい**から**、お金を貯めている。（特别想去，所以在攒钱。）
> - △ 絶対に行きたい**ので**、お金を貯めている。（用ので语气偏弱）
:::

:::en
| | から | ので |
|---|---|---|
| Tone | Subjective, direct | Objective, gentle |
| Politeness | Standard | More polite |
| Followed by requests | ✅ acceptable | ✅ more appropriate |
| Setting | Casual / with friends | Formal / with superiors |
| Conjugation | Plain or polite form | Usually plain form |

> When making a request or apology to someone, ので is more appropriate:
> - ✅ すみません、電車が遅れた**ので**…
> - △ すみません、電車が遅れた**から**…(slightly blunt)
>
> When emphasizing personal feelings or will, から sounds more natural:
> - ✅ 絶対に行きたい**から**、お金を貯めている。(I really want to go, so I'm saving up.)
> - △ 絶対に行きたい**ので**、お金を貯めている。(ので weakens the personal resolve)
:::

## 6. 〜し（列举理由/并列）||6. 〜し (Listing Reasons / Parallel Points)

### 接続

:::zh
简体 + し
:::

:::en
Plain form + し
:::

### 含义

:::zh
"又……又……而且……"，列举多个理由或特征。可以只列一个し，表示"不只这个理由，还有别的"。
:::

:::en
"Not only… but also…" — lists multiple reasons or characteristics. Even with just one し, it implies "this is one reason among others."
:::

### 例句

:::zh
1. あの店は安い**し**、美味しい**し**、よく行きます。（那家店又便宜又好吃，常去。）
2. 今日は雨だ**し**、寒い**し**、出かけたくない。（今天又下雨又冷，不想出门。）
3. 彼は頭がいい**し**、スポーツもできる**し**、モテるだろう。（他又聪明又擅长运动，大概很受欢迎吧。）
4. 時間もない**し**、お金もない**し**、旅行は無理だ。（又没时间又没钱，旅行不可能。）
5. この部屋は広い**し**、駅に近い**し**、いい部屋だね。（这个房间又大又离车站近，是好房间啊。）
6. 日本語は面白い**し**、将来に役に立つ**し**、頑張ろう。（日语又有趣又对将来有用，加油吧。）
:::

:::en
1. あの店は安い**し**、美味しい**し**、よく行きます。(That shop is cheap and good, so I go there often.)
2. 今日は雨だ**し**、寒い**し**、出かけたくない。(It's raining and cold today, so I don't want to go out.)
3. 彼は頭がいい**し**、スポーツもできる**し**、モテるだろう。(He's smart and good at sports, so he's probably popular.)
4. 時間もない**し**、お金もない**し**、旅行は無理だ。(I have no time and no money, so a trip is impossible.)
5. この部屋は広い**し**、駅に近い**し**、いい部屋だね。(This room is spacious and close to the station — it's a great room.)
6. 日本語は面白い**し**、将来に役に立つ**し**、頑張ろう。(Japanese is both fun and useful for the future, so let's keep at it.)
:::

### 辨析：し vs から||Contrast: し vs から

:::zh
| | し | から |
|---|---|---|
| 理由数量 | 列举多个（暗示还有别的） | 通常一个明确理由 |
| 语气 | 含蓄、列举 | 直接 |
| 例句 | 安いし、美味しいし、よく行く | 安いから、よく行く |

> し还可以只说一个理由，暗示"除此之外还有别的理由"：
> 疲れた**し**、もう帰ろう。（累了（等等原因），回去吧。）
:::

:::en
| | し | から |
|---|---|---|
| Number of reasons | Multiple (implies there are more) | Usually one clear reason |
| Tone | Implicit, listing | Direct |
| Example | 安いし、美味しいし、よく行く | 安いから、よく行く |

> し can be used with just one reason, hinting "there are other reasons too":
> 疲れた**し**、もう帰ろう。(I'm tired (among other reasons), so let's go home.)
:::

## 7. 疑问词 + か/も/でも||7. Question Words + か/も/でも

### 疑問词 + か = "某……"（不確定）||Question word + か = "some…" (unspecified)

:::zh

|---|---|---|---|
| 誰か | someone | 誰かが来た (someone came) | 誰かに聞いてみよう (let me ask someone) |

| どこか | somewhere | どこかに行きたい (want to go somewhere) | どこかで会った気がする (I feel like we've met somewhere) |
| いつか | someday | いつか日本に行きたい (someday I want to go to Japan) | いつか必ずできるようになる (someday I'll definitely be able to do it) |

### 疑問词 + も + 否定 = "全否定"||Question word + も + negative = "total negation"

:::zh
| | 表达 | 含义 | 例句 |
|---|---|---|---|
| 誰も〜ない | 谁都不 | 誰も来なかった（谁都没来） | 誰も答えがわからない（谁都不知道答案） |

:::en
| | Expression | Meaning | Example |
|---|---|---|---|
| 誰も〜ない | nobody | 誰も来なかった (nobody came) | 誰も答えがわからない (nobody knows the answer) |
| 何も〜ない | nothing | 何も食べなかった (didn't eat anything) | 何も聞こえない (can't hear anything) |

| いつも | ⚠️ always (NOT negative!) | いつも遅刻する (always late) | いつも優しい (always kind) |

> **Special note about いつも!** All other "question word + も + negative" patterns are total negations, but いつも means "always/all the time" (affirmative) — it is an exception!
:::

### 疑問词 + でも = "無論……都"||Question word + でも = "no matter… / any…"

| 何でも | anything | 何でも食べる (eats anything) | 何でも聞いてください (feel free to ask anything) |
| どこでも | anywhere | どこでも寝られる (can sleep anywhere) | どこでもWi-Fiが使える (Wi-Fi available everywhere) |

## 8. 〜すぎる（過度）||8. 〜すぎる (Excess)

### 接続

:::zh
- 动词ます形（去ます）+ すぎる
- い形容词（去い）+ すぎる
- な形容词 语干 + すぎる
:::

:::en
- Verb ます-stem + すぎる
- い-adjective (drop い) + すぎる
- な-adjective stem + すぎる
:::

### 含义

:::zh
"太……了"，表示程度过度，通常含有**负面评价**的语气。
:::

:::en
"Too…" — expresses that something is excessive, usually with a **negative connotation**.
:::

### 例句

:::zh
1. 昨日飲み**すぎた**。（昨天喝太多了。）
2. 食べ**すぎて**、お腹が痛い。（吃太多了，肚子疼。）
3. この問題は難し**すぎる**。（这道题太难了。）
4. 高**すぎます**。もう少し安いのはありますか。（太贵了。有便宜点的吗？）
5. 静か**すぎて**、怖い。（太安静了，有点怕。）
6. 昨日ゲームをやり**すぎて**、寝不足だ。（昨天打游戏打太多，睡眠不足。）
7. この映画は長**すぎる**。3時間もある。（这部电影太长了。有3个小时。）
:::

:::en
1. 昨日飲み**すぎた**。(I drank too much yesterday.)
2. 食べ**すぎて**、お腹が痛い。(I ate too much and my stomach hurts.)
3. この問題は難し**すぎる**。(This problem is too difficult.)
4. 高**すぎます**。もう少し安いのはありますか。(It's too expensive. Do you have anything a bit cheaper?)
5. 静か**すぎて**、怖い。(It's so quiet it's scary.)
6. 昨日ゲームをやり**すぎて**、寝不足だ。(I played games too much yesterday and didn't get enough sleep.)
7. この映画は長**すぎる**。3時間もある。(This movie is too long — it's 3 hours!)
:::

### 辨析：すぎる vs とても||Contrast: すぎる vs とても

:::zh
| | すぎる | とても |
|---|---|---|
| 语气 | 超过了限度（负面） | 程度很高（中性/正面） |
| 例句 | 高すぎる（太贵了→不好） | とても高い（非常贵→描述事实） |
| 能否正面用 | 基本负面 | 可正面可负面 |

> すぎる本身也是动词（二类动词），可以变形：すぎた、すぎて、すぎない……
:::

:::en
| | すぎる | とても |
|---|---|---|
| Tone | Exceeds a limit (negative) | High degree (neutral/positive) |
| Example | 高すぎる (too expensive → bad) | とても高い (very expensive → just a description) |
| Positive use? | Basically negative | Can be positive or negative |

> すぎる is itself a verb (Group 2) and can conjugate: すぎた, すぎて, すぎない…
:::

5. 壊れ**やすい**から、気をつけて。(It breaks easily, so be careful.)

| 级别 | N5 | N3 |

## 10. 〜ながら（同時）||10. 〜ながら (Simultaneous Actions)

### 接続

:::zh
动词ます形（去ます）+ ながら
:::

:::en
Verb ます-stem + ながら
:::

### 含义

:::zh
"一边做A一边做B"，两个动作同时进行。**后面的动作是主要动作。**
:::

:::en
"While doing A, do B" — two actions happening at the same time. **The action after ながら is the main/primary action.**
:::

### 例句

:::zh
1. 音楽を聴き**ながら**、勉強する。（一边听音乐一边学习。→ 主要在学习）
2. 歩き**ながら**、スマホを見ないでください。（请不要边走边看手机。）
3. コーヒーを飲み**ながら**、新聞を読む。（边喝咖啡边看报纸。）
4. 笑い**ながら**、話した。（笑着说了。）
5. テレビを見**ながら**、ご飯を食べる。（边看电视边吃饭。）
6. 友達と話し**ながら**、歩いて帰った。（一边和朋友聊天一边走回家。）
7. 料理を作り**ながら**、歌を歌う。（一边做饭一边唱歌。）
:::

:::en
1. 音楽を聴き**ながら**、勉強する。(I study while listening to music. → main action is studying)
2. 歩き**ながら**、スマホを見ないでください。(Please don't look at your phone while walking.)
3. コーヒーを飲み**ながら**、新聞を読む。(I read the newspaper while drinking coffee.)
4. 笑い**ながら**、話した。(They spoke with a smile / while laughing.)
5. テレビを見**ながら**、ご飯を食べる。(I eat while watching TV.)
6. 友達と話し**ながら**、歩いて帰った。(I walked home while chatting with a friend.)
7. 料理を作り**ながら**、歌を歌う。(I sing while cooking.)
:::

### 辨析：ながら vs ている||Contrast: ながら vs ている

:::zh
| | ながら | ている |
|---|---|---|
| 功能 | 两个动作同时进行 | 一个动作的进行状态 |
| 主语 | 必须相同（同一个人） | 不限 |
| 例句 | 食べながら話す（边吃边说） | 食べている（正在吃） |
| 注意 | 不能用于两个不同主语 | — |

> **ながら的主语必须相同：**
> - ✅ 私は音楽を聴きながら勉強する。（我边听音乐边学习）
> - ❌ 私が音楽を聴きながら、彼が勉強する。（主语不同，不能用ながら）
:::

:::en
| | ながら | ている |
|---|---|---|
| Function | Two simultaneous actions | One action in progress |
| Subject | Must be the same person | Not restricted |
| Example | 食べながら話す (talk while eating) | 食べている (is eating) |
| Note | Cannot be used with two different subjects | — |

> **ながら requires the same subject:**
> - ✅ 私は音楽を聴きながら勉強する。(I study while listening to music.)
> - ❌ 私が音楽を聴きながら、彼が勉強する。(Different subjects — cannot use ながら.)
:::

Verb ます-stem + 方（かた）

### 含义

:::zh

:::

:::en
"The way to do…" / "How to do…" — nominalizes the action; the resulting noun can take particles like の, が, を, etc.
:::

### 例句

:::zh
1. この漢字の読み**方**を教えてください。（请教我这个汉字的读法。）

3. 作り**方**は簡単です。（做法很简单。）
4. 行き**方**を教えてもらえますか。（能告诉我怎么去吗？）
5. 食べ**方**が汚い。（吃相难看。）
6. 日本語の勉強の仕**方**を変えたい。（想改变日语的学习方法。）
7. この機械の動かし**方**を教えてください。（请教我这个机器的使用方法。）
:::

:::en
1. この漢字の読み**方**を教えてください。(Please teach me how to read this kanji.)
2. 使い**方**がわからない。(I don't know how to use it.)
3. 作り**方**は簡単です。(The way to make it is simple.)
4. 行き**方**を教えてもらえますか。(Could you tell me how to get there?)
5. 食べ**方**が汚い。(Your table manners are bad. / The way you eat is messy.)
6. 日本語の勉強の仕**方**を変えたい。(I want to change the way I study Japanese.)
7. この機械の動かし**方**を教えてください。(Please show me how to operate this machine.)
:::

### 辨析：方（かた）vs 方法（ほうほう）||Contrast: 〜方（かた）vs 方法（ほうほう）

:::zh
| | 〜方（かた） | 方法（ほうほう） |
|---|---|---|
| 词性 | 接尾词（接动词后） | 名词（独立使用） |
| 用法 | 読み方、使い方 | 勉強の方法、解決の方法 |
| 语感 | 口语常用 | 稍微正式 |
:::

:::en
| | 〜方（かた）| 方法（ほうほう）|
|---|---|---|
| Word class | Suffix (attaches to verb) | Independent noun |
| Usage | 読み方、使い方 | 勉強の方法、解決の方法 |
| Register | Commonly used in speech | Slightly more formal |
:::

### 二、ことがある vs ことはない：选择正确的表达

1. 疲れた日は夕飯を作らない（ことがある / ことはない）。→ 有时候不做
2. そんなに怒る（ことがある / ことはない）よ。→ 没必要
3. 彼は時々遅刻する（ことがある / ことはない）。→ 有时候
4. 一人で行く（ことがある / ことはない）。私が一緒に行くから。→ 不用

<details>
<summary>答案</summary>

:::zh
1. ことがある（有时候累了不做晚饭）
2. ことはない（没必要那么生气）
3. ことがある（他有时候会迟到）
4. ことはない（不用一个人去，我和你一起）
:::

:::en
1. ことがある (sometimes too tired to make dinner)
2. ことはない (no need to be that angry)
3. ことがある (he is sometimes late)
4. ことはない (no need to go alone — I'll come with you)
:::

</details>

### 三、のだ/んだ：用んです改写

1. 明日は来られません。用事があります。→ 用事が＿＿＿。

3. 元気がないですね。何かありましたか。→ 何か＿＿＿。

<details>
<summary>答案</summary>

2. お酒を飲んだんです。（解释原因）
3. 何かあったんですか。（追问理由）
:::

:::en
1. 用事があるんです。(explaining the reason)
2. お酒を飲んだんです。(explaining the reason)
3. 何かあったんですか。(asking for an explanation)
:::

</details>

### 四、のに vs けど：根据语境选択

1. 薬を飲んだ（のに / けど）、まだ治らない。→ 说话人不満
2. 雨が降った（のに / けど）、試合はあった。→ 客观叙述
3. ダイエット中な（のに / けど）、ケーキを食べてしまった。→ 后悔
4. 日本語は難しい（のに / けど）、面白い。→ 中性転折

<details>
<summary>答案</summary>

:::zh
1. のに（明明吃了药还没好→不满）
2. けど（虽然下了雨但比赛照常→客观）
3. のに（明明在减肥却吃了蛋糕→后悔）
4. けど（虽然难但有趣→中性对比）
:::

:::en
1. のに (even though I took medicine, not better yet → dissatisfaction)
2. けど (it rained but the match went ahead → neutral)
3. のに (even though I'm dieting, I ate the cake → regret)
4. けど (Japanese is hard but interesting → neutral contrast)
:::

</details>

### 五、ので vs から：选択更合適的

1. 先生、体調が悪い（ので / から）、早退してもいいですか。→ 対老师
2. 疲れた（ので / から）、もう寝よう。→ 自言自語
3. お客様、ただいま満席な（ので / から）、少々お待ちください。→ 対客人
4. 嫌いだ（ので / から）、食べない。→ 个人意志

<details>
<summary>答案</summary>

:::zh
1. ので（对老师，用更礼貌的ので）
2. から（自言自语，主观感受用から更自然）
3. ので（对客人，正式场合用ので）
4. から（强调个人好恶，から更自然）
:::

:::en
1. ので (talking to a teacher — the more polite ので is better)
2. から (talking to yourself — subjective feeling makes から more natural)
3. ので (talking to a customer — formal setting calls for ので)
4. から (expressing personal preference — から feels more natural)
:::

</details>

### 七、疑問词+か/も/でも：填空

1. ＿＿＿が来たみたいだ。（有人好像来了）
2. 昨日は＿＿＿食べなかった。（什么都没吃）
3. この問題は＿＿＿できるよ。（谁都能做）
4. ＿＿＿に忘れ物をした。（在某个地方忘了东西）
5. ＿＿＿行きたくない。（哪里都不想去）
6. ＿＿＿いいですよ。（什么时候都行）

<details>
<summary>答案</summary>

:::zh
1. 誰か（某人→不确定）
2. 何も（什么都→全否定）
3. 誰でも（谁都→无论谁）
4. どこか（某处→不确定）
5. どこにも（哪里都→全否定）
6. いつでも（什么时候都→无论何时）
:::

:::en
1. 誰か (someone → unspecified)
2. 何も (nothing → total negation)
3. 誰でも (anyone → no matter who)
4. どこか (somewhere → unspecified)
5. どこにも (nowhere → total negation)
6. いつでも (anytime → no matter when)
:::

</details>

</details>

### 十二、综合問題：选择合適的語法填空

1. せっかく作った＿＿＿、誰も食べなかった。（のに / ので）
2. 日本語は難しい＿＿＿、楽しい＿＿＿、毎日勉強している。（し / から）
3. あの映画は長＿＿＿。3時間もあった。（すぎる / やすい）
4. 料理を作り＿＿＿、音楽を聴いている。（ながら / 方）
5. 明日試験がある＿＿＿、今日は早く寝ます。（ので / のに）
6. ＿＿＿でもいいから、何か食べたい。（何 / 誰）
7. この字の読み＿＿＿がわからない。（方 / やすい）
8. そんなに心配する＿＿＿。大丈夫だよ。（ことがある / ことはない）

<details>
<summary>答案</summary>

:::zh
1. のに（明明特意做了却没人吃→不满）
2. し、し（又难又有趣→列举多个理由）
3. すぎる（太长了→过度）
4. ながら（一边做饭一边听音乐→同时）
5. ので（因为明天有考试→解释原因）
6. 何（什么都行→无论什么）
7. 方（读法→方法）
8. ことはない（没必要→没必要那么担心）
:::

:::en
1. のに (even though I made it specially, no one ate it → dissatisfaction)
2. し、し (both difficult and fun → listing multiple reasons)
3. すぎる (too long → excessive)
4. ながら (listening to music while cooking → simultaneous)
5. ので (because there's an exam tomorrow → explaining reason)
6. 何 (anything will do → no matter what)
7. 方 (the reading → method)
8. ことはない (no need → no need to worry so much)
:::

</details>
