## 1. 可能形（能力/可能）||1. Potential Form (Ability/Possibility)

### 含义

:::zh
"能做……"、"可以做……"，表示能力或客观上的可能性。
:::

:::en
"Can do…", "It is possible to do…" — expresses ability or objective possibility.
:::

### 变形规则||Conjugation Rules

:::zh
| 类别 | 规则 | 例子 |
|------|------|------|
| 一类动词 | う段→え段+る | 書く→書**ける**、飲む→飲**める**、話す→話**せる** |
| 二类动词 | る→**られる** | 食べる→食べ**られる**、見る→見**られる** |
| する | **できる** | する→できる |
| 来る | **来られる**（こられる） | 来る→来られる |
:::

:::en
| Type | Rule | Example |
|------|------|---------|
| Group 1 verbs | う-row → え-row + る | 書く→書**ける**、飲む→飲**める**、話す→話**せる** |
| Group 2 verbs | る→**られる** | 食べる→食べ**られる**、見る→見**られる** |
| する | **できる** | する→できる |
| 来る | **来られる**（こられる） | 来る→来られる |
:::

### 一类动词可能形 全表

| 辞书形 | 可能形 |
|--------|--------|
| 書く | 書ける |
| 泳ぐ | 泳げる |
| 話す | 話せる |
| 待つ | 待てる |
| 死ぬ | 死ねる |
| 遊ぶ | 遊べる |
| 飲む | 飲める |
| 帰る（一类）| 帰れる |
| 買う | 買える |

:::zh
> **记忆法**：一类动词可能形 = 假定形去ば+る。書けば→書ける、飲めば→飲める。
:::

:::en
> **Memory tip**: Group 1 potential form = conditional form minus ば + る. 書けば→書ける、飲めば→飲める.
:::

### 二类动词的「ら抜き」现象||The ら-Drop Phenomenon in Group 2 Verbs

:::zh
正式：食べ**られる**、見**られる**
口语：食べ**れる**、見**れる**（省略了ら）

> 「ら抜き言葉」在日常会话中非常普遍，年轻人几乎都这么说。但考试和正式场合用「られる」。
:::

:::en
Formal: 食べ**られる**、見**られる**
Colloquial: 食べ**れる**、見**れる** (ら dropped)

> "ら-drop" (ら抜き言葉) is extremely common in everyday conversation — nearly all young speakers use it. Use 「られる」 in exams and formal settings.
:::

### 可能形的助词变化（を→が）||Particle Shift in Potential Form (を→が)

:::zh
原句用を的宾语，可能形中通常变成が：

- 日本語**を**話す → 日本語**が**話せる（能说日语）
- 寿司**を**食べる → 寿司**が**食べられる（能吃寿司）
- 漢字**を**読む → 漢字**が**読める（能读汉字）

> 实际使用中を也可以，但が更标准。考试建议用が。
:::

:::en
Objects marked with を in the plain sentence typically shift to が in the potential form:

- 日本語**を**話す → 日本語**が**話せる (can speak Japanese)
- 寿司**を**食べる → 寿司**が**食べられる (can eat sushi)
- 漢字**を**読む → 漢字**が**読める (can read kanji)

> を is also acceptable in practice, but が is more standard. Use が in exams.
:::

### 例句

:::zh
1. 日本語**が**話**せます**か？（你能说日语吗？）
2. この漢字**が**読**めない**。（这个汉字读不了。）
3. 明日は来**られます**か？（明天能来吗？）
4. お箸で食べ**られる**？（能用筷子吃吗？→ 口语）
5. ここからは富士山**が**見**える**。（从这里能看到富士山。）
6. 朝早く起き**られない**。（早上起不来。）
7. 一人で**できます**か？（一个人能做吗？）
8. 彼は３ヶ国語**が**話**せる**。（他能说三国语言。）
:::

:::en
1. 日本語**が**話**せます**か？(Can you speak Japanese?)
2. この漢字**が**読**めない**。(I can't read this kanji.)
3. 明日は来**られます**か？(Can you come tomorrow?)
4. お箸で食べ**られる**？(Can you eat with chopsticks? → casual)
5. ここからは富士山**が**見**える**。(You can see Mt. Fuji from here.)
6. 朝早く起き**られない**。(I can't get up early in the morning.)
7. 一人で**できます**か？(Can you do it by yourself?)
8. 彼は３ヶ国語**が**話**せる**。(He can speak three languages.)
:::

### 辨析：見える/聞こえる vs 見られる/聞ける||Contrast: 見える/聞こえる vs 見られる/聞ける

:::zh
| | 自然进入感官 | 主动去做 |
|---|---|---|
| 看 | 見**える**（看得见） | 見**られる**（能去看） |
| 听 | 聞こ**える**（听得见） | 聞**ける**（能去听） |

- 富士山が**見える**。（能看见富士山。→ 自然映入眼帘）
- 映画が**見られる**。（能看电影。→ 有条件/有机会去看）
- 鳥の声が**聞こえる**。（听得见鸟叫。→ 声音自然传来）
- ラジオが**聞ける**。（能听收音机。→ 可以去听）
- 隣の部屋の音楽が**聞こえる**。（隔壁的音乐听得见。→ 不想听也传过来）
- このアプリで音楽が**聞ける**。（用这个APP能听音乐。→ 有能力/条件去听）

> **判断技巧**：自己有没有主动意图？没有→見える/聞こえる；有→見られる/聞ける。
:::

:::en
| | Naturally enters perception | Active intention to do |
|---|---|---|
| Seeing | 見**える** (can be seen) | 見**られる** (able to go watch) |
| Hearing | 聞こ**える** (can be heard) | 聞**ける** (able to listen) |

- 富士山が**見える**。(Mt. Fuji is visible. → enters view naturally)
- 映画が**見られる**。(I can watch the movie. → have the means/opportunity)
- 鳥の声が**聞こえる**。(I can hear the birds. → sound reaches me naturally)
- ラジオが**聞ける**。(I can listen to the radio. → have the ability/access)
- 隣の部屋の音楽が**聞こえる**。(I can hear the music from the next room. → comes through whether I want it or not)
- このアプリで音楽が**聞ける**。(I can listen to music with this app. → have the means)

> **Key tip**: Is there active intention? No → 見える/聞こえる; Yes → 見られる/聞ける.
:::

## 2. 受身形（被动态）||2. Passive Form

### 含義

:::zh
"被……做"，表示主语是动作的承受者。
:::

:::en
"To be done to…" — the subject is the recipient of the action.
:::

### 变形规则||Conjugation Rules

:::zh
| 类别 | 规则 | 例子 |
|------|------|------|
| 一类动词 | う段→あ段+れる | 書く→書**かれる**、飲む→飲**まれる** |
| 二类动词 | る→**られる** | 食べる→食べ**られる**、見る→見**られる** |
| する | **される** | する→される |
| 来る | **来られる**（こられる） | 来る→来られる |

> 注意：二类动词的可能形和受身形**形式完全一样**（食べられる），要靠语境区分。
:::

:::en
| Type | Rule | Example |
|------|------|---------|
| Group 1 verbs | う-row → あ-row + れる | 書く→書**かれる**、飲む→飲**まれる** |
| Group 2 verbs | る→**られる** | 食べる→食べ**られる**、見る→見**られる** |
| する | **される** | する→される |
| 来る | **来られる**（こられる） | 来る→来られる |

> Note: Group 2 potential and passive forms are **identical** (食べられる) — context determines the meaning.
:::

### 一类动词受身形 全表

| 辞书形 | 受身形 |
|--------|--------|
| 書く | 書かれる |
| 泳ぐ | 泳がれる |
| 話す | 話される |
| 待つ | 待たれる |
| 遊ぶ | 遊ばれる |
| 飲む | 飲まれる |
| 帰る | 帰られる |
| 買う | 買われる |

:::zh
> **记忆法**：受身形 = ない形去ない+れる。書かない→書かれる、飲まない→飲まれる。
:::

:::en
> **Memory tip**: Passive form = ない-form minus ない + れる. 書かない→書かれる、飲まない→飲まれる.
:::

### 受身的三种类型||Three Types of Passive

#### ① 直接被动（直接受身）

:::zh
主语直接承受动作，对应中文的"被"。结构：**AはBに＋受身形**

1. 先生**に**褒め**られた**。（被老师表扬了。）
2. 彼は警察**に**捕まえ**られた**。（他被警察抓了。）
3. この本は多くの人**に**読ま**れている**。（这本书被很多人读着。）
4. この歌は世界中で歌わ**れている**。（这首歌在全世界被传唱。）
5. 彼女は皆**に**愛さ**れている**。（她被大家喜爱。）

> **直接被动不一定是坏事**：褒められた（被表扬）、愛されている（被爱）都是正面的。
:::

:::en
The subject directly receives the action, equivalent to "to be done" in English. Structure: **AはBに＋passive form**

1. 先生**に**褒め**られた**。(I was praised by the teacher.)
2. 彼は警察**に**捕まえ**られた**。(He was caught by the police.)
3. この本は多くの人**に**読ま**れている**。(This book is being read by many people.)
4. この歌は世界中で歌わ**れている**。(This song is sung all around the world.)
5. 彼女は皆**に**愛さ**れている**。(She is loved by everyone.)

> **Direct passive isn't always negative**: 褒められた (was praised) and 愛されている (is loved) are both positive.
:::

#### ② 间接被动（迷惑の受身）— 日语特色！||② Indirect Passive (Nuisance Passive) — Unique to Japanese!

:::zh
表示"别人做了某事，我因此受到了（不好的）影响"。中文翻不出来的"被"。

1. 雨**に**降**られた**。（被雨淋了。→ 下雨这件事让我困扰）
2. 電車の中で足を踏ま**れた**。（在电车里脚被踩了。）
3. 隣の人**に**タバコを吸わ**れて**、嫌だった。（旁边的人抽烟，我很不爽。）
4. 彼女**に**泣か**れて**、困った。（她哭了，我很为难。→ 她哭这件事让我困扰）
5. 夜中に赤ちゃん**に**泣か**れて**、眠れなかった。（半夜被孩子哭得睡不着。）
6. 友達にケーキを食べ**られた**。（蛋糕被朋友吃了。→ 我的蛋糕没了，很烦）

> **重点**：间接被动带有"我是受害者/我很困扰"的语气。这是日语独有的表达，中文没有对应。
> 即使原本不及物动词（降る、泣く、死ぬ）也能用间接被动——只要"我"受到了影响。
:::

:::en
Expresses "someone did something, and as a result I was adversely affected." There is no direct equivalent in English or Chinese.

1. 雨**に**降**られた**。(The rain came down on me. → The rain caused me inconvenience.)
2. 電車の中で足を踏ま**れた**。(My foot was stepped on in the train.)
3. 隣の人**に**タバコを吸わ**れて**、嫌だった。(The person next to me smoked, and I was bothered.)
4. 彼女**に**泣か**れて**、困った。(She cried on me and I didn't know what to do. → Her crying inconvenienced me.)
5. 夜中に赤ちゃん**に**泣か**れて**、眠れなかった。(The baby cried in the middle of the night and I couldn't sleep.)
6. 友達にケーキを食べ**られた**。(My friend ate my cake. → I'm annoyed the cake is gone.)

> **Key point**: The indirect passive carries a "I'm the victim / I'm troubled by this" nuance. This is unique to Japanese with no direct equivalent in Chinese.
> Even intransitive verbs (降る, 泣く, 死ぬ) can appear in the indirect passive — as long as "I" am affected.
:::

#### ③ 持ち主の受身（所有者被动）||③ Possessor Passive

:::zh
身体部位或所有物受到动作，日语把"所有者"变成主语：

1. 足を踏ま**れた**。（脚被踩了。→ 主语是"我"）
2. 財布を盗ま**れた**。（钱包被偷了。→ 主语是"我"）
3. 先生に名前を呼ば**れた**。（被老师叫了名字。）
4. 知らない人**に**肩を叩か**れた**。（被陌生人拍了肩膀。）
5. 弟**に**パソコンを壊さ**れた**。（电脑被弟弟弄坏了。）

> 中文说"我的脚被踩了"，日语说"我**被**踩了脚"。所有者做主语，所有物用を。
:::

:::en
When a body part or possession is acted upon, Japanese makes the owner the subject:

1. 足を踏ま**れた**。(My foot was stepped on. → subject is "I")
2. 財布を盗ま**れた**。(My wallet was stolen. → subject is "I")
3. 先生に名前を呼ば**れた**。(My name was called by the teacher.)
4. 知らない人**に**肩を叩か**れた**。(A stranger tapped me on the shoulder.)
5. 弟**に**パソコンを壊さ**れた**。(My computer was broken by my younger brother.)

> Chinese says "my foot was stepped on," but Japanese says "I was stepped on [as to] my foot." The owner is the subject; the body part/possession takes を.
:::

## 3. 可能形 vs 受身形 对照||3. Potential vs Passive Comparison

:::zh
| | 可能形 | 受身形 |
|---|---|---|
| 一类动词 | う→え段+る（書**ける**） | う→あ段+れる（書**かれる**） |
| 二类动词 | 食べ**られる** | 食べ**られる** |
| する | **できる** | **される** |
| 来る | 来**られる** | 来**られる** |
:::

:::en
| | Potential Form | Passive Form |
|---|---|---|
| Group 1 verbs | う-row → え-row + る (書**ける**) | う-row → あ-row + れる (書**かれる**) |
| Group 2 verbs | 食べ**られる** | 食べ**られる** |
| する | **できる** | **される** |
| 来る | 来**られる** | 来**られる** |
:::

### 辨析：二类动词的「られる」——可能还是受身？||Distinguishing Group 2 「られる」: Potential or Passive?

:::zh
二类动词和来る的可能形与受身形同形，靠以下线索判断：

| 线索 | 可能（能做） | 受身（被做） |
|------|-------------|-------------|
| 助词 | 〜**が**できる/られる | 〜**を**〜られた |
| 语气 | 中性/积极 | 常带迷惑感 |
| 被动者 | 无"に"标记动作者 | 常有"Bに" |

- ケーキ**が**食べられる。（能吃蛋糕 → 可能）
- ケーキ**を**食べられた。（蛋糕被（人）吃了 → 受身/被害）
- この映画**が**見られる。（这部电影能看 → 可能）
- 皆**に**見られて恥ずかしかった。（被大家看了很害羞 → 受身）

> **实用技巧**：口语中可能形用「ら抜き」（食べれる、見れる），这样就和受身形区分开了。这也是ら抜き流行的原因之一。
:::

:::en
Group 2 verbs and 来る have identical potential and passive forms — use these cues to tell them apart:

| Cue | Potential (can do) | Passive (being done to) |
|-----|--------------------|------------------------|
| Particle | 〜**が** + られる | 〜**を** + られた |
| Tone | Neutral/positive | Often conveys annoyance |
| Agent marker | No "Bに" | Often has "Bに" |

- ケーキ**が**食べられる。(Can eat cake → potential)
- ケーキ**を**食べられた。(The cake was eaten [by someone] → passive/nuisance)
- この映画**が**見られる。(Can watch this film → potential)
- 皆**に**見られて恥ずかしかった。(Was watched by everyone, felt embarrassed → passive)

> **Practical tip**: In casual speech, potential uses ら-drop (食べれる、見れる), clearly separating it from the passive. This is actually one reason the ら-drop form became so widespread.
:::

## 4. 意向形（意志/邀请）||4. Volitional Form (Intent/Invitation)

### 含义

:::zh
"我要做……" / "一起做……吧"，表示说话人的意志或邀请。
:::

:::en
"I'm going to…" / "Let's…" — expresses the speaker's will or an invitation.
:::

### 变形规则||Conjugation Rules

:::zh
| 类别 | 规则 | 例子 |
|------|------|------|
| 一类动词 | う段→お段+う | 書く→書**こう**、飲む→飲**もう**、行く→行**こう** |
| 二类动词 | る→**よう** | 食べる→食べ**よう**、見る→見**よう** |
| する | **しよう** | する→しよう |
| 来る | **来よう**（こよう） | 来る→来よう |
:::

:::en
| Type | Rule | Example |
|------|------|---------|
| Group 1 verbs | う-row → お-row + う | 書く→書**こう**、飲む→飲**もう**、行く→行**こう** |
| Group 2 verbs | る→**よう** | 食べる→食べ**よう**、見る→見**よう** |
| する | **しよう** | する→しよう |
| 来る | **来よう**（こよう） | 来る→来よう |
:::

### 一类动词意向形 全表

| 辞书形 | 意向形 |
|--------|--------|
| 書く | 書こう |
| 泳ぐ | 泳ごう |
| 話す | 話そう |
| 待つ | 待とう |
| 死ぬ | 死のう |
| 遊ぶ | 遊ぼう |
| 飲む | 飲もう |
| 帰る | 帰ろう |
| 買う | 買おう |

### 用法||Usage

#### ① 自己的意志决定（"我要……"）||① Speaker's Own Decision ("I'm going to…")

:::zh
1. そろそろ帰**ろう**。（差不多该回去了。）
2. 明日から毎日運動し**よう**。（从明天起每天运动。）
3. もう寝**よう**。（该睡了。）
4. よし、今日は頑張**ろう**。（好，今天加油。）
5. 来年こそN2に合格し**よう**。（明年一定要过N2。）
:::

:::en
1. そろそろ帰**ろう**。(I should head home soon.)
2. 明日から毎日運動し**よう**。(I'll exercise every day starting tomorrow.)
3. もう寝**よう**。(Time to sleep.)
4. よし、今日は頑張**ろう**。(Alright, I'll give it my all today.)
5. 来年こそN2に合格し**よう**。(I'll definitely pass N2 next year.)
:::

#### ② 邀请/提议（"一起……吧"）||② Invitation/Suggestion ("Let's…")

:::zh
1. 一緒に食べ**よう**。（一起吃吧。→ 简体/朋友间）

:::en

#### ③ 〜（よ）うと思う / 〜（よ）うと思っている（"我打算……"）||③ 〜（よ）うと思う / 〜（よ）うと思っている ("I'm thinking of…")

:::zh
意向形+と思う = 表达打算/计划

1. 来年日本に行**こうと思っています**。（我打算明年去日本。）
2. 転職し**ようと思っている**。（在考虑换工作。）
3. 明日早く起き**ようと思う**。（打算明天早起。）
4. 週末は料理を作**ろうと思う**。（周末打算做饭。）
5. もっと日本語を勉強し**ようと思っている**。（想着要更努力学日语。）

> **〜ようと思う vs 〜ようと思っている**：
> - と思う = 刚刚决定、临时起意
> - と思っている = 一直在考虑/已经决定一段时间了

:::en
Volitional form + と思う = express a plan or intention

1. 来年日本に行**こうと思っています**。(I'm planning to go to Japan next year.)

3. 明日早く起き**ようと思う**。(I'm thinking of getting up early tomorrow.)

5. もっと日本語を勉強し**ようと思っている**。(I've been thinking I should study Japanese harder.)

> **〜ようと思う vs 〜ようと思っている**:
> - と思う = just decided, spur-of-the-moment
> - と思っている = have been thinking about it / decided some time ago

## 5. 〜つもりだ（打算/计划）||5. 〜つもりだ (Intention/Plan)

### 接续

:::zh
- 动词辞书形 + つもりだ（打算做）
- 动词ない形 + つもりだ（打算不做）
:::

:::en
- Verb dictionary form + つもりだ (intend to do)
- Verb ない-form + つもりだ (intend not to do)
:::

### 含义

:::zh
"打算做……"，表示已经确定的意图/计划。比「〜ようと思う」更确定。
:::

:::en
"Plan to do…" — expresses a firm intention or plan. More definite than 「〜ようと思う」.
:::

### 例句

:::zh
1. 来月引っ越す**つもりです**。（下个月打算搬家。）
2. 大学院に行く**つもりだ**。（打算读研。）
3. もう二度とあの店には行かない**つもりだ**。（打算再也不去那家店了。）
4. 夏休みに何をする**つもりですか**。（暑假打算做什么？）
5. 彼と結婚する**つもりはない**。（没打算和他结婚。→ つもりはない=完全没这个打算）
6. 今年中にN2に受かる**つもりだ**。（打算今年内考过N2。）
7. お酒はもう飲まない**つもりだ**。（打算不再喝酒了。）
:::

:::en
1. 来月引っ越す**つもりです**。(I'm planning to move next month.)
2. 大学院に行く**つもりだ**。(I intend to go to graduate school.)
3. もう二度とあの店には行かない**つもりだ**。(I'm never going to that shop again.)
4. 夏休みに何をする**つもりですか**。(What are you planning to do during summer vacation?)
5. 彼と結婚する**つもりはない**。(I have no intention of marrying him. → つもりはない = zero intention)
6. 今年中にN2に受かる**つもりだ**。(I intend to pass N2 within this year.)
7. お酒はもう飲まない**つもりだ**。(I'm planning to stop drinking.)
:::

### つもり的微妙语感||Nuances of つもり

:::zh
- 自分のことに使う：我打算…… ✅
- 他人に聞く：你打算……吗？ ✅
- 他人を描述する：他打算…… △（有点窥探内心的感觉，不太礼貌）

> 描述他人的计划时，更自然的说法是：彼は〜**そうだ**（听说他要……）/ 〜**らしい**（好像要……）
:::

:::en
- Talking about yourself: I plan to… ✅
- Asking another person: Do you plan to…? ✅
- Describing a third party: He plans to… △ (feels intrusive, slightly impolite)

> When describing someone else's plans, more natural alternatives are: 彼は〜**そうだ** (I heard he's going to…) / 〜**らしい** (it seems he's going to…)
:::

## 6. 〜予定だ（预定/安排）||6. 〜予定だ (Schedule/Arrangement)

### 接続

:::zh
- 动词辞书形 + 予定だ
- 名词 + の予定だ
:::

:::en
- Verb dictionary form + 予定だ
- Noun + の予定だ
:::

### 含义

:::zh
"按计划/按安排……"，比つもり更客观、更正式。常用于工作/正式场合。
:::

:::en
"According to the plan/schedule…" — more objective and formal than つもり. Common in work and formal contexts.

### 例句

:::zh
1. 来週出張する**予定です**。（下周按计划要出差。）

3. 来年卒業する**予定だ**。（预计明年毕业。）

5. 工事は来月終わる**予定でしたが**、遅れています。（工程本预计下月完工，但延迟了。）
6. 午後3時に会議がある**予定です**。（下午3点预定有会议。）
7. 新しいビルは来年完成する**予定だ**。（新大楼预定明年完工。）
:::

:::en
1. 来週出張する**予定です**。(I'm scheduled to go on a business trip next week.)
2. 会議は3時に始まる**予定です**。(The meeting is scheduled to start at 3.)
3. 来年卒業する**予定だ**。(I'm scheduled to graduate next year.)
4. 明日は休みの**予定です**。(Tomorrow is scheduled to be a day off. → noun + の予定)
5. 工事は来月終わる**予定でしたが**、遅れています。(The construction was scheduled to finish next month, but it's running late.)
6. 午後3時に会議がある**予定です**。(A meeting is scheduled for 3 PM.)
7. 新しいビルは来年完成する**予定だ**。(The new building is scheduled to be completed next year.)
:::

## 7. つもり vs 予定 vs ようと思う 比较表||7. Comparison: つもり vs 予定 vs ようと思う

:::zh
| 表达 | 确定度 | 语感 | 场景 | 例 |
|------|--------|------|------|---|
| 〜ようと思う | ★★☆ | 在考虑中/刚决定 | 个人内心想法 | 行こうと思う（在想着要去） |
| 〜つもりだ | ★★★ | 已决定的意图 | 个人坚定计划 | 行くつもりだ（打算去） |
| 〜予定だ | ★★★★ | 已确定的安排 | 工作/正式安排 | 行く予定だ（计划去/定好了） |
:::

:::en
| Expression | Certainty | Nuance | Context | Example |
|------------|-----------|--------|---------|---------|
| 〜ようと思う | ★★☆ | Considering / just decided | Personal inner thoughts | 行こうと思う (thinking of going) |
| 〜つもりだ | ★★★ | Firm personal intention | Personal committed plan | 行くつもりだ (intend to go) |
| 〜予定だ | ★★★★ | Confirmed arrangement | Work / formal schedule | 行く予定だ (scheduled to go) |
:::

### 同一件事的三种说法||Three Ways to Express the Same Event

:::zh
> 关于"暑假去日本旅行"这件事：

- 夏休みに日本に行**こうと思っている**。（在想着暑假去日本。→ 还在考虑阶段）
- 夏休みに日本に行く**つもりだ**。（暑假打算去日本。→ 自己已经决定了）
- 夏休みに日本に行く**予定だ**。（暑假计划去日本。→ 机票酒店可能已定）
:::

:::en
> About "going to Japan for a trip during summer vacation":

- 夏休みに日本に行**こうと思っている**。(I'm thinking of going to Japan over summer. → still in the considering stage)
- 夏休みに日本に行く**つもりだ**。(I'm planning to go to Japan over summer. → personally decided)
- 夏休みに日本に行く**予定だ**。(I'm scheduled to go to Japan over summer. → flights/hotel may already be booked)
:::

### 辨析要点||Key Comparison Points

:::zh
| 比较项 | ようと思う | つもり | 予定 |
|--------|-----------|--------|------|
| 能否用于他人 | ✅（〜と思っているらしい） | △（不太礼貌） | ✅（客观描述） |
| 否定形 | ようと思わない（不太常用） | 〜ないつもりだ / 〜つもりはない | 〜ない予定だ |
| 可以变化吗 | 容易改变 | 比较坚定 | 很难改变（已安排） |
| 主观/客观 | 主观 | 主观 | 客观 |
:::

:::en
| Point | ようと思う | つもり | 予定 |
|-------|-----------|--------|------|
| Used for others? | ✅ (〜と思っているらしい) | △ (slightly impolite) | ✅ (objective description) |
| Negative form | ようと思わない (less common) | 〜ないつもりだ / 〜つもりはない | 〜ない予定だ |

| Subjective/Objective | Subjective | Subjective | Objective |

:::zh
| | ましょう | ませんか |
|---|---|---|
| 语气 | 提议（比较直接） | 邀请（更委婉） |
| 前提 | 觉得对方会同意 | 尊重对方意愿 |
| 翻译 | "一起……吧" | "要不要一起……？" |

- 食べ**ませんか**。（要不要一起吃？→ 给对方选择权）
- 映画を見に行き**ましょう**。（去看电影吧。→ 说走就走的语气）

:::

:::en

|---|---|---|

| Translation | "Let's…" | "Would you like to…?" |

- 食べ**ませんか**。(Would you like to eat together? → gives the other a choice)
- 映画を見に行き**ましょう**。(Let's go see a movie. → ready-to-go tone)
- 映画を見に行き**ませんか**。(Would you like to go see a movie? → more like a formal invitation)

> **Memory tip**: ませんか has 「か」= question = polite; ましょう = direct rally.
:::

## 9. 〜たい / 〜たがる / 〜ほしい（愿望表达）||9. 〜たい / 〜たがる / 〜ほしい (Expressing Desires)

### 〜たい（我想做……）||〜たい (I want to do…)

:::zh
**接续**：动词ます形（去ます）+ たい

1. 日本に行き**たい**。（想去日本。）
2. 水が飲み**たい**。（想喝水。→ 对象用が）
3. 何も食べ**たくない**。（什么都不想吃。→ 否定：たくない）
4. 海外で働き**たかった**。（以前想在海外工作。→ 过去：たかった）
5. もっと日本語が上手になり**たい**。（想变得更擅长日语。）

> たい的活用和い形容词完全一样：
> たい→たくない→たかった→たくなかった→たくて
:::

:::en
**Conjugation**: Verb ます-form (drop ます) + たい

1. 日本に行き**たい**。(I want to go to Japan.)
2. 水が飲み**たい**。(I want to drink water. → object takes が)
3. 何も食べ**たくない**。(I don't want to eat anything. → negative: たくない)
4. 海外で働き**たかった**。(I used to want to work abroad. → past: たかった)
5. もっと日本語が上手になり**たい**。(I want to get better at Japanese.)

> たい conjugates exactly like an い-adjective:
> たい→たくない→たかった→たくなかった→たくて
:::

### 〜たがる（第三人称想做……）||〜たがる (Third person wants to do…)

:::zh
**接续**：动词ます形（去ます）+ たがる

日语中たい只用于说自己的愿望，描述别人"想做什么"要用たがる：

1. 彼はいつもビールを飲み**たがる**。（他总是想喝啤酒。）
2. 妹は一緒に行き**たがっている**。（妹妹想一起去。）
3. 子供は夜更かしし**たがる**。（小孩总想熬夜。）
4. 彼女はいつも甘いものを食べ**たがる**。（她总想吃甜食。）
5. 学生たちは早く帰り**たがっている**。（学生们想早点回去。）

> 不过实际会话中，间接引用也很常用：「彼は行きたい**と言っていた**」（他说想去）。
:::

:::en
**Conjugation**: Verb ます-form (drop ます) + たがる

In Japanese, たい is only used to express one's own desires. To describe what someone else wants to do, use たがる:

1. 彼はいつもビールを飲み**たがる**。(He always wants to drink beer.)
2. 妹は一緒に行き**たがっている**。(My younger sister wants to come along.)
3. 子供は夜更かしし**たがる**。(Kids always want to stay up late.)
4. 彼女はいつも甘いものを食べ**たがる**。(She always wants to eat something sweet.)
5. 学生たちは早く帰り**たがっている**。(The students want to go home early.)

> In actual conversation, indirect quotation is also very common: 「彼は行きたい**と言っていた**」(He said he wanted to go).
:::

### 〜ほしい（想要某物）||〜ほしい (Want something)

:::zh
**接续**：名词+がほしい

1. 新しいパソコンが**ほしい**。（想要新电脑。）
2. 何も**ほしくない**。（什么都不想要。）
3. 子供の時、犬が**ほしかった**。（小时候想要狗。）
4. 自分の部屋が**ほしい**。（想要自己的房间。）
5. もっと時間が**ほしい**。（想要更多时间。）

> 第三人称用「ほしがる」：子供はおもちゃを**欲しがる**。（小孩想要玩具。）
:::

:::en
**Conjugation**: Noun + がほしい

1. 新しいパソコンが**ほしい**。(I want a new computer.)
2. 何も**ほしくない**。(I don't want anything.)
3. 子供の時、犬が**ほしかった**。(When I was a child, I wanted a dog.)
4. 自分の部屋が**ほしい**。(I want my own room.)
5. もっと時間が**ほしい**。(I want more time.)

> For third person, use 「ほしがる」: 子供はおもちゃを**欲しがる**。(Children want toys.)
:::

### たい vs ほしい 对照表||たい vs ほしい Comparison

:::zh
| | たい | ほしい |
|---|---|---|
| 含义 | 我想**做**某事 | 我想**要**某物 |
| 接续 | 动词ます+たい | 名词+がほしい |
| 第三人称 | たがる | ほしがる |
| 活用 | い形容词型 | い形容词型 |
| 例 | コーヒーが飲みたい（想喝咖啡） | コーヒーがほしい（想要咖啡） |

> **易混淆点**：「コーヒーが飲みたい」和「コーヒーがほしい」意思接近，但飲みたい强调"想喝"这个动作，ほしい强调"想得到"这个东西。
:::

:::en
| | たい | ほしい |
|---|---|---|
| Meaning | I want to **do** something | I want to **have** something |
| Conjugation | Verb ます + たい | Noun + がほしい |
| Third person | たがる | ほしがる |
| Conjugation type | い-adjective type | い-adjective type |
| Example | コーヒーが飲みたい (want to drink coffee) | コーヒーがほしい (want coffee) |

> **Easy to confuse**: 「コーヒーが飲みたい」and「コーヒーがほしい」are close in meaning, but 飲みたい emphasizes the act of drinking, while ほしい emphasizes wanting to have/obtain the thing.
:::
