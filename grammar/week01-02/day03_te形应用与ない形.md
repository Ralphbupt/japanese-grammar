## 1. 〜ている（进行/状态/习惯）||〜ている (ongoing action / state / habit)

### 接続||Conjugation

:::zh
动词て形 + いる（敬体：います）
:::
:::en
Verb て form + いる (polite: います)
:::

### 含义||Meaning

:::zh
ている有三种完全不同的含义，取决于动词的类型：
:::
:::en
ている has three completely different meanings depending on the type of verb:
:::

### 用法①：正在进行（持続動詞）||Use ①: Ongoing action (continuous verbs)

:::zh
用于"有持续过程"的动词（食べる、読む、走る等）。

1. 今、ご飯を食べ**ている**。（正在吃饭。）
2. 彼は本を読ん**でいます**。（他正在看书。）
3. 子供たちが外で遊ん**でいる**。（孩子们正在外面玩。）
4. 母は今、電話で話し**ている**。（妈妈现在正在打电话。）
5. 弟はシャワーを浴び**ている**。（弟弟正在洗澡。）
:::
:::en
Used with verbs that describe a continuous process (食べる, 読む, 走る, etc.).

1. 今、ご飯を食べ**ている**。(I'm eating right now.)
2. 彼は本を読ん**でいます**。(He is reading a book.)
3. 子供たちが外で遊ん**でいる**。(The children are playing outside.)
4. 母は今、電話で話し**ている**。(My mom is on the phone right now.)
5. 弟はシャワーを浴び**ている**。(My younger brother is taking a shower.)
:::

### 用法②：結果状態（瞬間動詞）||Use ②: Resultant state (instantaneous verbs)

:::zh
用于"一瞬间就完成"的动词（死ぬ、結婚する、知る、住む等），表示**动作完成后的持续状态**。

1. 彼はもう死ん**でいる**。（他已经死了。→ 不是"正在死"）
2. 田中さんは結婚し**ている**。（田中已经结婚了。→ 结婚后的状态）
3. 東京に住ん**でいます**。（住在东京。→ 搬来之后的状态）
4. あ、窓が開い**ている**。（啊，窗户开着呢。→ 打开后的状态）
5. この言葉を知っ**ています**。（知道这个词。→ 知道了之后的状态）
6. 電気が消え**ている**。（灯灭着呢。→ 灭了之后的状态）
7. 彼はもう座っ**ている**。（他已经坐下了。→ 坐着的状态）
:::
:::en
Used with verbs that complete instantaneously (死ぬ, 結婚する, 知る, 住む, etc.) to express **the continuing state after the action is complete**.

1. 彼はもう死ん**でいる**。(He is already dead. → NOT "currently dying")
2. 田中さんは結婚し**ている**。(Tanaka is married. → the state of being married)
3. 東京に住ん**でいます**。(I live in Tokyo. → the state after moving there)
4. あ、窓が開い**ている**。(Oh, the window is open. → the state after being opened)
5. この言葉を知っ**ています**。(I know this word. → the state of knowing)
6. 電気が消え**ている**。(The light is off. → the state after being turned off)
7. 彼はもう座っ**ている**。(He is already seated. → the state of being seated)
:::

:::zh
> **超重要区分**：
> - 食べている = 正在吃（进行）
> - 死んでいる = 已经死了（状态）
> 
> 怎么判断？看动词本身是"持续的动作"还是"一瞬间的变化"。
:::
:::en
> **Critical distinction**:
> - 食べている = currently eating (ongoing action)
> - 死んでいる = is already dead (resultant state)
> 
> How to tell? Ask yourself: is the verb a "continuous process" or "an instantaneous change"?
:::

### 用法③：習慣/反復（搭配頻率詞）||Use ③: Habit / repeated action (with frequency words)

:::zh
1. 毎日ジョギングをし**ている**。（每天跑步。→ 习惯）
2. 最近、よく日本の映画を見**ている**。（最近常看日本电影。）
3. 父は銀行で働い**ています**。（父亲在银行工作。→ 职业=长期习惯）
4. 毎朝コーヒーを飲ん**でいる**。（每天早上喝咖啡。→ 习惯）
5. 彼女は毎週日本語を勉強し**ている**。（她每周都在学日语。→ 习惯）
:::
:::en
1. 毎日ジョギングをし**ている**。(I jog every day. → habit)
2. 最近、よく日本の映画を見**ている**。(I've been watching Japanese movies a lot lately.)
3. 父は銀行で働い**ています**。(My father works at a bank. → occupation = long-term habit)
4. 毎朝コーヒーを飲ん**でいる**。(I drink coffee every morning. → habit)
5. 彼女は毎週日本語を勉強し**ている**。(She studies Japanese every week. → habit)
:::

### 知る的特殊用法||Special usage of 知る

:::zh
| | 肯定 | 否定 |
|---|---|---|
| 知っている | 知道（状态） | **知らない**（不是知っていない！） |
| 知りました | 刚知道/了解了 | — |

> 「知っていません」虽然语法上能说，但日本人几乎不用，统一说「知りません」。
:::
:::en
| | Affirmative | Negative |
|---|---|---|
| 知っている | I know (state) | **知らない** (NOT 知っていない!) |
| 知りました | I just found out / I learned | — |

> 「知っていません」is grammatically possible but almost never used by native speakers — just say 「知りません」.
:::

### 辨析：ている vs た||Comparison: ている vs た

:::zh
| | ている | た |
|---|---|---|
| 持续动词 | 正在做（食べている＝正在吃） | 做了/做完了（食べた＝吃了） |
| 瞬间动词 | 状态持续（結婚している＝结了婚的状态） | 发生了（結婚した＝结了婚，焦点在事件） |

> 「結婚している」强调**现在的状态**（已婚）；「結婚した」强调**过去的事件**（那时结了婚）。
:::
:::en
| | ている | た |
|---|---|---|
| Continuous verb | Currently doing (食べている = eating right now) | Did / finished doing (食べた = ate) |
| Instantaneous verb | Ongoing resulting state (結婚している = is married) | Event occurred (結婚した = got married, focus on the event) |

> 「結婚している」emphasizes the **current state** (being married); 「結婚した」emphasizes the **past event** (the act of getting married).
:::

## 2. 〜てください / 〜ないでください（請求）||〜てください / 〜ないでください (Request)

### 接続||Conjugation

:::zh
- 肯定请求：动词て形 + ください
- 否定请求：动词ない形 + でください
:::
:::en
- Affirmative request: verb て form + ください
- Negative request: verb ない form + でください
:::

### 含義||Meaning

:::zh
- てください = "请你做……"，礼貌的请求。
- ないでください = "请不要做……"，礼貌的否定请求。
:::
:::en
- てください = "Please do…" — a polite request.
- ないでください = "Please don't do…" — a polite negative request.
:::

### 〜てください 例句||〜てください Example sentences

:::zh
1. ちょっと待っ**てください**。（请稍等。）
2. この漢字の読み方を教え**てください**。（请教我这个汉字的读法。）
3. 名前をここに書い**てください**。（请在这里写名字。）
4. もう一度言っ**てください**。（请再说一遍。）
5. 写真を撮っ**てください**。（请拍照。）
6. ここに座っ**てください**。（请坐这里。）
:::
:::en
1. ちょっと待っ**てください**。(Please wait a moment.)
2. この漢字の読み方を教え**てください**。(Please teach me how to read this kanji.)
3. 名前をここに書い**てください**。(Please write your name here.)
4. もう一度言っ**てください**。(Please say it one more time.)
5. 写真を撮っ**てください**。(Please take a photo.)
6. ここに座っ**てください**。(Please sit here.)
:::

### 〜ないでください 例句||〜ないでください Example sentences

:::zh
1. ここで写真を撮ら**ないでください**。（请不要在这里拍照。）
2. 心配し**ないでください**。（请不要担心。）
3. タバコを吸わ**ないでください**。（请不要抽烟。）
4. 授業中にスマホを使わ**ないでください**。（上课时请不要用手机。）
5. ここに触ら**ないでください**。（请不要碰这里。）
:::
:::en
1. ここで写真を撮ら**ないでください**。(Please don't take photos here.)
2. 心配し**ないでください**。(Please don't worry.)
3. タバコを吸わ**ないでください**。(Please don't smoke.)
4. 授業中にスマホを使わ**ないでください**。(Please don't use your phone in class.)
5. ここに触ら**ないでください**。(Please don't touch this.)
:::

### 辨析：てください vs ましょうか||Comparison: てください vs ましょうか

:::zh
| 表达 | 说话者 | 含义 |
|------|--------|------|
| 〜てください | 请求对方做 | 请你做…… |
| 〜ましょうか | 提议自己做 | 我来……好吗？ |

- 窓を開け**てください**。（请你开窗。→ 让对方做）
- 窓を開け**ましょうか**。（我来开窗好吗？→ 自己做）
:::
:::en
| Expression | Speaker's role | Meaning |
|------|--------|------|
| 〜てください | Asking the other person to do it | Please do… |
| 〜ましょうか | Offering to do it yourself | Shall I…? |

- 窓を開け**てください**。(Please open the window. → asking the other person)
- 窓を開け**ましょうか**。(Shall I open the window? → offering to do it yourself)
:::

## 3. 〜てもいい（許可）||〜てもいい (Permission)

### 接続||Conjugation

:::zh
动词て形 + もいいです
:::
:::en
Verb て form + もいいです
:::

### 含義||Meaning

:::zh
"可以做……"，表示许可。
:::
:::en
"May/can do…" — expresses permission.
:::

### 例句||Example sentences

:::zh
1. ここで写真を撮っ**てもいいですか**。（可以在这里拍照吗？）
   — はい、撮ってもいいですよ。（可以。）
   — いいえ、撮ってはいけません。（不行。→ 见下一个语法）
2. この辞書を使っ**てもいいですか**。（可以用这本词典吗？）
3. 窓を開け**てもいいですか**。（可以开窗吗？）
4. もう帰っ**てもいいですよ**。（你可以回去了。）
5. ここに座っ**てもいいですか**。（可以坐这里吗？）
6. 電気を消し**てもいいですか**。（可以关灯吗？）
:::
:::en
1. ここで写真を撮っ**てもいいですか**。(May I take photos here?)
   — はい、撮ってもいいですよ。(Yes, you may.)
   — いいえ、撮ってはいけません。(No, you may not. → see next grammar point)
2. この辞書を使っ**てもいいですか**。(May I use this dictionary?)
3. 窓を開け**てもいいですか**。(May I open the window?)
4. もう帰っ**てもいいですよ**。(You may go home now.)
5. ここに座っ**てもいいですか**。(May I sit here?)
6. 電気を消し**てもいいですか**。(May I turn off the light?)
:::

### 形容詞/名詞版||For adjectives and nouns

:::zh
- 安く**ても**いいです。（便宜也行。→ い形容词：い→くても）
- 静かじゃなく**ても**いいです。（不安静也行。→ な形容词否定：じゃなくても）
- 学生じゃなく**ても**いいです。（不是学生也行。→ 名词否定：じゃなくても）
:::
:::en
- 安く**ても**いいです。(Cheap is fine too. → い-adj: い→くても)
- 静かじゃなく**ても**いいです。(It doesn't have to be quiet. → な-adj negative: じゃなくても)
- 学生じゃなく**ても**いいです。(You don't have to be a student. → noun negative: じゃなくても)
:::

### 辨析：てもいい vs いい||Comparison: てもいい vs いい

:::zh
- 帰っ**てもいい**ですか。（可以回去吗？→ 征求许可）
- 帰って**いい**？（能走了吗？→ 口语简略，更随意）

> 口语中「てもいい」经常省略成「ていい」。
:::
:::en
- 帰っ**てもいい**ですか。(May I go home? → formally asking for permission)
- 帰って**いい**？(Can I go? → casual colloquial shortening)

> In casual speech, 「てもいい」is often shortened to 「ていい」.
:::

## 4. 〜てはいけない（禁止）||〜てはいけない (Prohibition)

### 接続||Conjugation

:::zh
动词て形 + はいけない（敬体：はいけません）
:::
:::en
Verb て form + はいけない (polite: はいけません)
:::

### 含義||Meaning

:::zh
"不可以做……"、"禁止做……"
:::
:::en
"Must not do…" / "It is forbidden to do…"
:::

### 例句||Example sentences

:::zh
1. ここで泳い**ではいけません**。（不可以在这里游泳。）
2. 授業中にスマホを使っ**てはいけない**。（上课中不能用手机。）
3. 嘘をつい**てはいけません**。（不可以说谎。）
4. 図書館で大きい声で話し**てはいけない**。（图书馆里不能大声说话。）
5. ここでタバコを吸っ**てはいけません**。（不可以在这里抽烟。）
6. この部屋に入っ**てはいけない**。（不可以进这个房间。）
:::
:::en
1. ここで泳い**ではいけません**。(You must not swim here.)
2. 授業中にスマホを使っ**てはいけない**。(You must not use your phone in class.)
3. 嘘をつい**てはいけません**。(You must not lie.)
4. 図書館で大きい声で話し**てはいけない**。(You must not speak loudly in the library.)
5. ここでタバコを吸っ**てはいけません**。(You must not smoke here.)
6. この部屋に入っ**てはいけない**。(You must not enter this room.)
:::

### 口語縮略||Colloquial contractions

:::zh
てはいけない → **ちゃいけない** / **ちゃだめ**
では → **じゃ**

- 食べ**ちゃ**いけない。= 食べてはいけない。
- 遊ん**じゃ**だめ。= 遊んではだめ。
- 触っ**ちゃ**だめだよ。= 触ってはだめだよ。
:::
:::en
てはいけない → **ちゃいけない** / **ちゃだめ**
では → **じゃ**

- 食べ**ちゃ**いけない。= 食べてはいけない。(You shouldn't eat that.)
- 遊ん**じゃ**だめ。= 遊んではだめ。(You can't play.)
- 触っ**ちゃ**だめだよ。= 触ってはだめだよ。(Don't touch it!)
:::

### 辨析：てはいけない vs ないでください||Comparison: てはいけない vs ないでください

:::zh
| 表达 | 语气 | 场景 |
|------|------|------|
| 〜てはいけない | 规则/禁止（客观） | 规定、法律、家长对孩子 |
| 〜ないでください | 请求/拜托（主观） | 请求对方不要做 |

- ここで泳い**ではいけない**。（这里禁止游泳。→ 规则）
- ここで泳が**ないでください**。（请不要在这里游泳。→ 请求）
:::
:::en
| Expression | Tone | Context |
|------|------|------|
| 〜てはいけない | Rule / prohibition (objective) | Regulations, laws, parent to child |
| 〜ないでください | Request / plea (subjective) | Asking someone not to do something |

- ここで泳い**ではいけない**。(Swimming is not allowed here. → a rule)
- ここで泳が**ないでください**。(Please don't swim here. → a request)
:::

## 5. 〜てから（之後）||〜てから (After doing)

### 接続||Conjugation

:::zh
动词て形 + から
:::
:::en
Verb て form + から
:::

### 含義||Meaning

:::zh
"做完A之后，再做B"，强调先后顺序。
:::
:::en
"After doing A, do B" — emphasizes that A must be completed before B.
:::

### 例句||Example sentences

:::zh
1. 手を洗っ**てから**、ご飯を食べます。（洗完手之后吃饭。）
2. 宿題をし**てから**、遊びに行く。（做完作业之后去玩。）
3. 日本に来**てから**、三年になります。（来日本之后已经三年了。）
4. よく考え**てから**、返事をします。（好好想之后再回答。）
5. シャワーを浴び**てから**、寝ます。（洗完澡之后睡觉。）
6. 準備し**てから**、出かけましょう。（准备好之后出门吧。）
:::
:::en
1. 手を洗っ**てから**、ご飯を食べます。(I wash my hands and then eat.)
2. 宿題をし**てから**、遊びに行く。(I'll go play after finishing my homework.)
3. 日本に来**てから**、三年になります。(It's been three years since I came to Japan.)
4. よく考え**てから**、返事をします。(I'll reply after thinking it over carefully.)
5. シャワーを浴び**てから**、寝ます。(I go to bed after taking a shower.)
6. 準備し**てから**、出かけましょう。(Let's head out after getting ready.)
:::

### 辨析：てから vs て||Comparison: てから vs て

:::zh
- ご飯を食べ**て**、出かけた。（吃了饭，出了门。→ 只是顺序）
- ご飯を食べ**てから**、出かけた。（吃完饭**之后**才出门。→ 强调必须先完成）

> **てから**强调"A完成是B的前提"。如果只是叙述先后发生的事，用**て**就够了。
:::
:::en
- ご飯を食べ**て**、出かけた。(I ate and went out. → just describing sequence)
- ご飯を食べ**てから**、出かけた。(I went out **after** eating. → emphasizes completion first)

> **てから** emphasizes that "completing A is a prerequisite for B." If you're simply narrating events in order, **て** alone is sufficient.

## 6. ない形（否定形）||ない Form (Negative Form)

### 含義||Meaning

:::zh
动词的简体否定形，"不做……"。是很多语法的接续基础。
:::
:::en
The plain negative form of verbs, meaning "don't do…" / "won't do…". It is the base for many grammar patterns.

- する → **しない**

| く | かない | 書く→書**かない** |

## 7. 〜なければならない / 〜なければいけない（必須）||〜なければならない / 〜なければいけない (Must / Have to)

### 接続||Conjugation

:::zh
动词ない形 → ない去い+ければならない

即：動詞なけれ**ば**なら**ない**
:::
:::en
Verb ない form → remove い + ければならない

In other words: verb stem + なけれ**ば**なら**ない**
:::

### 含義||Meaning

:::zh
"必须做……"、"不做不行"。

> 字面意思是"如果不做的话，就不行"→ 所以必须做。
:::
:::en
"Must do…" / "Have to do…"

> Literally: "If (I) don't do it, it won't do" → therefore (I) must do it.
:::

### 例句||Example sentences

:::zh
1. 明日までにレポートを出さ**なければならない**。（必须在明天之前交报告。）
2. 毎日薬を飲ま**なければなりません**。（必须每天吃药。）
3. 8時に起き**なければいけない**。（必须8点起床。）
4. もっと勉強し**なければならない**。（必须更加努力学习。）
5. パスポートを持って来**なければいけません**。（必须带护照来。）
6. 遅刻しないように、早く出**なければならない**。（为了不迟到，必须早出门。）
7. この書類にサインし**なければいけません**。（必须在这份文件上签名。）
:::
:::en
1. 明日までにレポートを出さ**なければならない**。(I must submit the report by tomorrow.)
2. 毎日薬を飲ま**なければなりません**。(I must take medicine every day.)
3. 8時に起き**なければいけない**。(I have to get up at 8.)
4. もっと勉強し**なければならない**。(I must study harder.)
5. パスポートを持って来**なければいけません**。(You must bring your passport.)
6. 遅刻しないように、早く出**なければならない**。(I must leave early so I won't be late.)
7. この書類にサインし**なければいけません**。(You must sign this document.)
:::

### 口語縮略形（非常常用！）||Colloquial contractions (very common!)

:::zh
| 书面/正式 | 口语 | 更随意 |
|----------|------|--------|
| 〜なければならない | 〜なきゃならない | 〜なきゃ |
| 〜なければいけない | 〜なきゃいけない | 〜なきゃ |

- 行かなければならない → 行か**なきゃ**。（得去了。）
- 早く寝なければ → 早く寝**なきゃ**。（得早点睡。）

> 口语中「〜なきゃ」单独使用就够了，后面省略。日常会话和动漫里超级常见。
:::
:::en
| Formal/written | Spoken | Casual |
|----------|------|--------|
| 〜なければならない | 〜なきゃならない | 〜なきゃ |
| 〜なければいけない | 〜なきゃいけない | 〜なきゃ |

- 行かなければならない → 行か**なきゃ**。(Gotta go.)
- 早く寝なければ → 早く寝**なきゃ**。(Gotta sleep early.)

> In casual speech, 「〜なきゃ」 alone is enough — the rest is dropped. You'll hear this constantly in everyday conversation and anime.
:::

### 類似表達：〜ないといけない||Similar expression: 〜ないといけない

:::zh
- 宿題をし**ないといけない**。（必须做作业。）
- 口语：し**ないと**。（得做。）
:::
:::en
- 宿題をし**ないといけない**。(I have to do my homework.)
- Casual: し**ないと**。(Gotta do it.)
:::

### 辨析：なければならない vs なければいけない||Comparison: なければならない vs なければいけない

:::zh
两者意思几乎相同，区别微妙：

| 表达 | 语感 |
|------|------|
| なければならない | 稍正式，书面语多，强调客观上"必须" |
| なければいけない | 稍口语，强调"不做的话会有不好的结果" |

> 实际使用中可以互换，考试也不会区分。口语统一用「なきゃ」就行。
:::
:::en
The two are almost identical in meaning; the difference is subtle:

| Expression | Nuance |
|------|------|
| なければならない | Slightly more formal/written; emphasizes objective obligation |
| なければいけない | Slightly more spoken; emphasizes "bad things will happen if you don't" |

> In practice they're interchangeable, and tests won't distinguish them. Just use 「なきゃ」in casual speech.
:::

## 10. 〜ないで vs 〜なくて（ない形的两种て形）||〜ないで vs 〜なくて (Two て forms of the ない form)

### 接続||Conjugation

:::zh
- 动词ない形 + で → ないで
- 动词ない形 → ない → なくて（像い形容词一样：い→くて）
:::
:::en
- Verb ない form + で → ないで
- Verb ない form → change ない to なくて (like い-adjective: い→くて)
:::

### 〜ないで = "不做A，做了B"（動作の替代/伴随状態）||〜ないで = "without doing A, did B" (substitution / manner)

:::zh
1. 朝ご飯を食べ**ないで**、学校に行った。（没吃早饭就去了学校。）
2. 辞書を使わ**ないで**、読んでみてください。（请不用词典试着读。）
3. 何も言わ**ないで**、出て行った。（什么都没说就出去了。）
4. 傘を持た**ないで**、出かけた。（没带伞就出门了。）
5. 薬を飲ま**ないで**、寝てしまった。（没吃药就睡了。）
:::
:::en
1. 朝ご飯を食べ**ないで**、学校に行った。(I went to school without eating breakfast.)
2. 辞書を使わ**ないで**、読んでみてください。(Please try reading without using a dictionary.)
3. 何も言わ**ないで**、出て行った。(He left without saying anything.)
4. 傘を持た**ないで**、出かけた。(I went out without taking an umbrella.)
5. 薬を飲ま**ないで**、寝てしまった。(I fell asleep without taking my medicine.)
:::

### 〜なくて = "因为没做A，所以B"（原因）/ 並列否定||〜なくて = "because I didn't do A, B" (reason) / parallel negation

:::zh
1. 時間がなく**て**、行けなかった。（因为没时间，所以没能去。→ 原因）
2. 安く**なくて**、高い。（不便宜，很贵。→ 并列）
3. わからなく**て**、困っています。（因为不明白，很困扰。→ 原因）
4. お金がなく**て**、買えなかった。（因为没钱，买不了。→ 原因）
5. 漢字が読め**なくて**、困った。（因为不会读汉字，很为难。→ 原因）
:::
:::en
1. 時間がなく**て**、行けなかった。(I couldn't go because I had no time. → reason)
2. 安く**なくて**、高い。(It's not cheap — it's expensive. → parallel)
3. わからなく**て**、困っています。(I'm in trouble because I don't understand. → reason)
4. お金がなく**て**、買えなかった。(I couldn't buy it because I had no money. → reason)
5. 漢字が読め**なくて**、困った。(I was troubled because I couldn't read the kanji. → reason)
:::

### 辨析口訣||Quick rule of thumb

:::zh
> **ないで**：描述行为方式（"没做A这个动作，直接做了B"）
> **なくて**：描述原因或状态（"因为不/没…"）

判断方法：把中文翻译里的连接词替换一下——
- "没……就……" → **ないで**
- "因为没……所以……" → **なくて**

| | ないで | なくて |
|---|---|---|
| 功能 | 行为方式/伴随 | 原因/并列 |
| 后句 | 通常是"做了另一件事" | 通常是"产生了某种结果/感受" |
| 例 | 食べないで行った（没吃就走了） | 食べなくて怒られた（因为没吃被骂了） |
:::
:::en
> **ないで**: describes manner of action ("without doing A, did B directly")
> **なくて**: describes cause or state ("because I didn't / wasn't…")

Test: replace the connector in English —
- "without doing…" → **ないで**
- "because I didn't…" → **なくて**

| | ないで | なくて |
|---|---|---|
| Function | Manner / accompanying state | Cause / parallel negation |
| Followed by | Usually another action taken | Usually a result or feeling |
| Example | 食べないで行った (left without eating) | 食べなくて怒られた (got scolded because I didn't eat) |
:::
