# N5 文法チェックリスト

> JLPT N5 核心语法约 70 条。掌握这些就能应付日常基础交流。

## 基本句型

- [ ] 〜です / 〜ではありません — 判断句（肯定/否定）
- [ ] 〜は〜です — 主题提示句型
- [ ] 〜も — 也
- [ ] 〜の — 所属/修饰助词
- [ ] 〜か — 疑问句标志
- [ ] 〜がある/〜がいる — 存在句（无生命/有生命）
- [ ] 〜になる — 变成~/变得~
- [ ] 〜にする — 决定~/选择~

## 助词

- [ ] は — 主题标志
- [ ] が — 主语标志
- [ ] を — 宾语标志
- [ ] に — 时间/地点/对象/方向
- [ ] で — 手段/场所/原因
- [ ] へ — 方向
- [ ] と — 并列/引用/共同
- [ ] から — 起点/原因
- [ ] まで — 终点
- [ ] より — 比较基准
- [ ] や — 不完全列举（A和B等）
- [ ] だけ — 只/仅仅
- [ ] けど/けれども — 但是/可是

## 动词活用

- [ ] ます形 — 礼貌体（肯定/否定/过去/过去否定）
- [ ] て形 — 连接形（五段动词音便规则）
- [ ] ない形 — 否定形
- [ ] た形 — 过去形
- [ ] 辞書形 — 原形
- [ ] 可能形 — 表示能力（〜られる/〜える）
- [ ] 意向形 — 意志/提议（〜よう/〜おう）

## て形应用

- [ ] 〜ている — 正在进行/结果状态/习惯
- [ ] 〜てください — 请求
- [ ] 〜てもいいです — 许可
- [ ] 〜てはいけません — 禁止
- [ ] 〜てから — 之后再
- [ ] 〜てあげる/てもらう/てくれる — 授受补助动词
- [ ] 〜てみる — 试着做~
- [ ] 〜てしまう — 做完了/不小心~了

## 义务与否定

- [ ] 〜なくてはいけない — 必须
- [ ] 〜なくてもいい — 不必
- [ ] 〜ないでください — 请不要

## 形容词

- [ ] い形容词活用 — 否定（〜くない）、过去（〜かった）
- [ ] な形容词活用 — 否定（〜ではない）、过去（〜だった）
- [ ] 〜くて/〜で — 形容词て形连接

## 比较与程度

- [ ] AはBより〜 — A比B更~
- [ ] AとBとどちらが〜 — A和B哪个更~
- [ ] 〜の中で〜が一番〜 — ~之中最~

## 愿望与目的

- [ ] 〜たい/〜たくない — 想要/不想要
- [ ] 〜たがる — 第三人称想要
- [ ] 〜ために — 为了（目的）
- [ ] 〜ように — 为了（能力/状态变化）

## 推测与样态

- [ ] 〜でしょう/〜だろう — 推测
- [ ] 〜かもしれない — 也许
- [ ] 〜そうだ（様態）— 看起来要~
- [ ] 〜そうだ（伝聞）— 听说~

## 说明与理由

- [ ] 〜のだ/〜んだ — 解释/强调/求解释
- [ ] 〜のに — 明明~却~（转折/遗憾）
- [ ] 〜ので — 因为~（比から更委婉）
- [ ] 〜し — 列举理由（而且~/又~又~）
- [ ] 〜たり〜たりする — 列举动作（又~又~）

## 计划与建议

- [ ] 〜つもりだ — 打算~/计划~
- [ ] 〜予定だ — 预定~/安排~
- [ ] 〜方がいい — 最好~（建议）
- [ ] 〜まだ〜ていない — 还没~
- [ ] 〜もう〜た — 已经~了

## 其他重要句型

- [ ] 〜ことができる — 能够（正式）
- [ ] 〜ことがある — 有时会/曾经~
- [ ] 〜前に/〜後で — 之前/之后
- [ ] 〜ながら — 一边~一边~
- [ ] 〜とき — ~的时候
- [ ] 〜と（条件）— 一~就~（自然/必然条件）
- [ ] 〜すぎる — 太~了
- [ ] 〜に行く/〜に来る — 去做~/来做~（目的）
- [ ] 〜のが好き/上手/下手 — 喜欢/擅长/不擅长做~
- [ ] 〜くらい/〜ぐらい — 大约~/左右
