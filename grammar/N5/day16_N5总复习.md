# Day 16 – N5 补充语法与总复习

## 7. 疑问词 + か/も/でも||7. Question Words + か/も/でも

### 疑問词 + か = "某……"（不確定）||Question word + か = "some…" (unspecified)

:::zh
| | 表达 | 含义 | 例句 |
|---|---|---|---|
| 誰か | 某人 | 誰かが来た（有人来了） | 誰かに聞いてみよう（问问某个人吧） |
| 何か | 某物 | 何か食べたい（想吃点什么） | 何か問題がありますか（有什么问题吗） |
| どこか | 某処 | どこかに行きたい（想去某个地方） | どこかで会った気がする（感觉在哪见过） |
| いつか | 某時 | いつか日本に行きたい（总有一天想去日本） | いつか必ずできるようになる（总有一天一定能做到） |
:::

:::en
| | Expression | Meaning | Example |
|---|---|---|---|
| 誰か | someone | 誰かが来た (someone came) | 誰かに聞いてみよう (let me ask someone) |
| 何か | something | 何か食べたい (want to eat something) | 何か問題がありますか (is there any problem?) |
| どこか | somewhere | どこかに行きたい (want to go somewhere) | どこかで会った気がする (I feel like we've met somewhere) |
| いつか | someday | いつか日本に行きたい (someday I want to go to Japan) | いつか必ずできるようになる (someday I'll definitely be able to do it) |
:::

### 疑問词 + も + 否定 = "全否定"||Question word + も + negative = "total negation"

:::zh
| | 表达 | 含义 | 例句 |
|---|---|---|---|
| 誰も〜ない | 谁都不 | 誰も来なかった（谁都没来） | 誰も答えがわからない（谁都不知道答案） |
| 何も〜ない | 什么都不 | 何も食べなかった（什么都没吃） | 何も聞こえない（什么都听不见） |
| どこも〜ない | 哪里都不 | どこにも行かなかった（哪里都没去） | どこにも売っていない（哪里都没卖） |
| いつも | ⚠️ 总是（不是否定！）| いつも遅刻する（总是迟到） | いつも優しい（总是很温柔） |

> **特别注意いつも！** 其他「疑问词+も+否定」都是全否定，但いつも是"总是/一直"（肯定），这是个例外！
:::

:::en
| | Expression | Meaning | Example |
|---|---|---|---|
| 誰も〜ない | nobody | 誰も来なかった (nobody came) | 誰も答えがわからない (nobody knows the answer) |
| 何も〜ない | nothing | 何も食べなかった (didn't eat anything) | 何も聞こえない (can't hear anything) |
| どこも〜ない | nowhere | どこにも行かなかった (didn't go anywhere) | どこにも売っていない (not sold anywhere) |
| いつも | ⚠️ always (NOT negative!) | いつも遅刻する (always late) | いつも優しい (always kind) |

> **Special note about いつも!** All other "question word + も + negative" patterns are total negations, but いつも means "always/all the time" (affirmative) — it is an exception!
:::

### 疑問词 + でも = "無論……都"||Question word + でも = "no matter… / any…"

:::zh
| | 表达 | 含义 | 例句 |
|---|---|---|---|
| 誰でも | 无论谁 | 誰でもできる（谁都能做） | 誰でも参加できます（谁都能参加） |
| 何でも | 无论什么 | 何でも食べる（什么都吃） | 何でも聞いてください（什么都可以问） |
| どこでも | 无论哪里 | どこでも寝られる（在哪都能睡） | どこでもWi-Fiが使える（到哪都能用Wi-Fi） |
| いつでも | 无论何时 | いつでも来てください（随时来） | いつでも連絡してね（随时联系我） |
:::

:::en
| | Expression | Meaning | Example |
|---|---|---|---|
| 誰でも | anyone | 誰でもできる (anyone can do it) | 誰でも参加できます (anyone can participate) |
| 何でも | anything | 何でも食べる (eats anything) | 何でも聞いてください (feel free to ask anything) |
| どこでも | anywhere | どこでも寝られる (can sleep anywhere) | どこでもWi-Fiが使える (Wi-Fi available everywhere) |
| いつでも | anytime | いつでも来てください (come anytime) | いつでも連絡してね (feel free to contact me anytime) |
:::

## 8. 〜すぎる（過度）||8. 〜すぎる (Excess)

### 接続

:::zh
- 动词ます形（去ます）+ すぎる
- い形容词（去い）+ すぎる
- な形容词 语干 + すぎる
:::

:::en
- Verb ます-stem + すぎる
- い-adjective (drop い) + すぎる
- な-adjective stem + すぎる
:::

### 含义

:::zh
"太……了"，表示程度过度，通常含有**负面评价**的语气。
:::

:::en
"Too…" — expresses that something is excessive, usually with a **negative connotation**.
:::

### 例句

:::zh
1. 昨日飲み**すぎた**。（昨天喝太多了。）
2. 食べ**すぎて**、お腹が痛い。（吃太多了，肚子疼。）
3. この問題は難し**すぎる**。（这道题太难了。）
4. 高**すぎます**。もう少し安いのはありますか。（太贵了。有便宜点的吗？）
5. 静か**すぎて**、怖い。（太安静了，有点怕。）
6. 昨日ゲームをやり**すぎて**、寝不足だ。（昨天打游戏打太多，睡眠不足。）
7. この映画は長**すぎる**。3時間もある。（这部电影太长了。有3个小时。）
:::

:::en
1. 昨日飲み**すぎた**。(I drank too much yesterday.)
2. 食べ**すぎて**、お腹が痛い。(I ate too much and my stomach hurts.)
3. この問題は難し**すぎる**。(This problem is too difficult.)
4. 高**すぎます**。もう少し安いのはありますか。(It's too expensive. Do you have anything a bit cheaper?)
5. 静か**すぎて**、怖い。(It's so quiet it's scary.)
6. 昨日ゲームをやり**すぎて**、寝不足だ。(I played games too much yesterday and didn't get enough sleep.)
7. この映画は長**すぎる**。3時間もある。(This movie is too long — it's 3 hours!)
:::

### 辨析：すぎる vs とても||Contrast: すぎる vs とても

:::zh
| | すぎる | とても |
|---|---|---|
| 语气 | 超过了限度（负面） | 程度很高（中性/正面） |
| 例句 | 高すぎる（太贵了→不好） | とても高い（非常贵→描述事实） |
| 能否正面用 | 基本负面 | 可正面可负面 |

> すぎる本身也是动词（二类动词），可以变形：すぎた、すぎて、すぎない……
:::

:::en
| | すぎる | とても |
|---|---|---|
| Tone | Exceeds a limit (negative) | High degree (neutral/positive) |
| Example | 高すぎる (too expensive → bad) | とても高い (very expensive → just a description) |
| Positive use? | Basically negative | Can be positive or negative |

> すぎる is itself a verb (Group 2) and can conjugate: すぎた, すぎて, すぎない…
:::

## 9. 〜やすい / 〜にくい（容易/難以）||9. 〜やすい / 〜にくい (Easy to / Hard to)

### 接続

:::zh
动词ます形（去ます）+ やすい / にくい
:::

:::en
Verb ます-stem + やすい / にくい
:::

### 含义

:::zh
- **やすい：** 容易做……（物品性质或个人倾向）
- **にくい：** 难以做……
:::

:::en
- **やすい:** Easy to do… (a property of something, or a personal tendency)
- **にくい:** Difficult/hard to do…
:::

### 例句

:::zh
1. この本は読み**やすい**。（这本书容易读。）
2. この漢字は覚え**にくい**。（这个汉字难记。）
3. この靴は歩き**やすい**。（这双鞋好走路。）
4. あの先生の話は分かり**にくい**。（那个老师的话难懂。）
5. 壊れ**やすい**から、気をつけて。（容易坏，小心点。）
6. このペンは書き**やすい**。気に入った。（这支笔好写，很喜欢。）
7. 黒い服は汚れが見え**にくい**。（黑衣服不容易看到脏。）
:::

:::en
1. この本は読み**やすい**。(This book is easy to read.)
2. この漢字は覚え**にくい**。(This kanji is hard to memorize.)
3. この靴は歩き**やすい**。(These shoes are easy to walk in.)
4. あの先生の話は分かり**にくい**。(That teacher's explanations are hard to understand.)
5. 壊れ**やすい**から、気をつけて。(It breaks easily, so be careful.)
6. このペンは書き**やすい**。気に入った。(This pen writes smoothly. I like it.)
7. 黒い服は汚れが見え**にくい**。(Dirt is hard to see on black clothes.)
:::

### 辨析：にくい vs がたい（N3预告）||Contrast: にくい vs がたい (N3 preview)

:::zh
| | にくい | がたい |
|---|---|---|
| 意思 | 物理上/客观上难以 | 心理上/感情上难以 |
| 例句 | 読みにくい（字小/模糊） | 信じがたい（难以置信） |
| 级别 | N5 | N3 |

> やすい/にくい活用和い形容词一样：読みやすかった、読みやすくない、読みやすくて……
:::

:::en
| | にくい | がたい |
|---|---|---|
| Meaning | Physically / objectively difficult | Emotionally / psychologically difficult |
| Example | 読みにくい (hard to read — small or blurry) | 信じがたい (hard to believe) |
| Level | N5 | N3 |

> やすい/にくい conjugate like い-adjectives: 読みやすかった、読みやすくない、読みやすくて…
:::

## 10. 〜ながら（同時）||10. 〜ながら (Simultaneous Actions)

### 接続

:::zh
动词ます形（去ます）+ ながら
:::

:::en
Verb ます-stem + ながら
:::

### 含义

:::zh
"一边做A一边做B"，两个动作同时进行。**后面的动作是主要动作。**
:::

:::en
"While doing A, do B" — two actions happening at the same time. **The action after ながら is the main/primary action.**
:::

### 例句

:::zh
1. 音楽を聴き**ながら**、勉強する。（一边听音乐一边学习。→ 主要在学习）
2. 歩き**ながら**、スマホを見ないでください。（请不要边走边看手机。）
3. コーヒーを飲み**ながら**、新聞を読む。（边喝咖啡边看报纸。）
4. 笑い**ながら**、話した。（笑着说了。）
5. テレビを見**ながら**、ご飯を食べる。（边看电视边吃饭。）
6. 友達と話し**ながら**、歩いて帰った。（一边和朋友聊天一边走回家。）
7. 料理を作り**ながら**、歌を歌う。（一边做饭一边唱歌。）
:::

:::en
1. 音楽を聴き**ながら**、勉強する。(I study while listening to music. → main action is studying)
2. 歩き**ながら**、スマホを見ないでください。(Please don't look at your phone while walking.)
3. コーヒーを飲み**ながら**、新聞を読む。(I read the newspaper while drinking coffee.)
4. 笑い**ながら**、話した。(They spoke with a smile / while laughing.)
5. テレビを見**ながら**、ご飯を食べる。(I eat while watching TV.)
6. 友達と話し**ながら**、歩いて帰った。(I walked home while chatting with a friend.)
7. 料理を作り**ながら**、歌を歌う。(I sing while cooking.)
:::

### 辨析：ながら vs ている||Contrast: ながら vs ている

:::zh
| | ながら | ている |
|---|---|---|
| 功能 | 两个动作同时进行 | 一个动作的进行状态 |
| 主语 | 必须相同（同一个人） | 不限 |
| 例句 | 食べながら話す（边吃边说） | 食べている（正在吃） |
| 注意 | 不能用于两个不同主语 | — |

> **ながら的主语必须相同：**
> - ✅ 私は音楽を聴きながら勉強する。（我边听音乐边学习）
> - ❌ 私が音楽を聴きながら、彼が勉強する。（主语不同，不能用ながら）
:::

:::en
| | ながら | ている |
|---|---|---|
| Function | Two simultaneous actions | One action in progress |
| Subject | Must be the same person | Not restricted |
| Example | 食べながら話す (talk while eating) | 食べている (is eating) |
| Note | Cannot be used with two different subjects | — |

> **ながら requires the same subject:**
> - ✅ 私は音楽を聴きながら勉強する。(I study while listening to music.)
> - ❌ 私が音楽を聴きながら、彼が勉強する。(Different subjects — cannot use ながら.)
:::

## 11. 〜方（かた）（做法/方法）||11. 〜方（かた）(How to / Method)

### 接続

:::zh
动词ます形（去ます）+ 方（かた）
:::

:::en
Verb ます-stem + 方（かた）
:::

### 含义

:::zh
"做……的方法"。变形后变成名词，可以加の、が、を等助词。
:::

:::en
"The way to do…" / "How to do…" — nominalizes the action; the resulting noun can take particles like の, が, を, etc.
:::

### 例句

:::zh
1. この漢字の読み**方**を教えてください。（请教我这个汉字的读法。）
2. 使い**方**がわからない。（不知道使用方法。）
3. 作り**方**は簡単です。（做法很简单。）
4. 行き**方**を教えてもらえますか。（能告诉我怎么去吗？）
5. 食べ**方**が汚い。（吃相难看。）
6. 日本語の勉強の仕**方**を変えたい。（想改变日语的学习方法。）
7. この機械の動かし**方**を教えてください。（请教我这个机器的使用方法。）
:::

:::en
1. この漢字の読み**方**を教えてください。(Please teach me how to read this kanji.)
2. 使い**方**がわからない。(I don't know how to use it.)
3. 作り**方**は簡単です。(The way to make it is simple.)
4. 行き**方**を教えてもらえますか。(Could you tell me how to get there?)
5. 食べ**方**が汚い。(Your table manners are bad. / The way you eat is messy.)
6. 日本語の勉強の仕**方**を変えたい。(I want to change the way I study Japanese.)
7. この機械の動かし**方**を教えてください。(Please show me how to operate this machine.)
:::

### 辨析：方（かた）vs 方法（ほうほう）||Contrast: 〜方（かた）vs 方法（ほうほう）

:::zh
| | 〜方（かた） | 方法（ほうほう） |
|---|---|---|
| 词性 | 接尾词（接动词后） | 名词（独立使用） |
| 用法 | 読み方、使い方 | 勉強の方法、解決の方法 |
| 语感 | 口语常用 | 稍微正式 |
:::

:::en
| | 〜方（かた）| 方法（ほうほう）|
|---|---|---|
| Word class | Suffix (attaches to verb) | Independent noun |
| Usage | 読み方、使い方 | 勉強の方法、解決の方法 |
| Register | Commonly used in speech | Slightly more formal |
:::

## 今日练习

### 一、ことができる：把可能形改写成ことができる

1. ピアノが弾ける。→ ？
2. この川で泳げない。→ ？
3. 漢字が読める。→ ？
4. ここで写真が撮れる。→ ？

<details>
<summary>答案</summary>

1. ピアノを弾くことができる。
2. この川で泳ぐことができない。
3. 漢字を読むことができる。
4. ここで写真を撮ることができる。

</details>

### 二、ことがある vs ことはない：选择正确的表达

1. 疲れた日は夕飯を作らない（ことがある / ことはない）。→ 有时候不做
2. そんなに怒る（ことがある / ことはない）よ。→ 没必要
3. 彼は時々遅刻する（ことがある / ことはない）。→ 有时候
4. 一人で行く（ことがある / ことはない）。私が一緒に行くから。→ 不用

<details>
<summary>答案</summary>

:::zh
1. ことがある（有时候累了不做晚饭）
2. ことはない（没必要那么生气）
3. ことがある（他有时候会迟到）
4. ことはない（不用一个人去，我和你一起）
:::

:::en
1. ことがある (sometimes too tired to make dinner)
2. ことはない (no need to be that angry)
3. ことがある (he is sometimes late)
4. ことはない (no need to go alone — I'll come with you)
:::

</details>

### 三、のだ/んだ：用んです改写

1. 明日は来られません。用事があります。→ 用事が＿＿＿。
2. A：顔が赤いですね。B：お酒を飲みました。→ お酒を＿＿＿。
3. 元気がないですね。何かありましたか。→ 何か＿＿＿。

<details>
<summary>答案</summary>

:::zh
1. 用事があるんです。（解释原因）
2. お酒を飲んだんです。（解释原因）
3. 何かあったんですか。（追问理由）
:::

:::en
1. 用事があるんです。(explaining the reason)
2. お酒を飲んだんです。(explaining the reason)
3. 何かあったんですか。(asking for an explanation)
:::

</details>

### 四、のに vs けど：根据语境选択

1. 薬を飲んだ（のに / けど）、まだ治らない。→ 说话人不満
2. 雨が降った（のに / けど）、試合はあった。→ 客观叙述
3. ダイエット中な（のに / けど）、ケーキを食べてしまった。→ 后悔
4. 日本語は難しい（のに / けど）、面白い。→ 中性転折

<details>
<summary>答案</summary>

:::zh
1. のに（明明吃了药还没好→不满）
2. けど（虽然下了雨但比赛照常→客观）
3. のに（明明在减肥却吃了蛋糕→后悔）
4. けど（虽然难但有趣→中性对比）
:::

:::en
1. のに (even though I took medicine, not better yet → dissatisfaction)
2. けど (it rained but the match went ahead → neutral)
3. のに (even though I'm dieting, I ate the cake → regret)
4. けど (Japanese is hard but interesting → neutral contrast)
:::

</details>

### 五、ので vs から：选択更合適的

### 十二、综合問題：选择合適的語法填空

1. せっかく作った＿＿＿、誰も食べなかった。（のに / ので）
2. 日本語は難しい＿＿＿、楽しい＿＿＿、毎日勉強している。（し / から）
3. あの映画は長＿＿＿。3時間もあった。（すぎる / やすい）
4. 料理を作り＿＿＿、音楽を聴いている。（ながら / 方）
5. 明日試験がある＿＿＿、今日は早く寝ます。（ので / のに）
6. ＿＿＿でもいいから、何か食べたい。（何 / 誰）
7. この字の読み＿＿＿がわからない。（方 / やすい）
8. そんなに心配する＿＿＿。大丈夫だよ。（ことがある / ことはない）

<details>
<summary>答案</summary>

:::zh
1. のに（明明特意做了却没人吃→不满）
2. し、し（又难又有趣→列举多个理由）
3. すぎる（太长了→过度）
4. ながら（一边做饭一边听音乐→同时）
5. ので（因为明天有考试→解释原因）
6. 何（什么都行→无论什么）
7. 方（读法→方法）
8. ことはない（没必要→没必要那么担心）
:::

:::en
1. のに (even though I made it specially, no one ate it → dissatisfaction)
2. し、し (both difficult and fun → listing multiple reasons)
3. すぎる (too long → excessive)
4. ながら (listening to music while cooking → simultaneous)
5. ので (because there's an exam tomorrow → explaining reason)
6. 何 (anything will do → no matter what)
7. 方 (the reading → method)
8. ことはない (no need → no need to worry so much)
:::

</details>
