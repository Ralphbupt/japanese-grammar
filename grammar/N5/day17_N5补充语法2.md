## 1. 〜がある / 〜がいる（存在表达）||1. 〜がある / 〜がいる (Existence)

### 接続

:::zh
- 場所 **に** 物/人 **が** ある/いる
- ある → 非生物（东西、植物、建筑等）
- いる → 生物（人、动物等）
:::

:::en
- Place **に** thing/person **が** ある/いる
- ある → non-living things (objects, plants, buildings, etc.)
- いる → living things (people, animals, etc.)
:::

### 含义

:::zh
表示某个地方**存在**某物/某人。"在那里有……"。

- に = 存在的地点
- が = 存在的主体

> **核心区别**：ある用于**无生命**的东西，いる用于**有生命**的东西。植物虽然有生命，但日语中用ある。
:::

:::en
Indicates that something/someone **exists** at a certain location. "There is/are..."

- に = location of existence
- が = the thing/person that exists

> **Key distinction**: ある is for **non-living** things; いる is for **living** things. Plants, though alive, use ある in Japanese.
:::

### 例句（ある — 非生物）||Example Sentences (ある — Non-living)

:::zh
1. 机の上に本**がある**。（桌子上有书。）
2. 冷蔵庫にケーキ**がある**よ。（冰箱里有蛋糕哦。）
3. 駅の前にコンビニ**があります**。（车站前面有便利店。）
4. 引き出しの中に財布**があった**。（抽屉里有钱包。）
5. あの公園にはきれいな花**がある**。（那个公园有漂亮的花。）
:::

:::en
1. 机の上に本**がある**。(There is a book on the desk.)
2. 冷蔵庫にケーキ**がある**よ。(There's cake in the fridge!)
3. 駅の前にコンビニ**があります**。(There is a convenience store in front of the station.)
4. 引き出しの中に財布**があった**。(There was a wallet in the drawer.)
5. あの公園にはきれいな花**がある**。(There are beautiful flowers in that park.)
:::

### 例句（いる — 生物）||Example Sentences (いる — Living)

:::zh
1. 庭に猫**がいる**。（院子里有猫。）
2. 公園に子供**がいます**。（公园里有小孩。）
3. 池に金魚**がいる**。（池塘里有金鱼。）
4. 教室に先生**がいますか**。（教室里有老师吗？）
5. 家に誰も**いない**。（家里谁都不在。）
:::

:::en
1. 庭に猫**がいる**。(There is a cat in the garden.)
2. 公園に子供**がいます**。(There are children in the park.)
3. 池に金魚**がいる**。(There are goldfish in the pond.)
4. 教室に先生**がいますか**。(Is there a teacher in the classroom?)
5. 家に誰も**いない**。(There is nobody at home.)
:::

### 辨析：「〜に〜がある」vs「〜は〜にある」||Contrast: 「〜に〜がある」vs「〜は〜にある」

:::zh
| | 場所に〜がある | 〜は場所にある |
|---|---|---|
| 焦点 | **存在するもの**（新情報） | **場所**（新情報） |
| 使い方 | "那里有什么？" | "那个东西在哪？" |
| 例 | 机の上に本**がある** | 本**は**机の上に**ある** |
| 翻译 | 桌上有书（介绍新事物） | 书在桌上（说明位置） |

**使い分け：**
- Q: あの部屋に何がありますか。→ テレビ**が**あります。（什么存在？→ 电视。）
- Q: 鍵はどこにありますか。→ テーブルの上に**あります**。（钥匙在哪？→ 桌上。）

> が = 引入新信息；は = 讨论已知事物
:::

:::en
| | 場所に〜がある | 〜は場所にある |
|---|---|---|
| Focus | **What exists** (new info) | **Where it is** (new info) |
| Usage | "What's there?" | "Where is it?" |
| Example | 机の上に本**がある** | 本**は**机の上に**ある** |
| Translation | There's a book on the desk (introducing) | The book is on the desk (locating) |

**How to choose:**
- Q: あの部屋に何がありますか。→ テレビ**が**あります。(What's in that room? → A TV.)
- Q: 鍵はどこにありますか。→ テーブルの上に**あります**。(Where are the keys? → On the table.)

> が = introduces new information; は = discusses known topics
:::

## 2. 〜に行く / 〜に来る / 〜に帰る（目的移动）||2. 〜に行く / 〜に来る / 〜に帰る (Purpose of Movement)

### 接続

:::zh
動詞ます形（去ます）+ **に** + 行く/来る/帰る

> ます形去掉ます后的部分（連用形）+ に行く。
> 例：見**ます** → 見 + に行く → 見に行く
:::

:::en
Verb ます-stem (remove ます) + **に** + 行く/来る/帰る

> Take the ます-stem (conjunctive form) + に行く.
> Example: 見**ます** → 見 + に行く → 見に行く
:::

### 含义

:::zh
"去/来/回去**做某事**"。に表示移动的**目的**。
:::

:::en
"Go/come/return **to do something**." に marks the **purpose** of movement.
:::

### 例句

:::zh
1. 映画を**見に行きます**。（去看电影。）
2. 友達に**会いに行った**。（去见了朋友。）
3. ご飯を**食べに来ました**。（来吃饭的。）
4. 日本語を**勉強しに来た**。（来学日语的。）
5. 本を**借りに行きませんか**。（要不要去借书？）
6. 忘れ物を**取りに帰った**。（回去拿忘记的东西了。）
7. お弁当を**買いに行こう**。（去买便当吧。）
8. 先生に質問を**聞きに行きます**。（去问老师问题。）
:::

:::en
1. 映画を**見に行きます**。(I'm going to see a movie.)
2. 友達に**会いに行った**。(I went to see a friend.)
3. ご飯を**食べに来ました**。(I came to eat.)
4. 日本語を**勉強しに来た**。(I came to study Japanese.)
5. 本を**借りに行きませんか**。(Shall we go borrow some books?)
6. 忘れ物を**取りに帰った**。(I went back to get something I forgot.)
7. お弁当を**買いに行こう**。(Let's go buy a bento.)
8. 先生に質問を**聞きに行きます**。(I'm going to ask the teacher a question.)
:::

### 辨析：〜に行く vs 〜ために行く||Contrast: 〜に行く vs 〜ために行く

:::zh
| | 〜に行く | 〜ために行く |
|---|---|---|
| 接続 | ます形（去ます）+ に行く | 辞書形 + ために行く |
| 语感 | 日常会话、自然 | 稍正式、强调目的 |
| 例 | 映画を見**に行く** | 映画を見る**ために行く** |
| 翻译 | 去看电影 | 为了看电影而去 |

> 日常会话中，「〜に行く」更自然。「〜ために行く」较正式，强调目的意识。

**注意：** 名词の场合用「に」直接接续：
- 買い物**に**行く（去买东西）
- 散歩**に**行く（去散步）
:::

:::en
| | 〜に行く | 〜ために行く |
|---|---|---|
| Form | ます-stem + に行く | Dictionary form + ために行く |
| Nuance | Daily conversation, natural | Slightly formal, emphasizes purpose |
| Example | 映画を見**に行く** | 映画を見る**ために行く** |
| Translation | Go see a movie | Go for the purpose of seeing a movie |

> In everyday conversation, 「〜に行く」 is more natural. 「〜ために行く」 is more formal and emphasizes conscious purpose.

**Note:** With nouns, use に directly:
- 買い物**に**行く (go shopping)
- 散歩**に**行く (go for a walk)
:::

## 3. 〜のが好き / 上手 / 下手 / 得意 / 苦手||3. 〜のが好き / 上手 / 下手 / 得意 / 苦手

### 接続

:::zh
動詞辞書形 + **のが** + 好き/上手/下手/得意/苦手

> 「の」把动词变成名词（名词化）。日语中形容词不能直接修饰动词，需要先把动词变成名词。
:::

:::en
Verb dictionary form + **のが** + 好き/上手/下手/得意/苦手

> 「の」 nominalizes the verb (turns it into a noun). In Japanese, these adjectives cannot directly modify verbs — the verb must first be nominalized.
:::

### 含义

:::zh
| 表达 | 含义 | 语感 |
|---|---|---|
| 〜のが**好き** | 喜欢做〜 | 个人喜好 |
| 〜のが**嫌い** | 讨厌做〜 | 个人喜好 |
| 〜のが**上手** | 做〜做得好 | 客观评价他人 |
| 〜のが**下手** | 做〜做得差 | 客观评价他人 |
| 〜のが**得意** | 擅长做〜 | 自我评价也可 |
| 〜のが**苦手** | 不擅长做〜 | 自我评价也可 |
:::

:::en
| Expression | Meaning | Usage |
|---|---|---|
| 〜のが**好き** | Like doing ~ | Personal preference |
| 〜のが**嫌い** | Dislike doing ~ | Personal preference |
| 〜のが**上手** | Good at doing ~ | Evaluating others (objective) |
| 〜のが**下手** | Bad at doing ~ | Evaluating others (objective) |
| 〜のが**得意** | Skilled at doing ~ | Self-evaluation OK |
| 〜のが**苦手** | Not skilled at doing ~ | Self-evaluation OK |
:::

### 例句

:::zh
1. 本を読む**のが好きです**。（喜欢读书。）
2. 朝早く起きる**のが嫌いです**。（讨厌早起。）
3. 田中さんは料理を作る**のが上手です**。（田中做菜做得好。）
4. 弟は絵を描く**のが下手です**。（弟弟画画画得差。）
5. 私は泳ぐ**のが得意です**。（我擅长游泳。）
6. 私は歌う**のが苦手です**。（我不擅长唱歌。）
7. 子供は走る**のが好き**だ。（小孩喜欢跑步。）
8. 彼女は踊る**のが上手**だ。（她跳舞跳得好。）
:::

:::en
1. 本を読む**のが好きです**。(I like reading books.)
2. 朝早く起きる**のが嫌いです**。(I hate waking up early.)
3. 田中さんは料理を作る**のが上手です**。(Tanaka is good at cooking.)
4. 弟は絵を描く**のが下手です**。(My younger brother is bad at drawing.)
5. 私は泳ぐ**のが得意です**。(I'm good at swimming.)
6. 私は歌う**のが苦手です**。(I'm not good at singing.)
7. 子供は走る**のが好き**だ。(Children like running.)
8. 彼女は踊る**のが上手**だ。(She is good at dancing.)
:::

### 辨析：好き vs 得意、下手 vs 苦手||Contrast: 好き vs 得意, 下手 vs 苦手

:::zh
| | 好き | 得意 |
|---|---|---|
| 意味 | **喜欢**（感情） | **擅长**（能力） |
| 例 | 歌うのが好き → 喜欢唱歌（但不一定唱得好） | 歌うのが得意 → 擅长唱歌（有能力） |

| | 下手 | 苦手 |
|---|---|---|
| 意味 | 做得**差**（客观技术差） | **不擅长/不喜欢**（主观感受） |
| 例 | 料理が下手 → 做菜难吃（事实） | 料理が苦手 → 做菜对我来说有困难/不想做 |

> **上手/下手** — 用于评价**别人**，不适合说自己。说自己用 **得意/苦手**。
> - ✅ 私は料理が**得意**です。
> - △ 私は料理が**上手**です。（听起来在自夸）
> - ✅ 田中さんは料理が**上手**です。
:::

:::en
| | 好き | 得意 |
|---|---|---|
| Meaning | **Like** (emotion) | **Good at** (ability) |
| Example | 歌うのが好き → like singing (may not be good) | 歌うのが得意 → good at singing (capable) |

| | 下手 | 苦手 |
|---|---|---|
| Meaning | **Poor at** (objectively bad skill) | **Not good at / uncomfortable with** (subjective) |
| Example | 料理が下手 → bad cooking (fact) | 料理が苦手 → cooking is hard for me / I'd rather not |

> **上手/下手** — used to evaluate **others**, not appropriate for yourself. Use **得意/苦手** for self-evaluation.
> - ✅ 私は料理が**得意**です。
> - △ 私は料理が**上手**です。(Sounds like bragging)
> - ✅ 田中さんは料理が**上手**です。
:::

## 4. 〜くらい / 〜ぐらい（大约）||4. 〜くらい / 〜ぐらい (Approximately)

### 接続

:::zh
数量/時間/名詞 + くらい/ぐらい

> くらい和ぐらい意思完全相同，可以互换。口语中ぐらい更常见。
:::

:::en
Quantity / Time / Noun + くらい/ぐらい

> くらい and ぐらい are completely interchangeable. ぐらい is slightly more common in speech.
:::

### 含义

:::zh
两种主要用法：

**① 大约、大概（概数）**
表示不精确的数量或时间。

**② 程度（〜くらい / 那种程度）**
表示某种程度、水平。
:::

:::en
Two main uses:

**① Approximately (rough number)**
Indicates an inexact quantity or time.

**② Degree / extent**
Indicates a certain degree or level.
:::

### 例句（① 大约）||Example Sentences (① Approximately)

:::zh
1. 30分**ぐらい**待ちました。（等了大约30分钟。）
2. 1000円**くらい**あれば足りる。（有1000日元左右就够了。）
3. 学生は50人**ぐらい**います。（学生大约有50人。）
4. 東京から大阪まで3時間**ぐらい**かかります。（从东京到大阪大约需要3小时。）
5. 毎日2時間**ぐらい**勉強しています。（每天学习大约2小时。）
:::

:::en
1. 30分**ぐらい**待ちました。(I waited about 30 minutes.)
2. 1000円**くらい**あれば足りる。(About 1000 yen would be enough.)
3. 学生は50人**ぐらい**います。(There are about 50 students.)
4. 東京から大阪まで3時間**ぐらい**かかります。(It takes about 3 hours from Tokyo to Osaka.)
5. 毎日2時間**ぐらい**勉強しています。(I study about 2 hours every day.)
:::

### 例句（② 程度）||Example Sentences (② Degree)

:::zh
1. 泣きたい**くらい**嬉しかった。（高兴得想哭。→ 达到了"想哭"的程度）
2. これ**くらい**なら、一人でできる。（这种程度的话，一个人就能做。）
3. 死ぬ**ほど**疲れた。／ 死ぬ**くらい**疲れた。（累得要死。→ 程度表达）
4. 子供でもわかる**くらい**簡単だ。（简单到小孩都能懂的程度。）
:::

:::en
1. 泣きたい**くらい**嬉しかった。(I was so happy I could cry. → to the degree of wanting to cry)
2. これ**くらい**なら、一人でできる。(If it's about this much, I can do it alone.)
3. 死ぬ**くらい**疲れた。(I'm dead tired. → degree expression)
4. 子供でもわかる**くらい**簡単だ。(It's so simple even a child would understand.)
:::

### 辨析：くらい vs ごろ||Contrast: くらい vs ごろ

:::zh
| | くらい/ぐらい | ごろ |
|---|---|---|
| 意味 | 大约（量/时长） | 大约（时间点） |
| 使い方 | 数量、期間 | 时刻（何時ごろ） |
| 例 | 3時間**ぐらい** | 3時**ごろ** |
| 翻译 | 大约3小时（时长） | 3点左右（时间点） |

> - 3時間**ぐらい**勉強した → 学了大约3小时（**时长**）
> - 3時**ごろ**帰った → 3点左右回来了（**时间点**）
:::

:::en
| | くらい/ぐらい | ごろ |
|---|---|---|
| Meaning | About (quantity / duration) | About (point in time) |
| Usage | Quantities, durations | Clock times (何時ごろ) |
| Example | 3時間**ぐらい** | 3時**ごろ** |
| Translation | About 3 hours (duration) | Around 3 o'clock (point in time) |

> - 3時間**ぐらい**勉強した → studied for about 3 hours (**duration**)
> - 3時**ごろ**帰った → came home around 3 o'clock (**point in time**)
:::

## 5. 〜や〜（など）（部分列举）||5. 〜や〜（など）(Partial Listing)

### 接続

:::zh
名詞 + **や** + 名詞 +（**など**）

> や用于列举**一部分**，暗示还有其他东西。など = 等等（可加可不加）。
:::

:::en
Noun + **や** + Noun + (**など**)

> や lists **some items**, implying there are others. など = "etc." (optional).
:::

### 含义

:::zh
"……和……等（之类的）"。非穷举列举。
:::

:::en
"Things like ~ and ~." A non-exhaustive listing.
:::

### 例句

:::zh
1. りんご**や**バナナ**など**を買いました。（买了苹果和香蕉等。）
2. 週末は映画**や**買い物をします。（周末看电影、买东西之类的。）
3. 日本語**や**英語を勉強しています。（在学日语和英语（等语言）。）
4. 机の上に本**や**ノートがあります。（桌上有书和笔记本等。）
5. 東京**や**大阪**や**京都に行きたい。（想去东京、大阪、京都（等地方）。）
6. 肉**や**野菜**や**果物**など**を食べましょう。（吃肉、蔬菜、水果等吧。）
:::

:::en
1. りんご**や**バナナ**など**を買いました。(I bought apples, bananas, and so on.)
2. 週末は映画**や**買い物をします。(On weekends I do things like watch movies and go shopping.)
3. 日本語**や**英語を勉強しています。(I'm studying Japanese, English, and the like.)
4. 机の上に本**や**ノートがあります。(There are books, notebooks, etc. on the desk.)
5. 東京**や**大阪**や**京都に行きたい。(I want to go to Tokyo, Osaka, Kyoto, and other places.)
6. 肉**や**野菜**や**果物**など**を食べましょう。(Let's eat meat, vegetables, fruit, and so on.)
:::

### 辨析：と vs や||Contrast: と vs や

:::zh
| | と | や |
|---|---|---|
| 列举方式 | **全部列举**（穷举） | **部分列举**（非穷举） |
| 含义 | A和B（只有这些） | A和B等（还有别的） |
| 例 | りんご**と**バナナを買った | りんご**や**バナナを買った |
| 翻译 | 买了苹果和香蕉（就这两样） | 买了苹果和香蕉等（还买了别的） |

> **と** = 穷举清单（全部列出）
> **や** = 非穷举清单（举几个例子，暗示还有其他）

例：
- 冷蔵庫に牛乳**と**卵がある。（冰箱里有牛奶和鸡蛋。→ 只有这两样）
- 冷蔵庫に牛乳**や**卵がある。（冰箱里有牛奶和鸡蛋之类的。→ 还有别的东西）
:::

:::en
| | と | や |
|---|---|---|
| Listing type | **Exhaustive** (all items) | **Partial** (non-exhaustive) |
| Meaning | A and B (that's all) | A and B, etc. (and more) |
| Example | りんご**と**バナナを買った | りんご**や**バナナを買った |
| Translation | Bought apples and bananas (only these) | Bought apples, bananas, etc. (and other things) |

> **と** = exhaustive list (everything listed)
> **や** = non-exhaustive list (examples, implying there are more)

Example:
- 冷蔵庫に牛乳**と**卵がある。(There's milk and eggs in the fridge. → only those two)
- 冷蔵庫に牛乳**や**卵がある。(There's milk, eggs, and stuff in the fridge. → and other things)
:::

## 6. 〜だけ（限定 — "只"）||6. 〜だけ (Limitation — "Only")

### 接続

:::zh
- 名詞 + だけ
- 動詞（辞書形/た形/ない形）+ だけ
- い形容詞 + だけ
- な形容詞 + な + だけ
:::

:::en
- Noun + だけ
- Verb (dictionary / た / ない form) + だけ
- い-adjective + だけ
- な-adjective + な + だけ
:::

### 含義

:::zh
"只、仅仅"。限定范围，表示除此之外没有别的。
:::

:::en
"Only, just." Limits the scope — nothing beyond what is stated.
:::

### 例句

:::zh
1. 水**だけ**飲みました。（只喝了水。）
2. 日本語**だけ**勉強しています。（只在学日语。）
3. 一人**だけ**来ました。（只来了一个人。）
4. 見る**だけ**で、買いません。（只看不买。）
5. ちょっと聞いた**だけ**です。（只是稍微问了一下而已。）
6. 少し**だけ**食べました。（只吃了一点点。）
7. あなた**だけ**に言います。（只对你说。）
8. 週末**だけ**アルバイトをしている。（只在周末打工。）
:::

:::en
1. 水**だけ**飲みました。(I only drank water.)
2. 日本語**だけ**勉強しています。(I'm only studying Japanese.)
3. 一人**だけ**来ました。(Only one person came.)
4. 見る**だけ**で、買いません。(I'm just looking, not buying.)
5. ちょっと聞いた**だけ**です。(I just asked a little, that's all.)
6. 少し**だけ**食べました。(I ate just a little.)
7. あなた**だけ**に言います。(I'll tell only you.)
8. 週末**だけ**アルバイトをしている。(I work part-time only on weekends.)
:::

### 辨析：だけ vs しか〜ない vs ばかり||Contrast: だけ vs しか〜ない vs ばかり

:::zh
| | だけ | しか〜ない | ばかり |
|---|---|---|---|
| 意味 | 只（客观限定） | 只（遗憾/不满足） | 尽（重复/过多） |
| 语气 | 中性 | 消极、觉得不够 | 偏多、偏过度 |
| 否定 | 不需要否定 | **必须接否定** | 不需要否定 |
| 例 | 1000円**だけ**ある | 1000円**しかない** | お菓子**ばかり**食べる |
| 翻译 | 只有1000日元（陈述事实） | 只有1000日元（太少了） | 尽吃零食（吃太多了） |

**对比例句：**
- 水**だけ**飲んだ。（只喝了水。→ 中性事实）
- 水**しか**飲ま**なかった**。（只喝了水。→ 觉得不够/遗憾，别的什么都没喝）
- 水**ばかり**飲んでいる。（尽喝水。→ 光喝水，该吃点东西）

> **しか** 后面**必须**接否定形（ない/ません）。这是最重要的规则。
:::

:::en
| | だけ | しか〜ない | ばかり |
|---|---|---|---|
| Meaning | Only (neutral limitation) | Only (insufficient / regret) | Nothing but (excessive / repetitive) |
| Tone | Neutral | Negative, "not enough" | Too much of one thing |
| Negation | Not required | **Must use negative** | Not required |
| Example | 1000円**だけ**ある | 1000円**しかない** | お菓子**ばかり**食べる |
| Translation | I have only 1000 yen (stating fact) | I only have 1000 yen (not enough!) | Eating nothing but snacks (too much!) |

**Comparison examples:**
- 水**だけ**飲んだ。(I only drank water. → neutral fact)
- 水**しか**飲ま**なかった**。(I only drank water. → not enough, drank nothing else)
- 水**ばかり**飲んでいる。(Drinking nothing but water. → too much water, should eat something)

> **しか** **must** be followed by a negative form (ない/ません). This is the most important rule.
:::
