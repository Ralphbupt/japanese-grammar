## 1. た形（简体过去）||1. た-form (Plain past tense)

:::zh
### 含义
动词的简体过去形，"做了……"。
:::

:::en
### Meaning
The plain past tense of verbs, meaning "did …" or "have done …".
:::

### 好消息||Good news

:::zh
**た形的变形规则和て形完全一样，只是把て→た、で→だ！**
:::

:::en
**The conjugation rules for た-form are exactly the same as て-form — just change て→た and で→だ!**
:::

### 变形规则||Conjugation rules

:::zh
| 辞书形结尾 | て形 | た形 |
|-----------|------|------|
| く | いて | い**た** |
| ぐ | いで | い**だ** |
| す | して | し**た** |
| つ・る・う | って | っ**た** |
| ぬ・ぶ・む | んで | ん**だ** |
| 二类 | 〜て | 〜**た** |
| する | して | し**た** |
| 来る | 来て | 来**た**（きた） |
| 行く | 行って | 行っ**た** |
:::

:::en
| Dictionary ending | て-form | た-form |
|-----------|------|------|
| く | いて | い**た** |
| ぐ | いで | い**だ** |
| す | して | し**た** |
| つ・る・う | って | っ**た** |
| ぬ・ぶ・む | んで | ん**だ** |
| Group 2 | 〜て | 〜**た** |
| する | して | し**た** |
| 来る | 来て | 来**た**（きた） |
| 行く | 行って | 行っ**た** |
:::

### 完整示例||Complete examples

| 辞书形 | て形 | た形 |
|--------|------|------|
| 書く | 書いて | 書い**た** |
| 泳ぐ | 泳いで | 泳い**だ** |
| 話す | 話して | 話し**た** |
| 待つ | 待って | 待っ**た** |
| 帰る | 帰って | 帰っ**た** |
| 買う | 買って | 買っ**た** |
| 飲む | 飲んで | 飲ん**だ** |
| 死ぬ | 死んで | 死ん**だ** |
| 遊ぶ | 遊んで | 遊ん**だ** |
| 食べる | 食べて | 食べ**た** |
| する | して | し**た** |

:::zh
> 你只要会て形，た形就自动会了。这就是为什么Day 2花了大量时间在て形上。
:::

:::en
> Once you know the て-form, you automatically know the た-form. That's why Day 2 spent so much time on て-form.
:::

### た形的用途||Uses of た-form

:::zh
- 简体过去：昨日、映画を見**た**。（昨天看了电影。）
- 修饰名词：昨日見**た**映画は面白かった。（昨天看的电影很有趣。）
- 各种语法接续：たことがある、たばかり、たら……
:::

:::en
- Plain past: 昨日、映画を見**た**。（I watched a movie yesterday.）
- Modifying nouns: 昨日見**た**映画は面白かった。（The movie I watched yesterday was interesting.）
- Various grammar patterns: たことがある, たばかり, たら…
:::

### 例句||Example sentences

:::zh
1. 朝ごはんを食べ**た**。（吃了早饭。）
2. 昨日、友達に会っ**た**。（昨天见了朋友。）
3. さっき電話し**た**けど、出なかった。（刚才打了电话，但没人接。）
4. 母が作っ**た**料理はおいしかった。（妈妈做的菜很好吃。）
5. 先週買っ**た**本をまだ読んでいない。（上周买的书还没看。）
:::

:::en
1. 朝ごはんを食べ**た**。（I ate breakfast.）
2. 昨日、友達に会っ**た**。（I met a friend yesterday.）
3. さっき電話し**た**けど、出なかった。（I called a moment ago, but nobody answered.）
4. 母が作っ**た**料理はおいしかった。（The food my mother made was delicious.）
5. 先週買っ**た**本をまだ読んでいない。（I still haven't read the book I bought last week.）
:::

## 2. 〜たことがある（经验）||2. 〜たことがある (Experience)

### 接続||Conjugation

:::zh
动词た形 + ことがある
:::

:::en
Verb た-form + ことがある
:::

### 含义||Meaning

:::zh
"曾经做过……"，表示过去的经验。
:::

:::en
"Have ever done …", expressing a past experience in one's life.
:::

### 例句||Example sentences

:::zh
1. 日本に行っ**たことがあります**。（我去过日本。）
2. 納豆を食べ**たことがありますか**？（你吃过纳豆吗？）
   — はい、あります。/ いいえ、ありません。
3. 富士山に登っ**たことがある**。（爬过富士山。）
4. あの映画は3回見**たことがある**。（那部电影看过3次。）
5. 一度も海外に行っ**たことがない**。（一次都没出过国。）
6. 馬に乗っ**たことがありますか**？（你骑过马吗？）
7. 日本語で手紙を書い**たことがある**。（用日语写过信。）
:::

:::en
1. 日本に行っ**たことがあります**。（I have been to Japan.）
2. 納豆を食べ**たことがありますか**？（Have you ever eaten natto?）
   — はい、あります。/ いいえ、ありません。
3. 富士山に登っ**たことがある**。（I have climbed Mt. Fuji.）
4. あの映画は3回見**たことがある**。（I have seen that movie three times.）
5. 一度も海外に行っ**たことがない**。（I have never been abroad even once.）
6. 馬に乗っ**たことがありますか**？（Have you ever ridden a horse?）
7. 日本語で手紙を書い**たことがある**。（I have written a letter in Japanese.）
:::

### 注意点||Notes

:::zh
- **否定**：たことが**ない**（没有做过）
- 不能用于最近刚发生的事：❌ 今朝ご飯を食べたことがある → 这不是"经验"
- 必须是"人生中的经历"，有一定时间距离感
:::

:::en
- **Negative**: たことが**ない** (have never done)
- Cannot be used for something that just happened: ❌ 今朝ご飯を食べたことがある → this is not a life "experience"
- Must refer to a lifetime experience with some temporal distance
:::

:::zh
> 判断标准：如果能加上「人生で」（在人生中），就适合用たことがある。
> ✅ 人生で一度だけスカイダイビングをしたことがある。
> ❌ 人生で今朝ご飯を食べたことがある。（奇怪！）
:::

:::en
> Test: if you can naturally add 「人生で」 ("in my life"), the sentence suits たことがある.
> ✅ 人生で一度だけスカイダイビングをしたことがある。（I have gone skydiving once in my life.）
> ❌ 人生で今朝ご飯を食べたことがある。（Sounds strange!）
:::

### 辨析：たことがある vs た||Comparison: たことがある vs た

:::zh
| | た | たことがある |
|---|---|---|
| 含义 | 单纯过去事实 | 人生经验 |
| 语感 | "做了" | "做过" |
| 例 | 日本に行った。（去了日本。） | 日本に行ったことがある。（去过日本。） |
| 时间 | 具体某次 | 不强调哪一次 |
:::

:::en
| | た | たことがある |
|---|---|---|
| Meaning | Simple past fact | Life experience |
| Nuance | "did" | "have done" |
| Example | 日本に行った。（I went to Japan.） | 日本に行ったことがある。（I have been to Japan.） |
| Time | A specific occasion | No emphasis on which time |
:::

## 3. 〜たり〜たりする（列举动作）||3. 〜たり〜たりする (Listing actions)

### 接続||Conjugation

:::zh
动词た形 + り …… 動詞た形 + り + する
:::

:::en
Verb た-form + り … verb た-form + り + する
:::

### 含义||Meaning

:::zh
"又做A又做B（之类的）"，从多个动作中举出代表性的几个。

> 暗示"不只做了这些，还做了其他事"，和て形并列不同。
:::

:::en
"Do things like A and B", listing a few representative actions from a larger set.

> It implies "not only these things, but also other things" — unlike て-form listing which is exhaustive.
:::

### 例句||Example sentences

:::zh
1. 休みの日は本を読ん**だり**、映画を見**たりします**。（休息日看看书、看看电影什么的。）
2. 週末は友達と遊ん**だり**、買い物し**たりした**。（周末和朋友玩了玩、逛了逛街。）
3. 最近、天気が暑かっ**たり**、寒かっ**たりする**。（最近天气忽冷忽热。）
4. この店は安かっ**たり**、高かっ**たりする**。（这家店价格忽高忽低。）
5. 電車の中で音楽を聴い**たり**、ゲームをし**たりしています**。（在电车上听听音乐、玩玩游戏。）
6. 日曜日は掃除し**たり**、散歩し**たりします**。（周日打扫打扫、散散步什么的。）
7. 旅行では写真を撮っ**たり**、おいしい物を食べ**たりした**。（旅行中拍拍照、吃吃好吃的。）
:::

:::en
1. 休みの日は本を読ん**だり**、映画を見**たりします**。（On days off I do things like read books and watch movies.）
2. 週末は友達と遊ん**だり**、買い物し**たりした**。（Over the weekend I hung out with friends, went shopping, and so on.）
3. 最近、天気が暑かっ**たり**、寒かっ**たりする**。（The weather has been alternately hot and cold lately.）
4. この店は安かっ**たり**、高かっ**たりする**。（The prices at this shop go up and down.）
5. 電車の中で音楽を聴い**たり**、ゲームをし**たりしています**。（On the train I listen to music, play games, and such.）
6. 日曜日は掃除し**たり**、散歩し**たりします**。（On Sundays I clean the house, go for walks, and the like.）
7. 旅行では写真を撮っ**たり**、おいしい物を食べ**たりした**。（During the trip I took photos, ate delicious food, and more.）
:::

### 形容词/名词版||Adjective / noun version

:::zh
| 词类 | たり形 | 例子 |
|------|--------|------|
| い形容词 | い→かったり | 暑い→暑かったり |
| な形容词 | だ→だったり | 暇だ→暇だったり |
| 名词 | だ→だったり | 雨だ→雨だったり |

- 天気は晴れ**だったり**、雨**だったりする**。（天气时晴时雨。）
- 気分が良かっ**たり**、悪かっ**たりする**。（心情时好时坏。）
:::

:::en
| Word type | たり form | Example |
|------|--------|------|
| い-adjective | い→かったり | 暑い→暑かったり |
| な-adjective | だ→だったり | 暇だ→暇だったり |
| Noun | だ→だったり | 雨だ→雨だったり |

- 天気は晴れ**だったり**、雨**だったりする**。（The weather alternates between sunny and rainy.）
- 気分が良かっ**たり**、悪かっ**たりする**。（My mood goes back and forth between good and bad.）
:::

### 注意点||Notes

:::zh
- 最后的「する」不能省略（虽然口语偶尔省）
- たり至少要两个才自然，一个的话有点突兀
- する会根据时态变化：したりする（现在）、したりした（过去）
:::

:::en
- The final 「する」 cannot be omitted (though casual speech sometimes does)
- At least two たり items are needed to sound natural; one alone feels abrupt
- する changes with tense: したりする (present), したりした (past)
:::

### 辨析：たり vs て形并列||Comparison: たり vs て-form listing

:::zh
| | たり〜たりする | て形并列 |
|---|---|---|
| 列举 | 选择性举例，暗示还有其他 | 全部列出或按顺序 |
| 顺序 | 不强调顺序 | 有时间顺序 |
| 例 | 本を読んだり映画を見たりする（看书看电影什么的） | 本を読んで映画を見た（先看书再看电影） |
| 语气 | 轻松随意 | 客观叙述 |
:::

:::en
| | たり〜たりする | て-form listing |
|---|---|---|
| Listing | Selective examples, implies more | All actions listed or in sequence |
| Order | No emphasis on order | Sequential / chronological |
| Example | 本を読んだり映画を見たりする（read books, watch movies, etc.） | 本を読んで映画を見た（first read, then watched） |
| Tone | Casual, relaxed | Objective narration |
:::

## 4. 〜たばかり（刚刚做完）||4. 〜たばかり (Just finished doing)

### 接続||Conjugation

:::zh
动词た形 + ばかり
:::

:::en
Verb た-form + ばかり
:::

### 含義||Meaning

:::zh
"刚刚做完……"，强调时间上刚发生不久。
:::

:::en
"Just did …", emphasising that the action happened a short time ago.
:::

### 例句||Example sentences

:::zh
1. さっき食べ**たばかり**なのに、もうお腹が空いた。（刚吃完，肚子又饿了。）
2. 日本に来**たばかり**で、まだ何もわからない。（刚来日本，什么都不懂。）
3. この本は買っ**たばかり**です。（这本书是刚买的。）
4. 彼女と別れ**たばかり**だ。（刚和她分手。）
5. 引っ越し**たばかり**で、まだ荷物がたくさんある。（刚搬家，还有很多行李。）
6. 大学を卒業し**たばかり**の新入社員です。（刚大学毕业的新员工。）
7. 今起き**たばかり**だから、まだ眠い。（刚起床，还很困。）
:::

:::en
1. さっき食べ**たばかり**なのに、もうお腹が空いた。（I just finished eating, but I'm already hungry again.）
2. 日本に来**たばかり**で、まだ何もわからない。（I just came to Japan and still don't understand anything.）
3. この本は買っ**たばかり**です。（I just bought this book.）
4. 彼女と別れ**たばかり**だ。（I just broke up with her.）
5. 引っ越し**たばかり**で、まだ荷物がたくさんある。（I just moved and still have a lot of boxes.）
6. 大学を卒業し**たばかり**の新入社員です。（A new employee who just graduated from university.）
7. 今起き**たばかり**だから、まだ眠い。（I just woke up, so I'm still sleepy.）
:::

### 辨析：たばかり vs たところ||Comparison: たばかり vs たところ

:::zh
| | たばかり | たところ |
|---|---|---|
| 含义 | 主观感觉"刚刚" | 客观上刚刚完成 |
| 时间幅度 | 可以是几小时甚至几天前 | 几乎是"此刻" |
| 例 | 先月引っ越したばかりだ。（上个月刚搬的家。→ 主观上觉得刚搬） | 今ご飯を食べたところだ。（现在刚吃完饭。→ 就是此刻） |
:::

:::en
| | たばかり | たところ |
|---|---|---|
| Meaning | Subjective feeling of "just" | Objectively just completed |
| Time span | Can be hours or even days ago | Almost "this very moment" |
| Example | 先月引っ越したばかりだ。（I moved last month — still feels recent.） | 今ご飯を食べたところだ。（I literally just finished eating.） |
:::

:::zh
> **たばかり**可以用于离现在有一段距离的事，只要说话人主观觉得"还是刚发生的"就行。
> **たところ**只能用于真正"刚才那一瞬间"完成的事。
:::

:::en
> **たばかり** can refer to something that happened a while ago, as long as the speaker subjectively feels "it still feels recent".
> **たところ** is only for something that was completed at literally that very moment.
:::
