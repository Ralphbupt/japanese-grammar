# Day 46 – 状态样态 — まま/だらけ/っぱなし/ずに/ように見える

:::zh
这组语法描述事物的状态或样态：保持原状、满是某物、放置不管、省略动作、外观判断。日常生活和考试中都非常常见。
:::

:::en
This group describes states and appearances: maintaining a condition, being covered in something, leaving things as-is, omitting an action, and judging by appearance. Very common in daily life and exams.
:::

## 1. 〜まま（保持原状/就那样~）||1. 〜まま (Remaining as-is / Just like that)

### 接続||Conjugation
:::zh
| 接续 | 形式 |
|------|------|
| 动词た形 | 食べた + まま |
| 动词ない形 | 食べない + まま |
| い形容词 | 小さい + まま |
| な形容词 | 静かな + まま |
| 名词の | 子供の + まま |
:::
:::en
| Conjugation | Form |
|------|------|
| Verb た-form | 食べた + まま |
| Verb ない-form | 食べない + まま |
| い-adjective | 小さい + まま |
| な-adjective | 静かな + まま |
| Noun の | 子供の + まま |
:::

### 含义||Meaning
:::zh
表示某个状态一直保持不变，没有发生预期的变化。"就那样……""保持着……的状态"。
:::
:::en
Indicates that a certain state remains unchanged without the expected change occurring. "Remaining in the state of..." / "Just as it is..."
:::

### 例句||Example Sentences
:::zh
1. 靴を履いた**まま**、家に入らないでください。（不要穿着鞋就进家门。）
2. 電気をつけた**まま**寝てしまった。（开着灯就睡着了。）
3. 窓を開けた**まま**出かけてしまった。（窗户开着就出门了。）
4. 子供の**まま**大人になれない人もいる。（也有人保持孩子的状态长不大。）
5. 朝ご飯を食べない**まま**、出勤した。（没吃早饭就去上班了。）
6. あの二人は会わない**まま**、10年が過ぎた。（那两人没见面就过了10年。）
:::
:::en
1. 靴を履いた**まま**、家に入らないでください。(Please don't enter the house with your shoes still on.)
2. 電気をつけた**まま**寝てしまった。(I fell asleep with the lights on.)
3. 窓を開けた**まま**出かけてしまった。(I went out with the window left open.)
4. 子供の**まま**大人になれない人もいる。(Some people can't grow up and remain childlike.)
5. 朝ご飯を食べない**まま**、出勤した。(I went to work without eating breakfast.)
6. あの二人は会わない**まま**、10年が過ぎた。(Ten years passed without those two meeting.)
:::

### 辨析||Comparison
:::zh
| 比较 | 含义 | 区别 |
|------|------|------|
| まま | 保持原状 | 强调没有变化 |
| っぱなし | 放置不管 | 含有"不好""该处理却没处理"的负面语气 |
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| まま | Remaining as-is | Emphasizes no change |
| っぱなし | Left unattended | Carries a negative nuance of neglecting to deal with something |
:::

## 2. 〜だらけ（满是~/全是~）||2. 〜だらけ (Covered with / Full of)

### 接続||Conjugation
:::zh
名词 + だらけ
:::
:::en
Noun + だらけ
:::

### 含义||Meaning
:::zh
表示到处都是不好的东西，带有负面含义。"满是……""全是……"。
:::
:::en
Indicates being full of or covered with something undesirable. Carries a negative connotation. "Full of..." / "Covered with..."
:::

### 例句||Example Sentences
:::zh
1. 部屋はほこり**だらけ**だ。（房间里全是灰尘。）
2. 子供が泥**だらけ**になって帰ってきた。（孩子浑身是泥地回来了。）
3. このレポートは間違い**だらけ**だ。（这份报告全是错误。）
4. 手が傷**だらけ**になった。（手上全是伤痕。）
5. 引っ越したばかりで、部屋は段ボール**だらけ**だ。（刚搬家，房间里全是纸箱。）
:::
:::en
1. 部屋はほこり**だらけ**だ。(The room is covered in dust.)
2. 子供が泥**だらけ**になって帰ってきた。(The child came home covered in mud.)
3. このレポートは間違い**だらけ**だ。(This report is full of mistakes.)
4. 手が傷**だらけ**になった。(My hands became covered in scratches.)
5. 引っ越したばかりで、部屋は段ボール**だらけ**だ。(I just moved, so the room is full of cardboard boxes.)
:::

### 辨析||Comparison
:::zh
| 比较 | 含义 | 区别 |
|------|------|------|
| だらけ | 满是~ | 负面、不好的东西 |
| まみれ | 沾满~ | 仅限物理上的附着（泥、血等），更强调表面 |
| ずくめ | 全是~ | 可正面可负面，多用于固定搭配（黒ずくめ/いいことずくめ） |
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| だらけ | Full of | Negative, undesirable things |
| まみれ | Smeared with | Only physical substances (mud, blood), emphasis on surface |
| ずくめ | All / entirely | Can be positive or negative, often in fixed expressions (黒ずくめ/いいことずくめ) |
:::

## 3. 〜っぱなし（一直~着/放置不管）||3. 〜っぱなし (Left doing / Left as-is)

### 接続||Conjugation
:::zh
动词ます形（去ます） + っぱなし
:::
:::en
Verb ます-stem (drop ます) + っぱなし
:::

### 含义||Meaning
:::zh
表示做了某动作后一直放着不管，或某状态持续未改变。带有"不应该这样"的负面语气。
:::
:::en
Indicates that after doing something, it was left unattended, or a state continues unchanged. Carries a negative nuance of "this shouldn't be so."
:::

### 例句||Example Sentences
:::zh
1. 水を出し**っぱなし**にしないでください。（请不要一直开着水龙头不关。）
2. ドアを開け**っぱなし**にしている。（门一直开着没关。）
3. テレビをつけ**っぱなし**で寝てしまった。（电视开着就睡了。）
4. 電車で2時間立ち**っぱなし**だった。（在电车上站了整整2小时。）
5. 彼は一日中しゃべり**っぱなし**だ。（他一整天都在不停地说。）
:::
:::en
1. 水を出し**っぱなし**にしないでください。(Please don't leave the water running.)
2. ドアを開け**っぱなし**にしている。(The door has been left open.)
3. テレビをつけ**っぱなし**で寝てしまった。(I fell asleep with the TV left on.)
4. 電車で2時間立ち**っぱなし**だった。(I was standing for two hours straight on the train.)
5. 彼は一日中しゃべり**っぱなし**だ。(He's been talking nonstop all day.)
:::

## 4. 〜ずに（不做~就~ =ないで书面）||4. 〜ずに (Without doing ~ = formal ないで)

### 接続||Conjugation
:::zh
| 接续 | 形式 |
|------|------|
| 动词ない形（去ない） | 食べ → 食べずに |
| する的特殊形式 | する → **せず**に |
:::
:::en
| Conjugation | Form |
|------|------|
| Verb ない-stem (drop ない) | 食べ → 食べずに |
| Special form of する | する → **せず**に |
:::

### 含义||Meaning
:::zh
"没有做……就……"，是「ないで」的书面/正式表达。
:::
:::en
"Without doing..." — the formal/written equivalent of 「ないで」.
:::

### 例句||Example Sentences
:::zh
1. 朝食を食べ**ずに**学校に行った。（没吃早饭就去了学校。）
2. 何も言わ**ずに**出て行った。（什么都没说就走了。）
3. 傘を持た**ずに**出かけて、濡れてしまった。（没带伞就出门，被淋湿了。）
4. 辞書を使わ**ずに**読めるようになりたい。（想不用词典也能读懂。）
5. 彼は休ま**ずに**働き続けた。（他不休息地持续工作。）
6. 予習**せずに**授業に出た。（没预习就去上课了。）
:::
:::en
1. 朝食を食べ**ずに**学校に行った。(I went to school without eating breakfast.)
2. 何も言わ**ずに**出て行った。(He left without saying anything.)
3. 傘を持た**ずに**出かけて、濡れてしまった。(I went out without an umbrella and got soaked.)
4. 辞書を使わ**ずに**読めるようになりたい。(I want to be able to read without using a dictionary.)
5. 彼は休ま**ずに**働き続けた。(He kept working without resting.)
6. 予習**せずに**授業に出た。(I attended class without preparing.)
:::

### 辨析||Comparison
:::zh
| 比较 | 含义 | 区别 |
|------|------|------|
| ずに | 没做~就~ | 书面/正式表达 |
| ないで | 没做~就~ | 口语表达 |
| ずに vs ないで | 意思相同 | ずに更正式；する→せずに是特殊变化 |
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| ずに | Without doing | Written/formal |
| ないで | Without doing | Spoken/casual |
| ずに vs ないで | Same meaning | ずに is more formal; する→せずに is irregular |
:::

## 5. 〜ように見える（看起来像~）||5. 〜ように見える (Looks like / Appears to be)

### 接続||Conjugation
:::zh
| 接续 | 形式 |
|------|------|
| 动词普通形 | 泣いている + ように見える |
| い形容词 | 若い + ように見える |
| な形容词 | 元気な / 元気の + ように見える |
| 名词の | 学生の + ように見える |
:::
:::en
| Conjugation | Form |
|------|------|
| Verb plain form | 泣いている + ように見える |
| い-adjective | 若い + ように見える |
| な-adjective | 元気な / 元気の + ように見える |
| Noun の | 学生の + ように見える |
:::

### 含义||Meaning
:::zh
根据外观或感觉做出判断。"看上去像……""看起来好像……"。
:::
:::en
Making a judgment based on appearance or impression. "Looks like..." / "Appears to be..."
:::

### 例句||Example Sentences
:::zh
1. 彼女は若く**見える**けど、実は40歳だ。（她看起来年轻，其实40岁了。）
2. 彼は幸せそうに**見える**が、実は悩んでいる。（他看起来很幸福，其实在烦恼。）
3. この絵は写真の**ように見える**。（这幅画看起来像照片。）
4. 彼は何も食べていない**ように見える**。（他看起来什么都没吃。）
5. あの建物は新しい**ように見える**が、実は築50年だ。（那栋建筑看起来很新，其实有50年了。）
:::
:::en
1. 彼女は若く**見える**けど、実は40歳だ。(She looks young, but she's actually 40.)
2. 彼は幸せそうに**見える**が、実は悩んでいる。(He appears happy, but he's actually troubled.)
3. この絵は写真の**ように見える**。(This painting looks like a photograph.)
4. 彼は何も食べていない**ように見える**。(He looks like he hasn't eaten anything.)
5. あの建物は新しい**ように見える**が、実は築50年だ。(That building looks new, but it's actually 50 years old.)
:::

### 辨析||Comparison
:::zh
| 比较 | 含义 | 区别 |
|------|------|------|
| ように見える | 看起来像~ | 基于视觉的客观判断 |
| そうだ（様態） | 看上去要~ | 基于直觉/征兆的推测（雨が降りそうだ） |
| らしい | 好像~的样子 | 基于间接信息的推测 |
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| ように見える | Looks like | Objective judgment based on visual appearance |
| そうだ (appearance) | Looks like it will | Intuitive guess based on signs (雨が降りそうだ) |
| らしい | Seems like | Inference based on indirect information |
:::

## 练习||Practice

:::zh
**一、用まま、だらけ、っぱなし填空：**

1. エアコンをつけ（　　）にして出かけてしまった。
2. 服が汚れ（　　）だから、洗わなきゃ。
3. メガネをかけた（　　）シャワーを浴びてしまった。
4. 彼女は座り（　　）で腰が痛くなった。

**二、用ずに改写以下句子：**

5. 何も食べないで寝た。→ 何も（　　）寝た。
6. 勉強しないでテストを受けた。→ 勉強（　　）テストを受けた。

**三、用ように見える完成句子：**

7. 彼は疲れている（　　）。（他看起来很累。）
8. この料理は簡単な（　　）が、実は難しい。（这道菜看起来简单，其实很难。）

**四、选择正确答案：**

9. 子供が泥（まま・だらけ・っぱなし）になって遊んでいる。
10. ドアを開けた（まま・だらけ・っぱなし）にしておいてください。
:::
:::en
**I. Fill in the blanks with まま, だらけ, or っぱなし:**

1. エアコンをつけ（　　）にして出かけてしまった。
2. 服が汚れ（　　）だから、洗わなきゃ。
3. メガネをかけた（　　）シャワーを浴びてしまった。
4. 彼女は座り（　　）で腰が痛くなった。

**II. Rewrite the following sentences using ずに:**

5. 何も食べないで寝た。→ 何も（　　）寝た。
6. 勉強しないでテストを受けた。→ 勉強（　　）テストを受けた。

**III. Complete the sentences using ように見える:**

7. 彼は疲れている（　　）。(He looks tired.)
8. この料理は簡単な（　　）が、実は難しい。(This dish looks easy but is actually difficult.)

**IV. Choose the correct answer:**

9. 子供が泥（まま・だらけ・っぱなし）になって遊んでいる。
10. ドアを開けた（まま・だらけ・っぱなし）にしておいてください。
:::
