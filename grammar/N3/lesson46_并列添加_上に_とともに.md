# Day 43 – 并列添加 — 上に/だけでなく〜も/はもちろん/をはじめ/とともに/に伴い

:::zh
这组语法都表示"不止A，还有B"或"A和B同时"。从简单的添加到正式的书面并列，掌握它们能让表达层次更丰富。
:::

This group expresses "not only A but also B" or "A and B simultaneously." From simple addition to formal written enumeration, mastering them enriches your expressiveness.

## 1. 〜上に（而且~/加之~）||1. 〜上に (Moreover~ / In addition to~)

### 接続||Conjugation
:::zh
| 词类 | 接续 | 例 |
|------|------|-----|
| 動詞普通形 | + 上に | 頭がいい**上に** |
| い形容詞 | + 上に | 安い**上に** |
| な形容詞 + な/である | + 上に | 静か**な上に** |
| 名詞 + の/である | + 上に | 学生**の上に** |
:::
:::en
| Word type | Conjugation | Example |
|-----------|-------------|---------|
| Verb plain form | + 上に | 頭がいい**上に** |
| い-adjective | + 上に | 安い**上に** |
| な-adjective + な/である | + 上に | 静か**な上に** |
| Noun + の/である | + 上に | 学生**の上に** |
:::

### 含义||Meaning
:::zh
"不仅A，而且还B"。前后通常是同方向的评价（都好或都坏）。
:::
:::en
"Not only A, but also B on top of that." Front and back are usually evaluations in the same direction (both positive or both negative).
:::

### 例句||Example Sentences
:::zh
1. この店は安い**上に**、おいしい。（这家店又便宜又好吃。）
2. 彼は成績が優秀な**上に**、スポーツも得意だ。（他成绩优秀，而且运动也擅长。）
3. 道に迷った**上に**、雨まで降ってきた。（迷了路，加之又下起了雨。）
4. 彼女は歌が上手な**上に**、ダンスも得意だ。（她唱歌好，而且跳舞也擅长。）
5. この部屋は広い**上に**、日当たりもいい。（这个房间又大又朝阳。）
:::
:::en
1. この店は安い**上に**、おいしい。(This restaurant is not only cheap but also delicious.)
2. 彼は成績が優秀な**上に**、スポーツも得意だ。(He has excellent grades and is good at sports on top of that.)
3. 道に迷った**上に**、雨まで降ってきた。(I got lost, and on top of that it started raining.)
4. 彼女は歌が上手な**上に**、ダンスも得意だ。(She's a good singer and a good dancer on top of that.)
5. この部屋は広い**上に**、日当たりもいい。(This room is spacious and gets good sunlight too.)
:::

### 辨析||Comparison
:::zh
| 表达 | 含义 | 区别 |
|------|------|------|
| 〜上に | 而且/加之 | 同方向评价叠加 |
| 〜し〜し | 又~又~ | 口语，列举理由 |
| 〜だけでなく | 不仅~ | 前后方向不限 |

- 安い**上に**おいしい → 正式地叠加好评
- 安い**し**おいしい**し** → 口语列举
:::
:::en
| Expression | Meaning | Difference |
|------------|---------|------------|
| 〜上に | Moreover | Same-direction evaluations stacked |
| 〜し〜し | And also~ and~ | Colloquial, listing reasons |
| 〜だけでなく | Not only~ | Direction of front/back unrestricted |

- 安い**上に**おいしい → formally stacking positive points
- 安い**し**おいしい**し** → colloquial listing
:::

## 2. 〜だけでなく〜も（不仅~还~）||2. 〜だけでなく〜も (Not only~ but also~)

### 接続||Conjugation
:::zh
| 词类 | 接续 |
|------|------|
| 動詞普通形 | + だけでなく |
| い形容詞 | + だけでなく |
| な形容詞 + な/である | + だけでなく |
| 名詞 | + だけでなく |
:::
:::en
| Word type | Conjugation |
|-----------|-------------|
| Verb plain form | + だけでなく |
| い-adjective | + だけでなく |
| な-adjective + な/である | + だけでなく |
| Noun | + だけでなく |
:::

### 含义||Meaning
:::zh
"不仅A，B也……"。最基本的并列添加表达，口语书面都能用。
:::
:::en
"Not only A, but B also…" The most basic additive expression, usable in both speech and writing.
:::

### 例句||Example Sentences
:::zh
1. 彼は日本語**だけでなく**、中国語**も**話せる。（他不仅会日语，中文也会。）
2. この問題は子ども**だけでなく**、大人**にも**関係がある。（这个问题不仅跟孩子有关，跟大人也有关。）
3. 国内**だけでなく**、海外**でも**活躍している。（不仅在国内，在海外也很活跃。）
4. 安い**だけでなく**、品質**も**いい。（不仅便宜，质量也好。）
5. 見た目がきれいな**だけでなく**、味**も**おいしい。（不仅外观漂亮，味道也好。）
:::
:::en
1. 彼は日本語**だけでなく**、中国語**も**話せる。(He can speak not only Japanese but also Chinese.)
2. この問題は子ども**だけでなく**、大人**にも**関係がある。(This issue concerns not only children but adults as well.)
3. 国内**だけでなく**、海外**でも**活躍している。(Active not only domestically but also overseas.)
4. 安い**だけでなく**、品質**も**いい。(Not only cheap but also good quality.)
5. 見た目がきれいな**だけでなく**、味**も**おいしい。(Not only beautiful to look at, but also delicious.)
:::

## 3. 〜はもちろん（~自不必说）||3. 〜はもちろん (~goes without saying)

### 接続||Conjugation
:::zh
名詞 + はもちろん（、〜も）
:::
:::en
Noun + はもちろん (, 〜も)
:::

### 含义||Meaning
:::zh
"A自不必说，B也……"。先说理所当然的事A，再添加超出预期的B。
:::
:::en
"A goes without saying, and B as well." First states something obvious (A), then adds something beyond expectation (B).
:::

### 例句||Example Sentences
:::zh
1. 日本語**はもちろん**、英語も話せる。（日语自不必说，英语也会。）
2. 平日**はもちろん**、休日も働いている。（平日自不必说，休息日也在工作。）
3. 大人**はもちろん**、子どもも楽しめる映画だ。（大人自不必说，孩子也能享受的电影。）
4. 味**はもちろん**、見た目も美しい料理だ。（味道自不必说，外观也很美的料理。）
5. 国内**はもちろん**、海外からも注文が来る。（国内自不必说，海外也有订单来。）
:::
:::en
1. 日本語**はもちろん**、英語も話せる。(Japanese goes without saying — she can speak English too.)
2. 平日**はもちろん**、休日も働いている。(Not just weekdays, of course — he works on holidays too.)
3. 大人**はもちろん**、子どもも楽しめる映画だ。(A movie enjoyable by children, not just adults of course.)
4. 味**はもちろん**、見た目も美しい料理だ。(The taste goes without saying — the presentation is beautiful too.)
5. 国内**はもちろん**、海外からも注文が来る。(Orders come from overseas too, not just domestically of course.)
:::

## 4. 〜をはじめ（以~为首）||4. 〜をはじめ (Starting with~ / Including~)

### 接続||Conjugation
:::zh
名詞 + をはじめ（として）
:::
:::en
Noun + をはじめ(として)
:::

### 含义||Meaning
:::zh
"以A为首，还有B、C等"。列举代表性事物后暗示还有更多。正式书面表达。
:::
:::en
"Starting with A, and also B, C, etc." Lists a representative item, implying there are more. Formal, written expression.
:::

### 例句||Example Sentences
:::zh
1. 大臣**をはじめ**、多くの関係者が出席した。（以大臣为首，很多相关人员出席了。）
2. 東京**をはじめ**、大阪や名古屋にも支店がある。（以东京为首，大阪和名古屋也有分店。）
3. 日本語**をはじめ**、英語や中国語も勉強している。（以日语为首，英语和中文也在学。）
4. 桜**をはじめ**、いろいろな花が咲いている。（以樱花为首，各种花都开了。）
5. サッカー**をはじめとする**スポーツが人気だ。（以足球为首的运动很受欢迎。）
:::
:::en
1. 大臣**をはじめ**、多くの関係者が出席した。(Many officials attended, starting with the minister.)
2. 東京**をはじめ**、大阪や名古屋にも支店がある。(There are branches in Osaka and Nagoya, starting with Tokyo.)
3. 日本語**をはじめ**、英語や中国語も勉強している。(I'm studying English and Chinese as well, starting with Japanese.)
4. 桜**をはじめ**、いろいろな花が咲いている。(Various flowers are blooming, starting with cherry blossoms.)
5. サッカー**をはじめとする**スポーツが人気だ。(Sports starting with soccer are popular.)
:::

## 5. 〜とともに（和~一起/随着~）||5. 〜とともに (Together with~ / Along with~)

### 接続||Conjugation
:::zh
| 含义 | 接续 |
|------|------|
| 和〜一起 | 名詞 + とともに |
| 随着〜 | 動詞辞書形/名詞 + とともに |
:::
:::en
| Meaning | Conjugation |
|---------|-------------|
| Together with | Noun + とともに |
| Along with (change) | Verb dictionary form / Noun + とともに |
:::

### 含义||Meaning
:::zh
① "和A一起"（共同动作）
② "随着A变化，B也变化"（同时变化）
:::
:::en
① "Together with A" (joint action)
② "As A changes, B also changes" (simultaneous change)
:::

### 例句||Example Sentences
:::zh
1. 家族**とともに**日本に来た。（和家人一起来了日本。）
2. 技術の発展**とともに**、生活が便利になった。（随着技术发展，生活变得方便了。）
3. 時代**とともに**、人々の考え方も変わる。（随着时代变迁，人们的想法也在改变。）
4. 経済の成長**とともに**、環境問題も深刻になった。（随着经济增长，环境问题也变严重了。）
5. 年を取る**とともに**、体力が落ちてきた。（随着年龄增长，体力下降了。）
:::
:::en
1. 家族**とともに**日本に来た。(I came to Japan together with my family.)
2. 技術の発展**とともに**、生活が便利になった。(Along with technological development, life has become more convenient.)
3. 時代**とともに**、人々の考え方も変わる。(People's ways of thinking change with the times.)
4. 経済の成長**とともに**、環境問題も深刻になった。(Along with economic growth, environmental problems have become serious.)
5. 年を取る**とともに**、体力が落ちてきた。(As I age, my physical strength has declined.)
:::

## 6. 〜に伴い / 〜に伴って（伴随着~）||6. 〜に伴い / 〜に伴って (Accompanying~ / As~ happens)

### 接続||Conjugation
:::zh
動詞辞書形 / 名詞 + に伴い / に伴って

修飾名詞時：〜に伴う + 名詞
:::
:::en
Verb dictionary form / Noun + に伴い / に伴って

Before a noun: 〜に伴う + Noun
:::

### 含义||Meaning
:::zh
"伴随着A的发生/变化，B也发生/变化"。比「とともに」更正式，多用于新闻和学术文章。
:::
:::en
"Accompanying the occurrence/change of A, B also occurs/changes." More formal than 「とともに」, often used in news and academic writing.
:::

### 例句||Example Sentences
:::zh
1. 人口の増加**に伴い**、住宅問題が深刻になった。（伴随人口增加，住房问题变严重了。）
2. 高齢化が進む**に伴って**、医療費が増えている。（伴随着老龄化加剧，医疗费在增加。）
3. 景気の悪化**に伴う**失業者の増加が問題だ。（伴随景气恶化的失业者增加是问题。）
4. 技術の進歩**に伴い**、新しい仕事が生まれた。（伴随技术进步，产生了新工作。）
5. 台風の接近**に伴って**、風が強くなってきた。（伴随台风接近，风变强了。）
:::
:::en
1. 人口の増加**に伴い**、住宅問題が深刻になった。(Accompanying population growth, the housing problem became serious.)
2. 高齢化が進む**に伴って**、医療費が増えている。(As aging progresses, medical costs are increasing.)
3. 景気の悪化**に伴う**失業者の増加が問題だ。(The increase in unemployment accompanying the economic downturn is a problem.)
4. 技術の進歩**に伴い**、新しい仕事が生まれた。(New jobs have emerged with technological progress.)
5. 台風の接近**に伴って**、風が強くなってきた。(As the typhoon approached, the wind grew stronger.)
:::

### 辨析||Comparison
:::zh
| 表达 | 含义 | 正式度 |
|------|------|--------|
| 〜とともに | 随着~/和~一起 | 书面 |
| 〜に伴い | 伴随着~ | 更正式（新闻/论文） |
| 〜につれて | 随着~ | 口语也可用 |
| 〜に従い | 随着~ | 正式，强调规律性 |

- 技術の発展**とともに** → 正式的"随着"
- 技術の発展**に伴い** → 更正式，强调因果关系
- 技術が発展する**につれて** → 口语/书面均可
:::
:::en
| Expression | Meaning | Formality |
|------------|---------|-----------|
| 〜とともに | Along with / together with | Written |
| 〜に伴い | Accompanying | More formal (news/papers) |
| 〜につれて | As~ | Both spoken/written |
| 〜に従い | As~ | Formal, emphasizes regularity |

- 技術の発展**とともに** → formal "along with"
- 技術の発展**に伴い** → more formal, emphasizes cause-effect
- 技術が発展する**につれて** → both spoken and written
:::
