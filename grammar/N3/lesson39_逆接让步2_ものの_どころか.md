## 1. 〜にしても（即使……也/就算……也）||1. 〜にしても (Even if / Even so…)

### 接続||Conjugation
:::zh
- 動詞普通形 + にしても
- い形容詞 + にしても
- な形容詞 + だ / である + にしても
- 名詞 + だ / である + にしても
:::
:::en
- Verb (plain form) + にしても
- i-adjective + にしても
- na-adjective + だ / である + にしても
- Noun + だ / である + にしても
:::

### 含义||Meaning
:::zh
"即使……也……"，"就算……也……"。承认前项为事实或可能，但后项依然成立。语气比「としても」更贴近实际情况，常用于已知事实的让步。
:::
:::en
"Even if / even so…" — acknowledges the preceding as fact or possibility, but the following still holds. Closer to actual situations than 「としても」, often used for concession of known facts.
:::

### 例句||Example Sentences
:::zh
1. 賛成する**にしても**、条件がある。（即使赞成，也有条件。）
2. 安い**にしても**、質が悪ければ買わない。（即使便宜，质量差的话也不买。）
3. 忙しい**にしても**、連絡くらいはできるだろう。（即使忙，联系一下总做得到吧。）
4. 冗談だった**にしても**、言い過ぎだ。（就算是玩笑，也说过头了。）
5. 転職する**にしても**、もう少し考えたほうがいい。（即使要跳槽，也应该再考虑一下。）
:::
:::en
1. 賛成する**にしても**、条件がある。(Even if I agree, I have conditions.)
2. 安い**にしても**、質が悪ければ買わない。(Even if it's cheap, I won't buy it if the quality is poor.)
3. 忙しい**にしても**、連絡くらいはできるだろう。(Even if you're busy, you could at least get in touch.)
4. 冗談だった**にしても**、言い過ぎだ。(Even if it was a joke, that was going too far.)
5. 転職する**にしても**、もう少し考えたほうがいい。(Even if you're going to change jobs, you should think a bit more.)
:::

### 辨析||Comparison
:::zh
| としても | にしても |
|---------|---------|
| 纯假设（可能不会发生） | 更接近事实（很可能发生） |
| たとえ失敗した**としても** | 失敗した**にしても**まだチャンスはある |
| "即使（万一）" | "就算（实际上）" |
| 常与「たとえ」搭配 | 更独立使用 |
:::
:::en
| としても | にしても |
|---------|---------|
| Pure hypothesis (may not happen) | Closer to reality (likely to happen) |
| たとえ失敗した**としても** | 失敗した**にしても**まだチャンスはある |
| "Even if (hypothetically)" | "Even so (in reality)" |
| Often paired with 「たとえ」 | Used more independently |
:::

## 2. 〜ものの（虽然……但是……）||2. 〜ものの (Although / …but…)

### 接続||Conjugation
:::zh
- 動詞普通形 + ものの
- い形容詞 + ものの
- な形容詞 + な / である + ものの
- 名詞 + である + ものの
:::
:::en
- Verb (plain form) + ものの
- i-adjective + ものの
- na-adjective + な / である + ものの
- Noun + である + ものの
:::

### 含义||Meaning
:::zh
"虽然……但是……"，承认前项是事实，但后项的结果与预期不同。书面语。常表达说话者的困惑、为难或意外。
:::
:::en
"Although… but…" — acknowledges the preceding as fact, but the result differs from expectation. Written language. Often expresses the speaker's perplexity, difficulty, or surprise.
:::

### 例句||Example Sentences
:::zh
1. 買った**ものの**、一度も使っていない。（虽然买了，但一次都没用过。）
2. 日本語を勉強している**ものの**、なかなか上達しない。（虽然在学日语，但怎么也进步不了。）
3. 景気は回復した**ものの**、まだ油断できない。（虽然经济恢复了，但还不能大意。）
4. 申し込んだ**ものの**、本当に参加できるか不安だ。（虽然报了名，但真的能参加吗，有点不安。）
5. 説明は聞いた**ものの**、よくわからなかった。（虽然听了说明，但没太懂。）
:::
:::en
1. 買った**ものの**、一度も使っていない。(Although I bought it, I haven't used it once.)
2. 日本語を勉強している**ものの**、なかなか上達しない。(Although I'm studying Japanese, I'm not improving easily.)
3. 景気は回復した**ものの**、まだ油断できない。(Although the economy has recovered, we can't let our guard down yet.)
4. 申し込んだ**ものの**、本当に参加できるか不安だ。(Although I signed up, I'm anxious about whether I can actually participate.)
5. 説明は聞いた**ものの**、よくわからなかった。(Although I heard the explanation, I didn't really understand.)
:::

### 辨析||Comparison
:::zh
| けど/が | ものの |
|--------|-------|
| 口语・书面 | 主要书面 |
| 广泛的转折 | 前项是事实，后项是意外/困惑 |
| 感情中性 | 带有"虽然是这样，但困扰/意外" |
| 買った**けど**使ってない | 買った**ものの**使ってない |
:::
:::en
| けど/が | ものの |
|--------|-------|
| Both spoken and written | Mainly written |
| Broad contrast | Preceding is fact, following is unexpected |
| Emotionally neutral | Carries "it's true, but troubling/surprising" |
| 買った**けど**使ってない | 買った**ものの**使ってない |
:::

## 3. 〜とはいえ（虽说……但……）||3. 〜とはいえ (Having said that / Although…)

### 接続||Conjugation
:::zh
- 動詞普通形 + とはいえ
- い形容詞 + とはいえ
- な形容詞 + だ / である + とはいえ
- 名詞 + だ / である + とはいえ
:::
:::en
- Verb (plain form) + とはいえ
- i-adjective + とはいえ
- na-adjective + だ / である + とはいえ
- Noun + だ / である + とはいえ
:::

### 含义||Meaning
:::zh
"虽说……但……"，承认前项的事实，紧接着提出补充、修正或例外。比「ものの」更强调"话虽如此"的转折。
:::
:::en
"Having said that / although…" — acknowledges the preceding fact, then immediately adds a supplement, correction, or exception. Emphasizes the "that said" pivot more than 「ものの」.
:::

### 例句||Example Sentences
:::zh
1. 春**とはいえ**、まだ寒い日が続いている。（虽说是春天了，寒冷的日子还在持续。）
2. 社長**とはいえ**、まだ若くて経験が少ない。（虽说是社长，但还年轻经验少。）
3. 景気が回復した**とはいえ**、失業率はまだ高い。（虽说景气恢复了，失业率还是很高。）
4. 合格した**とはいえ**、ギリギリだった。（虽说合格了，但是勉强过的。）
5. 簡単だ**とはいえ**、油断してはいけない。（虽说简单，也不能大意。）
:::
:::en
1. 春**とはいえ**、まだ寒い日が続いている。(Although it's spring, cold days continue.)
2. 社長**とはいえ**、まだ若くて経験が少ない。(Although he's the president, he's still young and inexperienced.)
3. 景気が回復した**とはいえ**、失業率はまだ高い。(Although the economy has recovered, the unemployment rate is still high.)
4. 合格した**とはいえ**、ギリギリだった。(Although I passed, it was just barely.)
5. 簡単だ**とはいえ**、油断してはいけない。(Although it's easy, you mustn't be careless.)
:::

### 辨析||Comparison
:::zh
| ものの | とはいえ |
|-------|---------|
| "虽然A，但困扰/意外" | "话虽如此A，但要注意B" |
| 侧重主观感受 | 侧重客观补充 |
| 書面語 | 書面語，也可用于口语 |
| 買ったものの使ってない | 安いとはいえ、買いすぎだ |
:::
:::en
| ものの | とはいえ |
|-------|---------|
| "Although A, troubling/surprising" | "That said A, but note B" |
| Emphasis on subjective feeling | Emphasis on objective supplement |
| Written | Written, sometimes spoken |
| 買ったものの使ってない | 安いとはいえ、買いすぎだ |
:::

## 4. 〜どころか（哪里是……简直是/不但不……反而）||4. 〜どころか (Far from… / Not only not… but actually…)

### 接続||Conjugation
:::zh
- 動詞普通形 + どころか
- い形容詞 + どころか
- な形容詞 + な / である + どころか
- 名詞 + どころか
:::
:::en
- Verb (plain form) + どころか
- i-adjective + どころか
- na-adjective + な / である + どころか
- Noun + どころか
:::

### 含义||Meaning
:::zh
"哪里是A，简直是B"，"不但不A，反而B"。强烈否定前项的预期，实际情况与预期完全相反，而且程度更极端。
:::
:::en
"Far from A, it's actually B" / "Not only not A, but in fact B" — strongly negates the expectation of the preceding, with reality being the complete opposite and more extreme.
:::

### 例句||Example Sentences
:::zh
1. 手伝う**どころか**、邪魔ばかりしている。（哪里是帮忙，简直净在添乱。）
2. 感謝される**どころか**、文句を言われた。（哪里是被感谢，反而被抱怨了。）
3. 休暇**どころか**、残業続きだ。（哪有什么假期，连续加班。）
4. 日本語**どころか**、英語もできない。（别说日语了，连英语都不会。）
5. 減る**どころか**、むしろ増えている。（哪里在减少，反而在增加。）
6. 簡単**どころか**、ものすごく難しかった。（哪里简单，超级难。）
:::
:::en
1. 手伝う**どころか**、邪魔ばかりしている。(Far from helping, they're just getting in the way.)
2. 感謝される**どころか**、文句を言われた。(Far from being thanked, I was complained at.)
3. 休暇**どころか**、残業続きだ。(Vacation? Far from it — it's been nothing but overtime.)
4. 日本語**どころか**、英語もできない。(Forget Japanese — they can't even speak English.)
5. 減る**どころか**、むしろ増えている。(Far from decreasing, it's actually increasing.)
6. 簡単**どころか**、ものすごく難しかった。(Far from easy, it was incredibly difficult.)
:::

### 辨析||Comparison
:::zh
| ばかりか / だけでなく | どころか |
|-------------------|---------|
| "不仅A，而且B"（追加） | "哪里是A，简直是B"（否定+反转） |
| A和B方向一致 | A和B方向相反 |
| 日本語**だけでなく**中国語もできる | 日本語**どころか**英語もできない |
| 积极追加 | 强烈否定 |
:::
:::en
| ばかりか / だけでなく | どころか |
|-------------------|---------|
| "Not only A, but also B" (addition) | "Far from A, actually B" (negation + reversal) |
| A and B in same direction | A and B in opposite directions |
| 日本語**だけでなく**中国語もできる | 日本語**どころか**英語もできない |
| Positive addition | Strong negation |
:::
