## 1. 〜ふりをする（假装~）||1. 〜ふりをする (To pretend to ~)

### 接続||Conjugation
:::zh
| 接续 | 形式 |
|------|------|
| 动词普通形 | 知っている + ふりをする |
| い形容词 | 楽しい + ふりをする |
| な形容词な/だった | 元気な + ふりをする |
| 名词の | 学生の + ふりをする |
:::
:::en
| Conjugation | Form |
|------|------|
| Verb plain form | 知っている + ふりをする |
| い-adjective | 楽しい + ふりをする |
| な-adjective な/だった | 元気な + ふりをする |
| Noun の | 学生の + ふりをする |
:::

### 含义||Meaning
:::zh
明明不是那样，却装作那个样子。"假装……""装作……"。
:::
:::en
Pretending to be something one is not. "Pretending to..." / "Acting as if..."
:::

### 例句||Example Sentences
:::zh
1. 知らない**ふりをした**。（假装不知道。）
2. 子供が寝た**ふりをして**いる。（孩子在装睡。）
3. 彼は聞こえない**ふりをして**、無視した。（他假装没听见，无视了。）
4. 楽しい**ふりをして**いたが、本当はつまらなかった。（假装很开心，但其实很无聊。）
5. 学生の**ふりをして**、美術館に無料で入った。（假装是学生，免费进了美术馆。）
:::
:::en
1. 知らない**ふりをした**。(I pretended not to know.)
2. 子供が寝た**ふりをして**いる。(The child is pretending to be asleep.)
3. 彼は聞こえない**ふりをして**、無視した。(He pretended not to hear and ignored it.)
4. 楽しい**ふりをして**いたが、本当はつまらなかった。(I pretended to be having fun, but I was actually bored.)
5. 学生の**ふりをして**、美術館に無料で入った。(I pretended to be a student and got into the museum for free.)
:::

### 辨析||Comparison
:::zh
| 比较 | 含义 | 区别 |
|------|------|------|
| ふりをする | 假装~（意图性） | 故意装出来的，明知不是 |
| そうだ（様態） | 看起来要~ | 说话人从外观推测，非故意 |
| ように見える | 看起来像~ | 客观描述外观 |
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| ふりをする | Pretend to | Intentional deception; knowingly false |
| そうだ (appearance) | Looks like it will | Speaker's inference from appearance; unintentional |
| ように見える | Appears to be | Objective description of appearance |
:::

## 2. 〜こそ（正是~/才是~　强调）||2. 〜こそ (It is precisely ~ / ~ is what really ~)

### 接続||Conjugation
:::zh
名词 / 副词 / て形 + こそ
:::
:::en
Noun / Adverb / て-form + こそ
:::

### 含义||Meaning
:::zh
强调某个词，表示"正是这个""才是关键"。常用于回应、强调重要性。
:::
:::en
Emphasizes a specific word, indicating "it is precisely this" or "this is what matters." Often used in responses and to stress importance.
:::

### 例句||Example Sentences
:::zh
1. 「ありがとう。」「こちら**こそ**。」（"谢谢。""我才要谢谢你。"）
2. 今度**こそ**合格したい。（这次才一定要合格。）
3. 努力して**こそ**、成功できる。（正因为努力了，才能成功。）
4. あなた**こそ**、リーダーにふさわしい。（正是你，才适合当领导。）
5. 失敗した時**こそ**、成長のチャンスだ。（正是失败的时候，才是成长的机会。）
6. 健康で**あればこそ**、何でもできる。（正因为健康，什么都能做。）
:::
:::en
1. 「ありがとう。」「こちら**こそ**。」("Thank you." "No, thank YOU.")
2. 今度**こそ**合格したい。(This time I really want to pass.)
3. 努力して**こそ**、成功できる。(It is precisely by making effort that you can succeed.)
4. あなた**こそ**、リーダーにふさわしい。(It is you who is suited to be the leader.)
5. 失敗した時**こそ**、成長のチャンスだ。(It is precisely when you fail that there is a chance to grow.)
6. 健康で**あればこそ**、何でもできる。(It is precisely because you are healthy that you can do anything.)
:::

## 3. 〜ことに（令人~的是）||3. 〜ことに (To one's ~ / It is ~ that)

### 接続||Conjugation
:::zh
| 接续 | 形式 |
|------|------|
| い形容词 | 驚いた + ことに |
| な形容词な | 残念な + ことに |
| 动词た形 | 困った + ことに |
:::
:::en
| Conjugation | Form |
|------|------|
| い-adjective | 驚いた + ことに |
| な-adjective な | 残念な + ことに |
| Verb た-form | 困った + ことに |
:::

### 含义||Meaning
:::zh
在句首表达说话人的感情/评价，然后叙述具体内容。"令人……的是""……的是"。常搭配的词：驚いたことに、嬉しいことに、悲しいことに、残念なことに、困ったことに。
:::
:::en
Expresses the speaker's emotion or evaluation at the beginning of a sentence, followed by the specific content. "To one's surprise/joy/regret..." Common collocations: 驚いたことに, 嬉しいことに, 悲しいことに, 残念なことに, 困ったことに.
:::

### 例句||Example Sentences
:::zh
1. **驚いたことに**、彼はもう日本語がペラペラだった。（令人吃惊的是，他日语已经说得很流利了。）
2. **嬉しいことに**、全員合格した。（令人高兴的是，全员合格了。）
3. **悲しいことに**、あの店は閉店してしまった。（令人悲伤的是，那家店关门了。）
4. **残念なことに**、試合は中止になった。（遗憾的是，比赛取消了。）
5. **困ったことに**、パスポートをなくしてしまった。（令人头疼的是，护照弄丢了。）
:::
:::en
1. **驚いたことに**、彼はもう日本語がペラペラだった。(To my surprise, he was already fluent in Japanese.)
2. **嬉しいことに**、全員合格した。(To my delight, everyone passed.)
3. **悲しいことに**、あの店は閉店してしまった。(Sadly, that shop has closed down.)
4. **残念なことに**、試合は中止になった。(Unfortunately, the match was cancelled.)
5. **困ったことに**、パスポートをなくしてしまった。(To make matters worse, I lost my passport.)
:::

## 4. 〜ごとに（每~每次/每隔）||4. 〜ごとに (Every ~ / Each ~)

### 接続||Conjugation
:::zh
| 接续 | 形式 |
|------|------|
| 动词辞书形 | 会う + ごとに |
| 名词 | 3日 + ごとに |
| 数量词 | 10分 + ごとに |
:::
:::en
| Conjugation | Form |
|------|------|
| Verb dictionary form | 会う + ごとに |
| Noun | 3日 + ごとに |
| Counter | 10分 + ごとに |
:::

### 含义||Meaning
:::zh
两层含义：① 每次做某事时。② 每隔一定间隔。
:::
:::en
Two meanings: ① Each time something happens. ② At regular intervals.
:::

### 例句||Example Sentences
:::zh
1. 会う**ごとに**、彼女はきれいになっている。（每次见面，她都更漂亮了。）
2. 信号**ごとに**止まるので、なかなか進まない。（每个红灯都停，怎么都走不动。）
3. 年**ごとに**、技術は進歩している。（每年技术都在进步。）
4. このバスは10分**ごとに**来る。（这辆公交每10分钟来一班。）
5. 試合**ごとに**、選手は成長している。（每场比赛，选手都在成长。）
:::
:::en
1. 会う**ごとに**、彼女はきれいになっている。(Every time I meet her, she looks more beautiful.)
2. 信号**ごとに**止まるので、なかなか進まない。(We stop at every light, so we can hardly move forward.)
3. 年**ごとに**、技術は進歩している。(Every year, technology advances.)
4. このバスは10分**ごとに**来る。(This bus comes every 10 minutes.)
5. 試合**ごとに**、選手は成長している。(With every match, the athletes grow.)
:::

### 辨析||Comparison
:::zh
| 比较 | 含义 | 区别 |
|------|------|------|
| ごとに | 每~/每隔~ | 可接名词和动词，强调规律性/个别性 |
| たびに | 每次~都~ | 仅接动词/名词の，强调"每次必然伴随" |
| おきに | 每隔~ | 仅用于数量间隔（3日おきに＝每隔3天） |
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| ごとに | Every / each | Attaches to nouns and verbs; emphasizes regularity or individuality |
| たびに | Every time | Only with verbs / Noun の; emphasizes consistent occurrence |
| おきに | At intervals of | Only for numerical intervals (3日おきに = every 3 days) |
:::

## 练习||Practice

3. 驚いた（　　）、彼はもう結婚していた。

:::
