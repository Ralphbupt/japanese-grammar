# N3 文法チェックリスト

> JLPT N3 核心语法约 150 条。从日常口语过渡到书面表达，开始接触正式用语。

## 书面助词（で/に 的正式替换）

- [ ] 〜において/〜における — 在~（场所/领域，=で）
- [ ] 〜に対して/〜に対する — 对于~/与~相反
- [ ] 〜について — 关于~
- [ ] 〜によって/〜による — 由于~/通过~/取决于~
- [ ] 〜にとって — 对~来说
- [ ] 〜に関して/〜に関する — 关于~（正式）
- [ ] 〜を通じて/〜を通して — 通过~/在整个~期间
- [ ] 〜をもとに — 以~为基础
- [ ] 〜に基づいて/〜に基づく — 基于~
- [ ] 〜にかけて — 从~到~（时间/空间范围）
- [ ] 〜にわたって/〜にわたる — 跨越~/长达~

## 原因・理由

- [ ] 〜おかげで — 多亏了~（积极结果）
- [ ] 〜せいで/〜せいか — 都怪~/也许是因为~
- [ ] 〜ために — 因为~（客观原因）
- [ ] 〜以上（は）— 既然~就~
- [ ] 〜上（は）— 既然~/在~方面
- [ ] 〜からには — 既然~就~（决心）
- [ ] 〜ことから — 因为~（推理根据）

## 逆接・让步

- [ ] 〜にもかかわらず — 尽管~却~
- [ ] 〜ながら（も）— 虽然~却~
- [ ] 〜くせに — 明明~却~（指责）
- [ ] 〜としても — 即使~也~
- [ ] 〜にしても — 就算~也~
- [ ] 〜ものの — 虽然~但是~
- [ ] 〜とはいえ — 虽说~但是~
- [ ] 〜どころか — 哪里是~简直是~

## 程度・范围

- [ ] 〜ほど — ~得（程度）/越~越~
- [ ] 〜くらい/〜ぐらい — ~到（程度）
- [ ] 〜ば〜ほど — 越~越~
- [ ] 〜さえ〜ば — 只要~就~
- [ ] 〜だけ — 尽可能地~/只~
- [ ] 〜しか〜ない — 只有~
- [ ] 〜ばかりか/〜ばかりでなく — 不仅~而且~
- [ ] 〜に限って/〜に限り — 仅限于~
- [ ] 〜に限らず — 不限于~

## 倾向・状态

- [ ] 〜がちだ — 容易~/动不动就~
- [ ] 〜っぽい — 像~的/容易~（口语）
- [ ] 〜気味 — 有点~/略微~
- [ ] 〜つつある — 正在逐渐~（书面）
- [ ] 〜一方だ — 一直在~（单向变化）
- [ ] 〜かけ — 做到一半/快要~

## 判断・推量

- [ ] 〜に違いない — 一定是~
- [ ] 〜に決まっている — 肯定是~（口语）
- [ ] 〜おそれがある — 恐怕会~（负面）
- [ ] 〜っけ — ~来着？（回忆确认）
- [ ] 〜ものか — 怎么会~/才不会~

## 动作相关

- [ ] 〜ようとする — 试图做~/正要做~
- [ ] 〜ことにする — 决定做~
- [ ] 〜ことになる — 变成了~/被决定~
- [ ] 〜ようがない/〜ようもない — 没办法~
- [ ] 〜ざるを得ない — 不得不~
- [ ] 〜かねる — 难以~/不便~
- [ ] 〜かねない — 有可能~（负面）
- [ ] 〜きる/〜きれる/〜きれない — 完全~/能否完全~
- [ ] 〜っこない — 绝对不可能~（口语）
- [ ] 〜得る/〜得ない（うる/えない）— 有可能/不可能

## 并列・添加

- [ ] 〜上に — 而且~/加之~
- [ ] 〜だけでなく〜も — 不仅~还~
- [ ] 〜はもちろん — ~自不必说
- [ ] 〜をはじめ — 以~为首
- [ ] 〜とともに — 和~一起/随着~
- [ ] 〜に伴い/〜に伴って — 伴随着~

## 话题・立场

- [ ] 〜として — 作为~
- [ ] 〜にしては — 就~来说（出乎意料）
- [ ] 〜割に（は）— 虽然~但是（不相称）
- [ ] 〜向け — 面向~/针对~
- [ ] 〜向き — 适合~

## 时间

- [ ] 〜たとたん（に）— 刚一~就~
- [ ] 〜次第 — 一~就立刻~
- [ ] 〜うちに — 趁~之际
- [ ] 〜最中に — 正在~的时候
- [ ] 〜際（に）— 在~之际（正式）

## 状态・样态

- [ ] 〜まま — 保持原状/就那样~
- [ ] 〜だらけ — 满是~/全是~
- [ ] 〜っぱなし — 一直~着/放置不管
- [ ] 〜ずに — 不做~就~（=ないで，书面）
- [ ] 〜ように見える — 看起来像~

## 附加・对比

- [ ] 〜代わりに — 代替~/作为交换~
- [ ] 〜ついでに — 顺便~
- [ ] 〜に比べて — 与~相比
- [ ] 〜によると — 据~说/根据~
- [ ] 〜たびに — 每次~都~
- [ ] 〜て初めて — 做了~才第一次~

## 表达方式

- [ ] 〜というより — 与其说~不如说~
- [ ] 〜といっても — 虽说~但其实~
- [ ] 〜というと/〜といえば — 说到~/提到~
- [ ] 〜とおり — 按照~/正如~
- [ ] 〜ふりをする — 假装~
- [ ] 〜こそ — 正是~/才是~（强调）
- [ ] 〜ことに — 令人~的是（ことに＋感情）
- [ ] 〜ごとに — 每~（每次/每隔）

## 否定相关

- [ ] 〜ないことはない — 不是不能~/也不是没有~
- [ ] 〜ずにはいられない — 忍不住~/不由得~
- [ ] 〜そうもない/〜そうにない — 看起来不会~
- [ ] 〜決して〜ない — 绝对不~
- [ ] 〜別に〜ない — 并不特别~
- [ ] 〜めったに〜ない — 很少~/难得~

## 建议・推荐

- [ ] 〜たらいい/〜といい — 做~就好了
- [ ] 〜ばいい — 做~就好了（假定）
- [ ] 〜ばよかった — 要是~就好了（后悔）
- [ ] 〜てごらん — 试试看~（轻松的命令）

## 复合表达

- [ ] 〜込む — 深入~/进入~（読み込む、飛び込む）
- [ ] 〜合う — 互相~（話し合う、助け合う）
- [ ] 〜からこそ — 正因为~才~
- [ ] 〜ことは〜が — 虽然是~但是~
- [ ] 〜ことになっている — 规定是~/按惯例~
- [ ] 〜ような気がする — 总觉得~/有种~的感觉
- [ ] 〜とは限らない — 未必~/不一定~
- [ ] 〜ところが — 然而/可是（转折）
