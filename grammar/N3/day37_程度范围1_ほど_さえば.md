# Day 37 – 程度・範囲① — ほど/くらい・ぐらい/ば〜ほど/さえ〜ば

:::zh
这一课学习表示"程度"和"条件极限"的语法。「ほど」「くらい」表示程度，「ば〜ほど」表示递进关系，「さえ〜ば」表示最低限度条件。这些是N3阅读理解中的高频考点。
:::

:::en
This lesson covers grammar expressing "degree" and "conditional extremes." 「ほど」 and 「くらい」 indicate degree, 「ば〜ほど」 shows progressive change, and 「さえ〜ば」 sets a minimum condition. These are high-frequency items in N3 reading comprehension.
:::

## 1. 〜ほど（……的程度 / 到……的地步）||1. 〜ほど (To the extent of… / So much that…)

### 接続||Conjugation
:::zh
- 動詞辞書形 / ない形 + ほど
- い形容詞 + ほど
- な形容詞 + な + ほど
- 名詞 + ほど
:::
:::en
- Verb dictionary form / nai-form + ほど
- i-adjective + ほど
- na-adjective + な + ほど
- Noun + ほど
:::

### 含义||Meaning
:::zh
表示程度，"到……的地步""……得（不得了）"。常用于夸张表达。否定时「〜ほど〜ない」表示"没有比……更……的"。
:::
:::en
Indicates degree — "to the extent that…" or "so much that…". Often used in exaggerated expressions. In the negative pattern 「〜ほど〜ない」, it means "nothing is more… than…".
:::

### 例句||Example Sentences
:::zh
1. 死ぬ**ほど**疲れた。（累得要死。）
2. 涙が出る**ほど**嬉しかった。（高兴得快哭了。）
3. 彼女**ほど**優しい人はいない。（没有比她更温柔的人了。）
4. 信じられない**ほど**きれいな景色だった。（美得令人难以置信的景色。）
5. 顔が見えない**ほど**暗かった。（暗到看不见脸。）
:::
:::en
1. 死ぬ**ほど**疲れた。(I'm dead tired — tired to the point of dying.)
2. 涙が出る**ほど**嬉しかった。(I was so happy that tears came out.)
3. 彼女**ほど**優しい人はいない。(There is no one as kind as her.)
4. 信じられない**ほど**きれいな景色だった。(The scenery was unbelievably beautiful.)
5. 顔が見えない**ほど**暗かった。(It was so dark that I couldn't see faces.)
:::

### 辨析||Comparison
:::zh
| 表达 | 含义 | 区别 |
|------|------|------|
| ほど | 到……的程度 | 书面/口语均可，语气强，常含夸张 |
| くらい/ぐらい | 到……的程度 | 更口语化，语气稍轻 |
| まで | 到……为止 | 强调终点，不强调程度 |

> 「死ぬほど疲れた」和「死ぬくらい疲れた」意思几乎一样，但「ほど」更常用于书面。
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| ほど | to the extent of | Works in both writing and speech; strong tone, often exaggerated |
| くらい/ぐらい | to the extent of | More colloquial; slightly lighter tone |
| まで | up to / until | Emphasizes endpoint, not degree |

> 「死ぬほど疲れた」 and 「死ぬくらい疲れた」 mean nearly the same, but 「ほど」 is more common in writing.
:::

## 2. 〜くらい / 〜ぐらい（……左右 / ……的程度）||2. 〜くらい / 〜ぐらい (About / To the extent of…)

### 接続||Conjugation
:::zh
- 動詞普通形 + くらい/ぐらい
- い形容詞 + くらい/ぐらい
- な形容詞 + な + くらい/ぐらい
- 名詞 + くらい/ぐらい
- 数量词 + くらい/ぐらい（表示大约）
:::
:::en
- Verb plain form + くらい/ぐらい
- i-adjective + くらい/ぐらい
- na-adjective + な + くらい/ぐらい
- Noun + くらい/ぐらい
- Quantity word + くらい/ぐらい (approximate amount)
:::

### 含义||Meaning
:::zh
1. **大约**：接数量词，表示概数。「30分くらい」= 大约30分钟。
2. **程度**：和「ほど」类似，表示"到……的地步"，但口语感更强。
3. **最低限度/轻视**：「これくらい」=这种程度（暗示很简单/很少）。
:::
:::en
1. **Approximate amount**: After quantity words. 「30分くらい」= about 30 minutes.
2. **Degree**: Similar to 「ほど」, meaning "to the extent that…", but more colloquial.
3. **Minimum / dismissive**: 「これくらい」= this much (implying it's trivial / small).
:::

### 例句||Example Sentences
:::zh
1. 駅まで歩いて10分**くらい**です。（到车站走路大约10分钟。）
2. 泣きたい**くらい**悔しかった。（懊悔得想哭。）
3. あいさつ**ぐらい**できるでしょう？（起码打个招呼总会吧？）
4. 子供でもわかる**くらい**簡単だ。（简单到连小孩都懂。）
5. 3日**ぐらい**休みたい。（想休息3天左右。）
:::
:::en
1. 駅まで歩いて10分**くらい**です。(It's about a 10-minute walk to the station.)
2. 泣きたい**くらい**悔しかった。(I was so frustrated I wanted to cry.)
3. あいさつ**ぐらい**できるでしょう？(You can at least say hello, right?)
4. 子供でもわかる**くらい**簡単だ。(It's so simple even a child can understand.)
5. 3日**ぐらい**休みたい。(I want to take about 3 days off.)
:::

### 辨析||Comparison
:::zh
| 表达 | 用法 | 例 |
|------|------|------|
| くらい（程度） | 到……地步 | 泣きたいくらい嬉しい |
| くらい（概数） | 大约 | 1時間くらいかかる |
| くらい（轻视） | 至少/起码 | これくらいは自分でやれ |

> 「くらい」和「ぐらい」完全相同，没有意义差异。「ぐらい」稍更口语化。
:::
:::en
| Usage | Meaning | Example |
|------|------|------|
| くらい (degree) | to the extent of | 泣きたいくらい嬉しい |
| くらい (approximate) | about | 1時間くらいかかる |
| くらい (dismissive) | at least | これくらいは自分でやれ |

> 「くらい」 and 「ぐらい」 are completely interchangeable with no meaning difference. 「ぐらい」 is slightly more colloquial.
:::

## 3. 〜ば〜ほど（越……越……）||3. 〜ば〜ほど (The more… the more…)

### 接続||Conjugation
:::zh
- 動詞ば形 + 動詞辞書形 + ほど
- い形容詞：～ければ ～い + ほど
- な形容詞：～であれば ～な + ほど

※ 同一个词重复两次。
:::
:::en
- Verb ba-form + verb dictionary form + ほど
- i-adjective: ～ければ ～い + ほど
- na-adjective: ～であれば ～な + ほど

※ The same word is repeated twice.
:::

### 含义||Meaning
:::zh
"越……越……"，表示一方的变化导致另一方同方向变化。是中文"越~越~"的精确对应。
:::
:::en
"The more… the more…" — one change leads to a proportional change in the same direction. This corresponds precisely to the Chinese "越~越~".
:::

### 例句||Example Sentences
:::zh
1. 練習すれ**ば**する**ほど**上手になる。（越练习越熟练。）
2. 考えれ**ば**考える**ほど**わからなくなる。（越想越不明白。）
3. この本は読め**ば**読む**ほど**面白い。（这本书越读越有意思。）
4. 給料は高けれ**ば**高い**ほど**いい。（工资越高越好。）
5. 試験が近づけ**ば**近づく**ほど**緊張する。（考试越近越紧张。）
6. 複雑であれ**ば**複雑な**ほど**やりがいがある。（越复杂越有干劲。）
:::
:::en
1. 練習すれ**ば**する**ほど**上手になる。(The more you practice, the better you get.)
2. 考えれ**ば**考える**ほど**わからなくなる。(The more I think about it, the less I understand.)
3. この本は読め**ば**読む**ほど**面白い。(The more you read this book, the more interesting it gets.)
4. 給料は高けれ**ば**高い**ほど**いい。(The higher the salary, the better.)
5. 試験が近づけ**ば**近づく**ほど**緊張する。(The closer the exam gets, the more nervous I become.)
6. 複雑であれ**ば**複雑な**ほど**やりがいがある。(The more complex it is, the more rewarding it feels.)
:::

### 辨析||Comparison
:::zh
| 表达 | 含义 | 区别 |
|------|------|------|
| ば〜ほど | 越……越…… | 正式，强调比例关系 |
| ほど | 到……的程度 | 单纯表示程度，不含递进 |
| たら〜たで | 如果……那就…… | 表示假设结果，不含递进 |

> 注意：「ば〜ほど」必须用同一个词的两种形式，不能用两个不同的词。
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| ば〜ほど | the more… the more… | Formal; emphasizes proportional relationship |
| ほど | to the extent of | Simply indicates degree, no progression |
| たら〜たで | if… then… | States a hypothetical result, no progression |

> Note: 「ば〜ほど」 must repeat the same word in two forms — you cannot use two different words.
:::

## 4. 〜さえ〜ば（只要……就……）||4. 〜さえ〜ば (If only… / As long as…)

### 接続||Conjugation
:::zh
- 動詞ます形（去ます）+ さえ + すれば
- 名詞 + さえ + あれば / なければ
- い形容詞（去い）+ くさえ + あれば
- な形容詞 + で + さえ + あれば
:::
:::en
- Verb masu-stem + さえ + すれば
- Noun + さえ + あれば / なければ
- i-adjective (drop い) + くさえ + あれば
- na-adjective + で + さえ + あれば
:::

### 含义||Meaning
:::zh
"只要……就……"，表示某一个最低限度的条件满足，后面的结果就能成立。「さえ」强调"连这个都/只要这个就够了"。
:::
:::en
"If only… then…" or "as long as…" — if one minimum condition is met, the result will follow. 「さえ」 emphasizes "even just this is enough."
:::

### 例句||Example Sentences
:::zh
1. 健康で**さえ**あれ**ば**、それで十分だ。（只要健康就足够了。）
2. お金**さえ**あれ**ば**、何でもできると思っている。（以为只要有钱什么都能做。）
3. 電気**さえ**あれ**ば**、この機械は動く。（只要有电，这台机器就能运转。）
4. 努力し**さえ**すれ**ば**、合格できる。（只要努力就能及格。）
5. あなた**さえ**よけれ**ば**、一緒に行きましょう。（只要你愿意，一起去吧。）
6. 天気が良く**さえ**あれ**ば**、ピクニックに行ける。（只要天气好就能去野餐。）
:::
:::en
1. 健康で**さえ**あれ**ば**、それで十分だ。(As long as I'm healthy, that's enough.)
2. お金**さえ**あれ**ば**、何でもできると思っている。(He thinks he can do anything as long as he has money.)
3. 電気**さえ**あれ**ば**、この機械は動く。(This machine runs as long as there is electricity.)
4. 努力し**さえ**すれ**ば**、合格できる。(If you just make the effort, you can pass.)
5. あなた**さえ**よけれ**ば**、一緒に行きましょう。(If it's fine with you, let's go together.)
6. 天気が良く**さえ**あれ**ば**、ピクニックに行ける。(As long as the weather is good, we can go on a picnic.)
:::

### 辨析||Comparison
:::zh
| 表达 | 含义 | 区别 |
|------|------|------|
| さえ〜ば | 只要……就…… | 强调最低限度条件 |
| ば | 如果……就…… | 一般条件 |
| たら | 如果……就…… | 更口语，含时间先后 |
| なら | 如果说……的话 | 基于对方的话题/信息 |

> 「さえ〜ば」暗含"其他都不需要，只要这一个条件就够了"的语气。  
> 「ば」只是单纯的条件，没有"只要"的限定感。
:::
:::en
| Expression | Meaning | Difference |
|------|------|------|
| さえ〜ば | if only / as long as | Emphasizes minimum condition |
| ば | if… then | General condition |
| たら | if / when | More colloquial; implies temporal sequence |
| なら | if (it's the case that) | Based on the listener's topic/info |

> 「さえ〜ば」 implies "nothing else is needed — just this one condition is enough."  
> 「ば」 is a plain conditional with no such limiting nuance.
:::
