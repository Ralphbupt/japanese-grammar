## 1. 〜て以来（自从……以来）||1. 〜て以来 (Ever since…)

### 接续||Conjugation
:::zh
动词て形 + 以来
:::
:::en
Verb て-form + 以来
:::

### 含义||Meaning
:::zh
"自从做了……之后（一直到现在）"，强调从过去某个时间点延续到现在。
:::
:::en
"Ever since doing…" — emphasizes that a state or situation has continued from a past point up to the present.
:::

### 例句||Example Sentences
:::zh
1. 日本に来**て以来**、一度も国に帰っていない。（自从来了日本，一次都没回过国。）
2. 大学を卒業し**て以来**、彼に会っていない。（自从大学毕业后，就没见过他。）
3. あの事故があっ**て以来**、怖くて車に乗れない。（自从那次事故后，害怕得不敢开车。）
4. 子供が生まれ**て以来**、生活が変わった。（自从孩子出生后，生活就变了。）
5. この薬を飲み始め**て以来**、調子がいい。（自从开始吃这个药，状态一直很好。）
:::
:::en
1. 日本に来**て以来**、一度も国に帰っていない。(Ever since coming to Japan, I haven't returned home even once.)
2. 大学を卒業し**て以来**、彼に会っていない。(Ever since graduating from university, I haven't seen him.)
3. あの事故があっ**て以来**、怖くて車に乗れない。(Ever since that accident, I've been too scared to get in a car.)
4. 子供が生まれ**て以来**、生活が変わった。(Ever since my child was born, life has changed.)
5. この薬を飲み始め**て以来**、調子がいい。(Ever since I started taking this medicine, I've been feeling well.)
:::

### て以来 vs てから||て以来 vs てから
:::zh
- てから = 做完A之后做B（单纯的先后顺序）
- て以来 = 自从A之后一直……（强调持续性，更书面）
:::
:::en
- てから = After doing A, do B (simple sequential order)
- て以来 = Ever since A… (emphasizes continuation, more formal/written)
:::

## 2. 〜にかけて（从……到……期间）||2. 〜にかけて (Spanning from… to…)

### 接续||Conjugation
:::zh
名词 + から + 名词 + にかけて
:::
:::en
Noun + から + Noun + にかけて
:::

### 含义||Meaning
:::zh
"从……到……（的期间）"，表示一段连续的范围或时间。
:::
:::en
"From … to …" — expresses a continuous range of time or space, with vague endpoints.
:::

### 例句||Example Sentences
:::zh
1. 今夜から明日の朝**にかけて**、雨が降るでしょう。（从今晚到明天早上，大概会下雨。）
2. 関東から東北**にかけて**、地震がありました。（从关东到东北地区发生了地震。）
3. 来週から再来週**にかけて**、出張の予定です。（从下周到下下周预定出差。）
4. 秋から冬**にかけて**、気温が下がる。（从秋天到冬天气温下降。）
:::
:::en
1. 今夜から明日の朝**にかけて**、雨が降るでしょう。(It will probably rain from tonight through tomorrow morning.)
2. 関東から東北**にかけて**、地震がありました。(There was an earthquake spanning from Kanto to Tohoku.)
3. 来週から再来週**にかけて**、出張の予定です。(I'm scheduled to be on a business trip from next week through the week after.)
4. 秋から冬**にかけて**、気温が下がる。(Temperatures drop from autumn through winter.)
:::

### にかけて vs まで||にかけて vs まで
:::zh
- まで = 精确的终点
- にかけて = 模糊的范围，"到……左右"
:::
:::en
- まで = a precise endpoint
- にかけて = a vague range, "around … to …"
:::

## 3. 〜において / 〜における（在……方面/场合）||3. 〜において / 〜における (In / At / In terms of…)

### 接续||Conjugation
:::zh
名词 + において（动词前）/ における（名词前）
:::
:::en
Noun + において (before a verb) / における (before a noun)
:::

### 含义||Meaning
:::zh
"在……（场所/场合/方面）"，「で」的书面正式版。
:::
:::en
"In / at / in the field of…" — the formal, written equivalent of the particle 「で」.
:::

### 例句||Example Sentences
:::zh
1. 会議は3階の会議室**において**行われます。（会议在3楼会议室举行。）
2. この分野**において**、彼は第一人者だ。（在这个领域，他是第一人。）
3. 現代社会**における**最大の問題は環境問題だ。（现代社会中最大的问题是环境问题。）
4. 日本の歴史**において**、重要な出来事だ。（在日本历史上，是重要事件。）
5. 国際会議**における**彼の発言は注目された。（他在国际会议上的发言受到了关注。）
:::
:::en
1. 会議は3階の会議室**において**行われます。(The meeting will be held in the conference room on the 3rd floor.)
2. この分野**において**、彼は第一人者だ。(In this field, he is the foremost expert.)
3. 現代社会**における**最大の問題は環境問題だ。(The biggest problem in modern society is environmental issues.)
4. 日本の歴史**において**、重要な出来事だ。(It is an important event in Japanese history.)
5. 国際会議**における**彼の発言は注目された。(His remarks at the international conference drew attention.)
:::

## 4. 〜に対して（对于……/与此相反）||4. 〜に対して (Toward / In contrast to…)

### 接续||Conjugation
:::zh
名词 + に対して / に対する + 名词
:::
:::en
Noun + に対して / に対する + Noun
:::

### 含义①：对于……（对象）||Meaning ①: Toward / Regarding (target of action or attitude)
:::zh
"对于/针对……"，表示态度、行为的对象。

1. お客様**に対して**、丁寧に話してください。（对客人请说话礼貌。）
2. 先生**に対する**態度が悪い。（对老师的态度不好。）
3. 環境問題**に対して**、何をすべきか。（对于环境问题，应该做什么。）
4. この質問**に対する**答えは簡単だ。（对于这个问题的回答很简单。）
:::
:::en
"Toward / regarding…" — marks the target of an attitude or action.

1. お客様**に対して**、丁寧に話してください。(Please speak politely toward customers.)
2. 先生**に対する**態度が悪い。(Your attitude toward the teacher is poor.)
3. 環境問題**に対して**、何をすべきか。(What should we do about environmental problems?)
4. この質問**に対する**答えは簡単だ。(The answer to this question is simple.)
:::

### 含義②：与此相反（对比）||Meaning ②: In contrast (comparison)
:::zh
"相对于……/与此相反……"

5. 兄が理系な**のに対して**、弟は文系だ。（哥哥是理科，弟弟相反是文科。）
6. 東京の人口が増えている**のに対して**、地方は減っている。（东京人口在增加，与此相反地方在减少。）
:::
:::en
"Whereas… / In contrast to…"

5. 兄が理系な**のに対して**、弟は文系だ。(Whereas the older brother studies science, the younger brother studies humanities.)
6. 東京の人口が増えている**のに対して**、地方は減っている。(While Tokyo's population is growing, regional areas are declining.)
:::

## 5. 〜について（关于……）||5. 〜について (About / Regarding…)

### 接续||Conjugation
:::zh
名词 + について / についての + 名词
:::
:::en
Noun + について / についての + Noun
:::

### 含义||Meaning
:::zh
"关于……"，引出话题。
:::
:::en
"About / regarding…" — introduces a topic.
:::

### 例句||Example Sentences
:::zh
1. 日本の文化**について**、レポートを書いた。（写了关于日本文化的报告。）
2. この問題**について**、意見を聞かせてください。（请就这个问题告诉我你的意见。）
3. 環境問題**についての**本を読んだ。（读了关于环境问题的书。）
4. 将来**について**、よく考えたほうがいい。（应该好好想想关于未来的事。）
:::
:::en
1. 日本の文化**について**、レポートを書いた。(I wrote a report about Japanese culture.)
2. この問題**について**、意見を聞かせてください。(Please share your opinion about this issue.)
3. 環境問題**についての**本を読んだ。(I read a book about environmental problems.)
4. 将来**について**、よく考えたほうがいい。(You should think carefully about the future.)
:::

## 6. 〜によって / 〜による（根据/因为/通过）||6. 〜によって / 〜による (By / Due to / Depending on…)

### 多种含义||Multiple Meanings

#### ① 手段/方式 — "通过……"||① Means / Method — "By means of…"
:::zh
- インターネット**によって**、情報を集めた。（通过网络收集了信息。）
- 努力**によって**、成功した。（通过努力成功了。）
:::
:::en
- インターネット**によって**、情報を集めた。(Information was gathered via the internet.)
- 努力**によって**、成功した。(Success was achieved through effort.)
:::

#### ② 原因 — "因为……"||② Cause — "Due to…"
:::zh
- 地震**によって**、多くの建物が壊れた。（因为地震，很多建筑倒塌了。）
- 台風**による**被害が大きかった。（台风造成的损害很大。）
:::
:::en
- 地震**によって**、多くの建物が壊れた。(Many buildings were destroyed due to the earthquake.)
- 台風**による**被害が大きかった。(The damage caused by the typhoon was severe.)
:::

#### ③ 根据不同 — "取决于……/根据……而不同"||③ Variation — "Depending on…"
:::zh
- 人**によって**、考え方は違う。（因人而异，想法不同。）
- 国**によって**、文化が違う。（根据国家不同，文化不同。）
- 場合**によって**は、中止になることもある。（根据情况，有时也会取消。）
:::
:::en
- 人**によって**、考え方は違う。(Ways of thinking differ from person to person.)
- 国**によって**、文化が違う。(Culture varies from country to country.)
- 場合**によって**は、中止になることもある。(Depending on the situation, it may be cancelled.)
:::

#### ④ 被动的执行者（书面）||④ Agent in passive sentences (formal/written)
:::zh
- この小説は夏目漱石**によって**書かれた。（这部小说由夏目漱石所写。）
:::
:::en
- この小説は夏目漱石**によって**書かれた。(This novel was written by Natsume Soseki.)
:::

## 7. 〜を通じて / 〜を通して（通过……/贯穿……）||7. 〜を通じて / 〜を通して (Through / Throughout…)

### 接续||Conjugation
:::zh
名词 + を通じて / を通して
:::
:::en
Noun + を通じて / を通して
:::

### 含义①：通过（手段/媒介）||Meaning ①: Through (means / medium)
:::zh
- 友達**を通じて**、彼と知り合った。（通过朋友认识了他。）
- インターネット**を通じて**、世界中の情報が手に入る。（通过网络能得到全世界的信息。）
:::
:::en
- 友達**を通じて**、彼と知り合った。(I got to know him through a friend.)
- インターネット**を通じて**、世界中の情報が手に入る。(You can access information from around the world through the internet.)
:::

### 含义②：贯穿（整个期间）||Meaning ②: Throughout (an entire period)
:::zh
- 一年**を通じて**、温暖な気候だ。（全年气候温暖。）
- 一生**を通して**、学び続けることが大切だ。（终生持续学习很重要。）
:::
:::en
- 一年**を通じて**、温暖な気候だ。(The climate is warm throughout the year.)
- 一生**を通して**、学び続けることが大切だ。(It is important to keep learning throughout one's life.)
:::

### 接续||Conjugation
:::zh
名词 + をもとに / をもとにして
:::
:::en
Noun + をもとに / をもとにして
:::

### 含义||Meaning
:::zh
"以……为基础/根据……"
:::

"Based on… / Drawing from…"
:::

2. アンケートの結果**をもとに**、報告書を書いた。(I wrote the report based on the survey results.)
3. 経験**をもとに**して、アドバイスする。(I give advice based on experience.)

## 总结：N4书面语法助词搭配表||Summary: N4 Formal Grammar Particle Reference

:::zh
| 表达 | 含义 | 口语替代 |
|------|------|---------|
| において | 在……（场所/方面） | で |
| に対して | 对于…… | に |
| について | 关于…… | のこと |
| によって | 通过/因为/取决于 | で/のせいで |
| を通じて | 通过/贯穿 | で/ずっと |
| をもとに | 以……为基础 | を使って |

> 这些语法本质上是助词「で」「に」的书面升级版。口语中用简单助词就够了，但阅读/考试中大量出现。
:::

:::en
| Expression | Meaning | Colloquial equivalent |
|------|------|---------|
| において | In / at (place / domain) | で |
| に対して | Toward / regarding | に |
| について | About / regarding | のこと |
| によって | Through / due to / depending on | で/のせいで |
| を通じて | Through / throughout | で/ずっと |
| をもとに | Based on | を使って |

> These expressions are essentially formal, written upgrades of the particles 「で」 and 「に」. Simple particles suffice in conversation, but these appear frequently in reading and exams.
:::
