## 1. 〜わけだ（难怪/当然/也就是说）||1. 〜わけだ (No wonder / Of course / In other words)

### 接続||Conjugation
:::zh
简体 + わけだ
:::
:::en
Plain form + わけだ
:::

### 核心语感||Core Nuance
:::zh
"原来如此，所以才会这样啊"——听了理由后恍然大悟的感觉。
:::
:::en
"Ah, that makes sense, so that's why…" — the feeling of sudden realization after hearing an explanation.
:::

### 用法①：难怪/怪不得（纳得推理）||Usage ①: No wonder / Makes sense (reasoning from a given cause)

:::zh
1. A：彼は10年も日本に住んでいたんですよ。
   B：**道理で**日本語が上手な**わけだ**。（难怪日语那么好。）
2. 昨日は一日中歩いた。疲れた**わけだ**。（昨天走了一整天。难怪累了。）
3. エアコンが壊れている。暑い**わけだ**。（空调坏了。难怪热。）
:::
:::en
1. A：彼は10年も日本に住んでいたんですよ。
   B：**道理で**日本語が上手な**わけだ**。(No wonder his Japanese is so good.)
2. 昨日は一日中歩いた。疲れた**わけだ**。(I walked all day yesterday. No wonder I'm tired.)
3. エアコンが壊れている。暑い**わけだ**。(The air conditioner is broken. No wonder it's hot.)
:::

### 用法②：当然/自然而然（逻辑推理）||Usage ②: Naturally / Of course (logical conclusion)

:::zh
4. 1ドルが150円なら、アメリカの物は高い**わけだ**。（1美元150日元的话，美国的东西当然贵。）
5. 3時間も寝ていないんだから、眠い**わけだ**。（才睡了不到3小时，当然困。）
:::
:::en
4. 1ドルが150円なら、アメリカの物は高い**わけだ**。(If 1 dollar is 150 yen, American goods are of course expensive.)
5. 3時間も寝ていないんだから、眠い**わけだ**。(I've barely slept 3 hours, so of course I'm sleepy.)
:::

### 用法③：也就是说（换种说法）||Usage ③: In other words (rephrasing)

:::zh
6. 全員反対ですか。つまり、この計画は中止する**わけですね**。（全员反对是吗。也就是说，这个计划取消了是吧。）
7. 彼は来月卒業する。もう大学4年生な**わけだ**。（他下月毕业。也就是说已经大四了。）
:::
:::en
6. 全員反対ですか。つまり、この計画は中止する**わけですね**。(Everyone is opposed? In other words, this plan is being cancelled, right?)
7. 彼は来月卒業する。もう大学4年生な**わけだ**。(He graduates next month. In other words, he's already a university senior.)
:::

## 2. 〜わけがない（不可能……）||2. 〜わけがない (There's no way… / It's impossible that…)

### 接続||Conjugation
:::zh
简体 + わけがない
:::
:::en
Plain form + わけがない
:::

### 含义||Meaning
:::zh
"不可能……"、"怎么可能……"。强烈否定某种可能性。
:::
:::en
"There's no way…" / "It's impossible that…" — strongly denies a possibility.
:::

### 例句||Example Sentences
:::zh
1. 彼が犯人の**わけがない**。（他不可能是犯人。）
2. そんな簡単に合格できる**わけがない**。（不可能那么简单就考过。）
3. 知らない**わけがない**だろう。（怎么可能不知道。）
4. あの優しい人が、そんなことをする**わけがない**。（那么温柔的人，不可能做那种事。）
:::
:::en
1. 彼が犯人の**わけがない**。(There's no way he's the culprit.)
2. そんな簡単に合格できる**わけがない**。(There's no way you can pass that easily.)
3. 知らない**わけがない**だろう。(There's no way you don't know.)
4. あの優しい人が、そんなことをする**わけがない**。(There's no way that kind person would do something like that.)
:::

### わけがない vs はずがない||わけがない vs はずがない
:::zh
两者意思接近，都是"不可能"：
- はずがない = 根据逻辑/证据判断不可能（理性）
- わけがない = 从感情/常识上觉得不可能（更强烈/感性）
:::
:::en
Both mean "impossible," but with a different nuance:
- はずがない = logically/evidentially impossible (rational judgment)
- わけがない = impossible from an emotional or common-sense standpoint (stronger, more emotional)
:::

## 3. 〜わけではない（并不是……/并非……）||3. 〜わけではない (It's not that… / Not necessarily…)

### 接続||Conjugation
:::zh
简体 + わけではない
:::
:::en
Plain form + わけではない
:::

### 含义||Meaning
:::zh
**部分否定**——"并不是完全……"、"也不是说……"。否定过度概括。
:::
:::en
**Partial negation** — "It's not that…" / "It doesn't mean…" — denies an overgeneralization or an assumed conclusion.
:::

### 例句||Example Sentences
:::zh
1. 日本語が嫌いな**わけではない**。ただ、難しいと思うだけだ。（并不是讨厌日语。只是觉得难而已。）
2. お金がない**わけではない**が、節約したい。（也不是没钱，但想省点。）
3. 忙しくない**わけではない**が、手伝いますよ。（不是说不忙，但我来帮忙。）
4. 全部分かった**わけではない**が、だいたい理解できた。（不是全都懂了，但大致理解了。）
5. 反対している**わけではありません**。ただ心配なだけです。（并不是反对。只是担心而已。）
:::
:::en
1. 日本語が嫌いな**わけではない**。ただ、難しいと思うだけだ。(It's not that I dislike Japanese. I just think it's difficult.)
2. お金がない**わけではない**が、節約したい。(It's not that I have no money, but I want to save.)
3. 忙しくない**わけではない**が、手伝いますよ。(It's not that I'm not busy, but I'll help.)
4. 全部分かった**わけではない**が、だいたい理解できた。(It's not that I understood everything, but I mostly got it.)
5. 反対している**わけではありません**。ただ心配なだけです。(It's not that I'm opposed. I'm just worried.)
:::

### わけではない vs ないわけではない（双重否定）||わけではない vs ないわけではない (double negative)
:::zh
- 好きなわけではない = 并不是喜欢（否定"喜欢"）
- 好きじゃないわけではない = 也不是不喜欢（双重否定≈其实有点喜欢）
:::
:::en
- 好きなわけではない = It's not that I like it (denies "liking")
- 好きじゃないわけではない = It's not that I don't like it (double negative ≈ actually kind of like it)
:::

## 4. 〜わけにはいかない（不能……/不得不……）||4. 〜わけにはいかない (Can't do… / Must do…)

### 接続||Conjugation
:::zh
- 动词辞书形 + わけにはいかない（不能做）
- 动词ない形 + わけにはいかない（不能不做 = 必须做）
:::
:::en
- Verb dictionary form + わけにはいかない (can't do something)
- Verb ない-form + わけにはいかない (can't not do = must do)
:::

### 含义||Meaning
:::zh
因为社会常理/道德/义务等原因，"不能做某事"或"不能不做某事"。
:::
:::en
Due to social norms, morality, or obligations, one "cannot do" or "cannot avoid doing" something.
:::

### 例句||Example Sentences

#### 不能做（辞书形+わけにはいかない）||Cannot do (dictionary form + わけにはいかない)
:::zh
1. まだ仕事が残っているから、帰る**わけにはいかない**。（还有工作没做完，不能回去。）
2. 秘密だから、言う**わけにはいかない**。（是秘密，不能说。）
3. 人のものを勝手に使う**わけにはいかない**。（不能擅自用别人的东西。）
:::
:::en
1. まだ仕事が残っているから、帰る**わけにはいかない**。(I still have work left, so I can't go home.)
2. 秘密だから、言う**わけにはいかない**。(It's a secret, so I can't tell you.)
3. 人のものを勝手に使う**わけにはいかない**。(I can't just use other people's things without permission.)
:::

#### 不能不做 = 必须做（ない形+わけにはいかない）||Cannot not do = Must do (ない-form + わけにはいかない)
:::zh
4. 明日は試験だから、勉強しない**わけにはいかない**。（明天考试，不能不学习。）
5. 約束したんだから、行かない**わけにはいかない**。（既然约好了，不能不去。）
:::
:::en
4. 明日は試験だから、勉強しない**わけにはいかない**。(I have an exam tomorrow, so I can't skip studying.)
5. 約束したんだから、行かない**わけにはいかない**。(I made a promise, so I can't not go.)
:::

## 5. 〜ものだ（感慨/回忆/当然/忠告）||5. 〜ものだ (Exclamation / Nostalgia / Natural expectation / Advice)

### 接続||Conjugation
:::zh
简体 + ものだ
:::
:::en
Plain form + ものだ
:::

### 多义语法，根据语境判断：||Multi-meaning grammar — determined by context:

### 用法①：感慨/惊叹||Usage ①: Exclamation / Admiration

:::zh
1. 時間が経つのは早い**ものだ**。（时间过得真快啊。）
2. 人生は分からない**ものだ**。（人生真是不可预知啊。）
3. 子供はよく食べる**ものだ**。（小孩子就是能吃啊。→ 一般性感慨）
:::
:::en
1. 時間が経つのは早い**ものだ**。(Time really does pass quickly.)
2. 人生は分からない**ものだ**。(Life really is unpredictable.)
3. 子供はよく食べる**ものだ**。(Kids sure do eat a lot. → general observation)
:::

### 用法②：回忆往事 — "以前常常……"||Usage ②: Nostalgic recollection — "used to…"

:::zh
4. 子供の頃、よくこの川で遊んだ**ものだ**。（小时候经常在这条河玩。→ 怀念）
5. 学生時代は毎日遅くまで勉強した**ものだ**。（学生时代每天学习到很晚啊。）
6. 昔はここに大きな木があった**ものだ**。（以前这里有棵大树。→ 现在没了）
:::
:::en
4. 子供の頃、よくこの川で遊んだ**ものだ**。(I used to play in this river as a child. → nostalgic)
5. 学生時代は毎日遅くまで勉強した**ものだ**。(I used to study late every day during my student days.)
6. 昔はここに大きな木があった**ものだ**。(There used to be a big tree here. → it's gone now)
:::

### 用法③：理所当然/应该 — "本来就应该……"||Usage ③: Natural expectation / Moral norm — "one should…"

:::zh
7. 約束は守る**ものだ**。（约定就应该遵守。→ 道德/常识）
8. 人の悪口は言わない**ものだ**。（不应该说别人坏话。）
9. 年上の人には敬語を使う**ものだ**。（对长辈应该用敬语。）
:::
:::en
7. 約束は守る**ものだ**。(One should keep their promises. → social norm)
8. 人の悪口は言わない**ものだ**。(One shouldn't speak badly of others.)
9. 年上の人には敬語を使う**ものだ**。(One should use polite speech with those older than you.)
:::

## 6. 〜ものだから / 〜もので（因为……所以——辩解语气）||6. 〜ものだから / 〜もので (Because… — excuse/apology tone)

### 接続||Conjugation
:::zh
简体 + ものだから / もので
:::
:::en
Plain form + ものだから / もので
:::

### 含义||Meaning
:::zh
"因为……（所以没办法）"，带有**辩解/借口**的语气。
:::
:::en
"Because… (and so it couldn't be helped)" — carries a tone of **making excuses or apologizing**.
:::

### 例句||Example Sentences
:::zh
1. 電車が遅れた**ものですから**、遅刻してしまいました。（因为电车延误了，所以迟到了。→ 在辩解）
2. 初めて**なもので**、よくわからないんです。（因为是第一次，不太明白。→ 请理解）
3. 急いでいた**ものだから**、忘れ物をしてしまった。（因为太着急了，忘了东西。）
4. あまりにも美味しかった**ものだから**、全部食べてしまった。（因为太好吃了，全吃完了。→ 辩解）
:::
:::en
1. 電車が遅れた**ものですから**、遅刻してしまいました。(Because the train was delayed, I ended up being late. → offering an excuse)
2. 初めて**なもので**、よくわからないんです。(Because it's my first time, I don't quite understand. → asking for understanding)
3. 急いでいた**ものだから**、忘れ物をしてしまった。(Because I was in a hurry, I forgot something.)
4. あまりにも美味しかった**ものだから**、全部食べてしまった。(Because it was so delicious, I ended up eating it all. → excuse)
:::

### ものだから vs から/ので||ものだから vs から/ので

:::zh
| | から | ので | ものだから |
|---|---|---|---|
| 语气 | 直接给理由 | 客观说明理由 | 辩解/借口 |
| 暗示 | — | — | "我也没办法" |
:::
:::en
| | から | ので | ものだから |
|---|---|---|---|
| Tone | Direct reason | Objective explanation | Excuse / apology |
| Implication | — | — | "I couldn't help it" |
:::

## 7. わけだ vs ものだ 区分||7. わけだ vs ものだ — Key Differences

:::zh
| | わけだ | ものだ |
|---|---|---|
| 核心 | 逻辑推理/恍然大悟 | 感慨/本质/回忆 |
| 例 | 暑いわけだ（难怪热→有原因） | 暑いものだ（就是热啊→感慨） |
| 否定 | わけではない（并非）/わけがない（不可能） | ものではない（不应该） |
:::
:::en
| | わけだ | ものだ |
|---|---|---|
| Core meaning | Logical reasoning / realization | Exclamation / nature / nostalgia |
| Example | 暑いわけだ (No wonder it's hot → there's a reason) | 暑いものだ (It sure is hot → general feeling) |
| Negation | わけではない (not that…) / わけがない (impossible) | ものではない (one shouldn't…) |
:::
