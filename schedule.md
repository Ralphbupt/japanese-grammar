## 遗忘曲线复习安排

新学的语法点，按以下间隔复习：

```
学习当天 → 当天晚上回顾（第1次）
      → 第2天复习（第2次）
      → 第4天复习（第3次）
      → 第7天复习（第4次）
      → 第14天复习（第5次）
      → 第30天复习（第6次）
```

**每日流程建议：**
1. 复习（30%时间）：按遗忘曲线复习之前的语法点
2. 新学（50%时间）：学习当天的新语法
3. 练习（20%时间）：造句/做题巩固

## 学习方法

### 每个语法点的学习步骤
1. **理解含义**：中文释义+核心语感
2. **记住接续**：动词/名词/形容词 + 语法的接续方式
3. **看例句**：至少3个例句，理解语境
4. **造句练习**：自己造2-3个句子
5. **对比辨析**：与相似语法区分

### 遗忘曲线执行要点
- 每天开始学习前，先完成当天的复习任务
- 复习时先尝试回忆，想不起来再看笔记
- 复习时重点关注：接续方式、易混淆语法的区别
- 如果第4次复习还记不住，重新进入复习循环
