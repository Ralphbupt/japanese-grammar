# Japanese Grammar Notes

Structured grammar notes for JLPT N2 in 8 weeks.

- Bilingual: Japanese + Chinese
- From zero grammar to N2
- Spaced repetition built-in

## Structure

```
grammar/
  week01-02/   N5 (day00-day08)
  week03-04/   N4 (day12-day20)
  week05-06/   N3 (WIP)
  week07-08/   N2 (WIP)
schedule.md    8-week master schedule
review/        Spaced repetition tracking
```

## Lesson Format

Each lesson covers multiple grammar points with:
- Conjugation rules
- 3+ example sentences
- Comparison with similar grammar
- Practice exercises
- Review checklist: same day > +1d > +4d > +7d > +14d > +30d
